## 0. エグゼクティブサマリー

| 項目 | 数値 | 備考 |
|---|---|---|
| **総資金需要** | **¥30,000,000** | 初期投資¥20M + 運転資金¥10M（6ヶ月分） |
| **損益分岐点** | **131杯/日** | 月商¥3.33M、客単価¥850 |
| **保守シナリオ営業利益率** | 21% | 200杯/日、月¥1.06M、回収28ヶ月 |
| **標準シナリオ営業利益率** | 38% | 400杯/日、月¥3.92M、回収7.7ヶ月 |
| **強気シナリオ営業利益率** | 45% | 700杯/日、月¥8.11M、回収3.7ヶ月 |
| **アラビカ嵐山級（参考）** | 52% | 2,000杯/日、月¥26.6M、ただし観光地依存 |

**結論の骨子**:
1. **標準シナリオ（400杯/日）が成立すれば、1年以内に投資回収可能**で極めて魅力的
2. **損益分岐点131杯/日は新規ブランドでも到達可能なレベル**（参考: THE MATCHA TOKYO 表参道 推定100-200杯/日）
3. **最大のリスクは「立ち上がり期間の現金枯渇」と「客数の不確実性」**。運転資金6ヶ月分の確保が必須
4. **アラビカ嵐山級（2,000杯/日）は東京の通常立地では実現不可能**。嵐山店は年間550万人の観光地という極端な立地の産物
5. 東京で勝負するなら **「観光地立地（浅草・銀座・原宿）」 vs 「カルチャー立地（中目黒・代官山・表参道）」** の戦略選択が決定的

## 7. アラビカ嵐山店との対比 — なぜ東京で2,000杯/日は出ないのか

### 7.1 嵐山店の特殊性

| 要因 | 嵐山店 | 東京一等地（表参道等） |
|---|---|---|
| 立地客流量 | 嵐山年間550万人観光客 | 表参道 1日約20万人通行 |
| 顧客の購買モード | 観光・SNS投稿目的（高単価OK） | 日常使い・カフェ巡り |
| 競合密度 | 嵐山に高品質コーヒー店ほぼ無し | スペシャルティカフェ激戦区 |
| 滞在時間 | テイクアウトのみ（高回転） | イートイン需要あり（回転低下） |
| 背景アセット | 渡月橋・嵐山・桂川（無料の世界遺産級背景） | 自前で内装で作る必要あり |
| インバウンド比率 | 60-70% | 20-30% |
| 季節変動 | 春・秋激高、冬激低 | 年間平準 |

### 7.2 物理的キャパシティ

仮に2,000杯/日を10坪店舗で出すには:
- 11時間営業 → 1時間あたり182杯 = **20秒に1杯**
- これを実現するには3-4人が並列で点て続ける専用オペレーション
- 嵐山店は実際にこのオペで成立しているが、**東京で同じ需要を生み出せる立地は浅草・銀座中心部の一部のみ**

### 7.3 東京で「嵐山級」を狙うなら

検討に値する立地候補:
- **浅草雷門周辺**: 訪日客密度は嵐山に次ぐ。嵐山型モデルが最も近い形で機能する可能性
- **銀座中央通り**: 高単価客の密度が極めて高い
- **明治神宮周辺**: 観光・地元の混合
- **新宿御苑南口**（[decided_cafe.md](../../decided_cafe.md) のピーマン物件）: 観光地アセットあり、家賃想定よりは安いはず

ただし、これらは家賃が¥800K-1.5M/月になる可能性が高い。**家賃倍増 vs 客数倍増のトレードオフ**を別途試算する必要あり。

## 9. 結論と推奨アクション

### 9.1 結論

1. **モデルとしては成立する**: 標準シナリオ（400杯/日）が達成できれば、年間¥40Mの営業利益を生む高収益事業
2. **損益分岐点（152杯/日）は新店でも到達可能**だが、立ち上がり3-6ヶ月の現金管理が生死を分ける
3. **総資金需要¥36M**は決して小さくないが、この規模感の事業としては妥当
4. **アラビカ嵐山級（2,000杯/日）は通常の東京立地では再現不可能**。観光地立地（浅草等）でのみ検討に値する
5. **「立地に妥協しない」は財務的にも正解**。家賃が¥30万増えても利益率3ptしか落ちないが、客数が半減すれば利益は1/3になる

### 9.2 数字で見る「絶対に妥協できないこと」

| 妥協しないべきもの | 理由（数字） |
|---|---|
| **立地（1F路面店）** | 2F/B1Fは1Fの50-60%の客数。利益は1/3以下になる |
| **内装の質** | 「映え」が客数を決める。坪単価¥600K未満ではアラビカ水準に届かない |
| **客単価** | ¥850→¥700に下げると利益率35%→25%に悪化 |
| **オペレーション速度** | 1時間14杯以上を出せないと損益分岐に届かない |
| **運転資金6ヶ月分** | 立ち上がり期の現金枯渇=即倒産 |

### 9.3 次のアクション

**物件交渉を始める前にやるべきこと:**

1. **物件候補の家賃実勢を3-5件調べ、家賃感度を再計算**
   - 表参道・中目黒・代官山・浅草・銀座の路面店の実勢家賃
2. **アラビカ・THE MATCHA TOKYO・Sacya の実店舗観察**
   - 1時間あたりの客数を実カウント
   - 客単価をレジ観察で推定
3. **内装の坪単価見積もりを2-3社から取る**
   - 「アラビカクオリティを目指す」と明示
4. **抹茶原料の卸価格を3農家から取る**
   - 標準シナリオの月商で必要な月間抹茶量: 約30-50kg
5. **資金調達ルートの整理**
   - 自己資金 / 日本政策金融公庫 / 銀行融資 / エンジェル
6. **撤退基準を事前文書化**
   - 6ヶ月・12ヶ月の判定基準

## 10. 想定質問への回答

**Q1. なぜ客単価¥850？もっと高くできないか？**
A. ¥1,000-1,200に上げることは可能だが、客数が15-25%減るリスクがある。標準シナリオでは¥850×400杯/日のほうが¥1,100×320杯/日より利益が大きい。価格戦略は開業後の実数を見て調整。

**Q2. 9坪なら家賃が安いのでは？**
A. 9坪と10坪の家賃差は¥35K-50K程度。固定費全体に対する影響は小さい。坪数を削るより**1F路面・立地グレードを死守する**ほうが財務的に正解。

**Q3. 内装に¥7M（設計込み）は本当に必要か？**
A. 「映え」と「品質感」の両立がブランドの全て。中途半端な内装で開業すると客数が半減し、半年で店を閉じることになる。内装に¥3-4Mで節約しても、客数が半減すれば利益は¥3M/月減る。**節約はせず、最大化に投資する**ほうがROIが高い。

**Q4. 強気シナリオの700杯/日は現実的か？**
A. 表参道・原宿のトップカフェの実勢から見れば達成可能水準。ただし新規ブランドが1年以内に達成するのは難しく、**2-3年かけて到達する目標**として位置付けるべき。1年目は標準（400杯/日）を目指す。

**Q5. 2号店出店のタイミングは？**
A. 1号店の月商が¥9M超で安定（=標準シナリオ達成）し、累積CFが¥30M以上溜まったタイミング。早ければ開業2年目末、現実的には3年目。
