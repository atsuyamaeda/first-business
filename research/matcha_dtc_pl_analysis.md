## 第三者レビュー：このモデルの弱点と盲点

### ✅ 成立している点
1. **利益構造は健全**: 変動利益率41.6%は十分。Amazon手数料33%を吸収できている
2. **初期投資が小さい**: ¥630,000で市場テストできる。失敗しても致命傷にならない
3. **損益分岐点が低い**: 200個（全体の40%）で固定費を回収
4. **関税免除**: 日本産緑茶は2025年11月に米国関税が免税化。追い風

### ⚠️ 楽観的すぎる前提
1. **「500個が9ヶ月で売れる」前提**: レビュー50件あっても、Amazon抹茶カテゴリでJade Leaf（80,000レビュー）やNaoki（6,000レビュー）と並ぶ棚で月56個は簡単ではない。PPC広告なしの場合、オーガニック検索での露出は限定的
2. **返品率3%**: 実際には「味が想像と違う」で5-8%になる可能性。5%なら利益は¥330,000に低下
3. **碾茶¥12,000/kg**: 2025年に碾茶価格が倍増している。¥12,000/kgで安定調達できるかは農家/卸業者との交渉次第
4. **為替¥150/$1**: 円安に振れれば有利、円高に振れれば不利。¥130/$1なら売上が目減り

### ❌ このモデルに含まれていないコスト
1. **自分の人件費**: 商品開発、出品、カスタマーサポート、SNS運用の時間コスト
2. **サンプル取り寄せ費用**: 農家/卸業者から品質確認用サンプルを複数取り寄せる費用（¥10,000-30,000）
3. **不良品・破損**: 輸送中の缶の凹み、粉漏れ等。2-5%のロスを想定すべき
4. **Amazon広告の必要性**: SNS+知人で50レビュー獲得後も、カテゴリ内でのランキング維持にはPPC広告が必要になる可能性が高い
5. **2回目の仕入れタイミング**: 500個が売れる前に次のロットを発注する必要があり、キャッシュが二重に必要になる

### 🔍 検証が必要な最重要仮定

| 仮定 | 検証方法 | 優先度 |
|------|---------|--------|
| 碾茶¥12,000/kgで仕入れ可能か | B2B卸業者に見積もり依頼 | **最優先** |
| ¥3,300（$22）で売れるか | Naoki/Jade Leafの実売価格と差別化軸の確認 | 高 |
| SNS+知人で50レビュー獲得可能か | 知人の購入意欲確認、SNSフォロワー数の棚卸し | 高 |
| FDA/FSVP対応のコストと手間 | 米国食品輸入の代行業者に相談 | 中 |

## 結論

| 指標 | 数値 |
|------|------|
| 必要資金 | **¥630,000** |
| 初回500個の見込み利益 | **¥363,000**（利益率22.7%） |
| 2回目以降500個の見込み利益 | **¥518,000**（利益率32.4%） |
| 損益分岐点 | **200個（9ヶ月で40%売れれば黒字）** |
| 最大リスク | 碾茶高騰（¥22,500/kg）+ 値下げ（¥2,700）= 赤字 |

**前回試算の¥708,500は約49%過大。現実的な利益は¥363,000。**
ただし2回目以降は¥518,000に改善し、ビジネスモデルとして成立する水準。

最初の一歩は**碾茶の見積もり取得**。¥12,000/kgで30g缶が作れることが確認できれば、¥630,000でこのテストを始められる。
