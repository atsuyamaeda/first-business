# 抹茶DTC 差別化フレームワーク

作成日: 2026年3月22日

---

## 13カテゴリの差別化要素

| # | 要素 | 概要 |
|---|------|------|
| 1 | 産地・グレード・シングルオリジン | 宇治・西尾等の産地ブランド、テロワール訴求 |
| 2 | 有機・クリーンラベル認証 | USDA/JAS認証。米国オーガニックティー市場は約3%だが成長中（※原案の45%は不正確） |
| 3 | 科学的エビデンス（機能性） | 主要3成分: L-テアニン・EGCG・カフェイン。副次的成分: ルテイン・クロロフィル |
| 4 | サプライチェーン透明性 | 直接取引、ブロックチェーントレーサビリティ |
| 5 | ブランドストーリー・ヘリテージ | 茶道文化、創業者ストーリー、職人技 |
| 6 | プロダクトイノベーション | 専用デバイス（Cuzen型）、RTD、即溶タイプ |
| 7 | サブスク・リテンション | キュレーション型定期便、リワード、会員限定品 |
| 8 | 顧客体験・アンボクシング | パッケージ演出、スターターキット、レシピ同梱 |
| 9 | カテゴリー横断展開 | コスメ、サプリ、食品、アルコール、CBD |
| 10 | コミュニティ・教育 | SNS動画、インフルエンサー、ワークショップ |
| 11 | サステナビリティ | 環境配慮包装、カーボンニュートラル、B Corp |
| 12 | 価格戦略 | プレミアム vs アクセシブル、ティアード設計 |
| 13 | オムニチャネル統合 | DTC EC＋ポップアップ＋B2B卸の段階展開 |

### 原案からの修正点
- **旧カテゴリ3「科学的エビデンス」と旧カテゴリ4「機能性」を統合** — 内容が完全に重複していたため1つに
- **「有機が市場の45%」→「約3%」に修正** — 米国ティー市場全体のオーガニック比率は約3%（Grand View Research）。45%は特定セグメントの数値の可能性あり
- **「ルテイン」を主要成分から副次成分に格下げ** — 抹茶に含まれるが、主要3成分はL-テアニン・EGCG・カフェイン

## 5レイヤーモデル（上ほど模倣困難）

```
コミュニティ・文化        ← 最も模倣困難
体験・サービス設計
ブランド・ストーリー
機能・品質の証明
プロダクト               ← 最も模倣容易
```

### 13カテゴリ → 5レイヤーへのマッピング

| レイヤー | カテゴリ |
|---------|---------|
| コミュニティ・文化 | 10（コミュニティ・教育） |
| 体験・サービス設計 | 7（サブスク）、8（顧客体験）、13（オムニチャネル） |
| ブランド・ストーリー | 5（ヘリテージ）、11（サステナビリティ） |
| 機能・品質の証明 | 2（有機認証）、3（科学的エビデンス）、4（サプライチェーン透明性） |
| プロダクト | 1（産地・グレード）、6（プロダクトイノベーション）、9（カテゴリー横断） |
| **横断要素** | 12（価格戦略）— 全レイヤーに影響するため特定レイヤーに帰属しない |

### 注記
- プロダクト層に3カテゴリが集中するのはDTCの性質上妥当（プロダクトの差別化軸が多い）
- 価格戦略はレイヤー横断的な要素として別扱い
- レイヤーの順序は概ね妥当。ただし「ブランド・ストーリー」の模倣困難度は深さによる（一保堂の300年 vs 新興ブランドの創業ストーリー）
