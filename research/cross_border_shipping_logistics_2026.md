# D2C抹茶ブランド 海外向け配送・関税・決済・コスト調査レポート
**調査日: 2026年3月22日**

---

## 1. 日本→米国の配送手段と現状（2026年3月時点）

### 1-1. EMS / SAL / 小型包装物の米国向けステータス

| サービス | ステータス | 詳細 |
|---------|----------|------|
| **EMS（物品）** | **一時停止中** | 2025年8月27日より、「消費を目的とする販売品」を含む米国宛て郵便物の引受けを一時停止。2026年3月時点で再開の公式発表なし |
| **SAL** | **全面停止中** | SAL扱いの郵便物は米国向け全面停止 |
| **小型包装物** | **一時停止中** | 販売品を含む場合は引受け停止。個人間の贈答品（内容品価格100ドル以下）のみ受付可能 |

**停止の背景**: 米国のde minimis（少額免税枠）ルール廃止に伴い、すべての輸入品が関税対象となったため、日本郵便が対応を一時停止。

### 1-2. De Minimis（$800以下の免税枠）の現在の状況

| 項目 | 内容 |
|------|------|
| **ステータス** | **完全に撤廃・停止中**（2026年3月時点） |
| **施行日** | 2025年8月29日（東部時間午前0時1分以降の通関分から適用） |
| **根拠** | 大統領令14324（国家安全保障上の理由） |
| **2026年2月の追加措置** | 大統領令により、de minimis免除停止の継続を再確認。2026年2月24日以降、従価税方式のみ適用（定額税$80-$200の選択肢は廃止） |
| **最高裁判決** | 2026年2月の最高裁判決でIEEPA関税に疑義が呈されたが、de minimis停止自体は維持 |
| **復活の見通し** | 現時点で復活の兆候なし |

**D2Cへの影響**: 以前は$800以下の注文は関税フリーだったが、現在はすべての荷物に関税がかかる。抹茶のような低単価商品でも関税負担が発生。

### 1-3. UGX（ゆうグローバルエクスプレス）

| 項目 | 内容 |
|------|------|
| **サービス名** | UGX越境EC配送サービス（新サービス） |
| **開始日** | 2026年3月2日 |
| **対応地域（米国）** | 米国・カナダ：2026年3月2日開始 |
| **対応地域（欧州）** | 英・伊・蘭・瑞・西・独・仏：準備完了後に展開予定 |
| **対応地域（アジア）** | 準備完了後に展開予定 |
| **特徴** | 配達先を個人あてに限定し、UGX公表運賃から割引した特約運賃を設定 |
| **関税対応** | **DDP（関税元払い）に対応**（従来の着払いに加えて選択可能） |
| **料金（概算）** | 具体的な公開料金表は未確認。法人契約で個別見積もりが基本。EMS停止の代替として注目 |

> **注目ポイント**: DDP対応は越境ECにとって大きなメリット。購入者に関税負担をかけず、スムーズな購買体験を提供できる。

## 3. 関税・輸入規制

### 3-1. 米国

| 項目 | 内容 |
|------|------|
| **HSコード** | 0902.10（緑茶、3kg以下の包装）/ 0902.20（その他の緑茶） |
| **基本関税率** | 一般：6.4%。日本産：**0%**（2025年11月13日の大統領令2025-17507により撤廃） |
| **De minimis撤廃後の影響** | 金額にかかわらず通関手続きが必要。ただし日本産緑茶/抹茶は0%税率のため、**関税自体は発生しない** |
| **FDA登録** | **必要**。FSMA（食品安全近代化法）に基づき、食品施設のFDA登録が義務。出荷ごとに事前通知（Prior Notice）も必要 |
| **必要書類** | FDA施設登録、事前通知、原産地証明書、植物検疫証明書、残留農薬検査報告書（第三者機関）、USDA有機認証（オーガニック表記する場合） |
| **IEEPA関税** | 2026年2月の最高裁判決でIEEPA関税は違法と判断。緑茶への15%相互関税は撤廃済み |

> **朗報**: 日本産抹茶は現在、米国への関税率0%。ただし通関手続き・FDA登録等のコンプライアンスコストは依然必要。

### 3-2. 欧州（EU）

| 項目 | 内容 |
|------|------|
| **HSコード** | 0902.10.00（緑茶、3kg以下）/ 0902.20.00（その他の緑茶） |
| **EU一般関税率** | 3.2%（MFN税率） |
| **日EU EPA適用時** | **0%**（原産地証明書が必要） |
| **残留農薬基準** | EU基準は日本より**極めて厳格**。特定農薬のMRL（残留基準値）が設定されていない場合、デフォルト許容値は**0.01ppm**（実質ゼロ） |
| **2026年の規制強化** | チアメトキサム・クロチアニジンのMRL規制が2026年3月7日から適用。茶・コーヒーにも影響 |
| **EU適合品の必要性** | EU輸出専用に栽培された「EU-Compliant」抹茶の調達が事実上必須 |
| **VAT** | 各国のVAT率が適用（ドイツ19%、フランス5.5%（食品軽減税率）、イギリス0%（基本食品）等） |

> **注意**: EU向けは関税よりも**残留農薬基準のクリア**が最大のハードル。EU基準に適合した茶葉の調達ルートを確保する必要あり。

### 3-3. DDP vs DDU

| 方式 | 内容 | 推奨度 |
|------|------|--------|
| **DDP（Delivered Duty Paid）** | 売り手が関税・税金を負担。購入者は追加支払いなし | **強く推奨** |
| **DDU（Delivered Duty Unpaid）** | 購入者が関税・税金を配達時に支払い | 非推奨（税関滞留リスク、顧客体験悪化） |

> De minimis撤廃後、DDUでは税関での滞留・受取拒否のリスクが大幅に上昇。D2CブランドはDDP一択。UGX越境EC配送サービスがDDP対応しているのは大きなメリット。

## 5. 海外向けEC特有のコスト

### 5-1. 英語サイト構築・翻訳コスト

| 方法 | コスト | 品質 | 備考 |
|------|------|------|------|
| **Shopify多言語機能（Translate & Adapt）** | 無料（Shopify標準機能） | 中 | 自動翻訳＋手動修正。最低限の多言語対応 |
| **AI翻訳（DeepL等）+ 人間チェック** | 5〜15万円 | 中〜高 | 商品説明・ブランドストーリー等。50〜100ページ規模 |
| **プロ翻訳会社** | 20〜50万円 | 高 | ブランドトーン&マナーの統一。ネイティブチェック込み |
| **WOVN.io等の多言語化SaaS** | 月額5,500円〜 | 中〜高 | 自動翻訳＋辞書管理。継続的な翻訳更新に強い |

> **推奨**: 初期はShopify標準機能 + DeepL翻訳 + ネイティブ校正（クラウドソーシング）で10〜20万円程度。ブランドストーリー・About等の重要ページのみプロ翻訳。

### 5-2. カスタマーサポート（英語対応）

| 方法 | コスト | 備考 |
|------|------|------|
| **自社対応（テンプレート化）** | 実質0円（人件費のみ） | FAQ充実＋定型文で中学英語レベルでも対応可能 |
| **Gorgias等のCS SaaS** | 月額$10〜60 | Shopify連携。自動応答・テンプレート機能 |
| **バイリンガルVA（バーチャルアシスタント）** | 月額5〜15万円 | フィリピン等の英語対応VA。Upwork/Fiverr経由 |
| **CS代行サービス** | 月額20〜50万円 | フル外注。規模が大きくなってから |

> **推奨**: 初期は自社対応（テンプレート化）+ Gorgias（月額$10）。月間100件超のCS対応が必要になったらVA雇用を検討。

### 5-3. 返品・交換

| 項目 | 内容 |
|------|------|
| **食品の返品** | 開封済み食品の返品は**原則受け付けない**ポリシーが業界標準 |
| **国際返送** | 非現実的（返送料が商品価格を上回る。500gで8,000〜10,000円） |
| **推奨ポリシー** | 未開封・不良品のみ全額返金（返送不要で返金）。誤配・破損はフォト確認で返金 |
| **返品率** | 海外ECの平均返品率は20〜30%（アパレル）。食品は5%未満が一般的 |
| **コスト見込み** | 売上の1〜2%（食品D2C、返送不要返金ポリシーの場合） |

> **重要**: 食品D2Cでは「返送不要・返金対応」が業界標準。国際返送は現実的でなく、不良品は写真確認のみで返金処理するのがベストプラクティス。

## 7. 総合シミュレーション（抹茶30g缶 1個を米国に直送する場合）

| コスト項目 | 金額（円） | 金額（USD） | 備考 |
|-----------|----------|------------|------|
| 商品原価 | 500円 | $3.1 | 宇治産抹茶30g |
| パッケージ | 200円 | $1.3 | 缶＋化粧箱 |
| 配送料（DHL） | 8,000円 | $50.3 | 500g以下・最安見積 |
| 関税 | 0円 | $0 | 日本産緑茶は0%（2025年11月〜） |
| 決済手数料（5.5%） | 約275円 | $1.7 | Shopify Payments海外カード＋為替変換 |
| Ship&co手数料 | 30円 | $0.2 | 送り状1件 |
| **合計コスト** | **約9,005円** | **約$56.6** | |
| 販売価格（想定） | 5,000円 | $31.4 | 米国市場の相場 |
| **損益** | **-4,005円** | **-$25.2** | **赤字** |

> **結論**: 単品直送モデルは配送料が商品価格を大幅に上回り、**経済的に成立しない**。以下の対策が必須：
> 1. **セット販売**（3〜5缶セットで$80〜120、配送料を按分）
> 2. **サブスクリプション**（定期便で配送効率を最適化）
> 3. **米国フルフィルメント拠点の確保**（Ippodo/Kettlモデル。ShipBob等の3PL活用）
> 4. **UGX越境EC配送サービス**の特約運賃を活用（法人契約で大幅割引の可能性）

## 8. 戦略的示唆

### Phase 1（立ち上げ期: 0〜6ヶ月）
- **配送**: UGX越境EC配送サービス（DDP対応）またはDHL Express（法人契約）
- **最低注文額**: $60〜80以上に設定（配送料の按分効率化）
- **決済**: Shopify Payments + PayPal
- **英語サイト**: Shopify多言語機能 + DeepL + ネイティブ校正（初期コスト15万円程度）
- **CS**: 自社テンプレート対応 + Gorgias
- **返品**: 返送不要返金ポリシー

### Phase 2（成長期: 6〜18ヶ月）
- **米国3PL拠点確立**（ShipBob等）。日本から米国倉庫にバルク輸送→国内配送
- **$150以上で送料無料**（Kettlモデル）
- **欧州展開**: EMS + DHL。EU適合抹茶の調達確保
- **バイリンガルVA雇用**

### Phase 3（拡大期: 18ヶ月〜）
- **米国内配送のみ**（完全現地フルフィルメント）
- **欧州フルフィルメント拠点**
- **CS代行サービス導入**
