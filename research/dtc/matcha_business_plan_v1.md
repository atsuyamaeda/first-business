### 撤退ライン

| 状況 | 判断 |
|------|------|
| 碾茶¥22,500/kg以上で仕入れ不可 | → 参入しない |
| 3ヶ月で50個未満しか売れない | → 残り在庫を値下げ処分、撤退検討 |
| 返品率が8%を超える | → 品質問題。商品改善 or 撤退 |
| レビュー平均3.5星以下 | → 品質問題。即座に原因究明 |

## 8. 実行チェックリスト

### Phase 0: 検証（今すぐ〜2週間）
- [ ] B2B卸業者に碾茶の見積もり依頼（¥12,000/kg以下で調達可能か確認）
- [ ] サンプル取り寄せ・品質確認（3-5社から比較）
- [ ] FDA/FSVP対応の代行業者に相談・見積もり
- [ ] Amazon Seller Centralアカウント作成

### Phase 1: 準備（2週間〜1ヶ月）
- [ ] 碾茶の仕入れ先確定・発注
- [ ] 缶パッケージ発注（中国製食品用缶）
- [ ] 英語栄養表示ラベルデザイン・入稿
- [ ] FDA米国代理人契約
- [ ] FSVP策定
- [ ] 商品写真撮影

### Phase 2: 製造・発送（1〜2ヶ月目）
- [ ] 充填・密封・殺菌
- [ ] 航空便で米国へ発送
- [ ] 通関手続き
- [ ] FBA倉庫へ納品

### Phase 3: 販売開始（2ヶ月目〜）
- [ ] Amazon出品ページ公開
- [ ] SNSで告知開始
- [ ] 知人に購入＋レビュー依頼（目標50件）
- [ ] 売上・レビュー・返品率のモニタリング開始
