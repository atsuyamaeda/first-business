## 統合データベース

### 凡例

- **形態**: S = スティック個包装 / C = 缶+茶杓/スクープ一体型 / T = ツール（空容器）
- **市場**: JP = 日本 / US = 米国 / UK = 英国 / EU = 欧州 / AU = 豪州 / CA = カナダ / KR = 韓国 / GL = グローバル
- **添加物**: ◯ = 純粋抹茶（無添加） / △ = 微量添加物 / × = 砂糖・ミルク等あり
- **ダマ対策**: 茶こし不要の訴求や仕組みがある場合に記載
- **在庫**: ◯ = 販売中 / △ = 一部欠品 / × = 売切
