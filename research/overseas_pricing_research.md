## 4. 「日本から直送」のプレミアム

### 海外市場での日本産抹茶のプレミアム

- **一般的なマークアップ**: 日本国内価格の**2倍以上**で海外販売されるケースが多い
  - 出典: 複数の業界レポートが「日本国外では日本と比べて2倍以上の価格で販売される」と指摘
- **卸売価格 vs 小売価格のギャップ**:
  - 卸売（カフェチェーン向け）: 3,000-5,000円/kg ($20-34/kg)
  - ミドルマーケット（オーガニック）: 6,000-9,000円/kg ($40-61/kg)
  - 欧州プレミアムオーガニック: 10,000円+/kg ($67+/kg)
  - 小売（消費者向け30g缶）: $20-40 → **$667-1,333/kg 換算**

### 「Direct from Japan」のプレミアム要素

| 要素 | 推定プレミアム | 根拠 |
|------|-------------|------|
| 産地証明（宇治、西尾等） | +10-20% | 産地名のブランド価値 |
| 鮮度（挽きたて直送） | +15-30% | Matcha Botanicals等のモデル |
| ストーリー（農家直結） | +10-20% | Kettl、Ippodoの価格帯比較 |
| 総合「Made in Japan」プレミアム | +20-50%（推定） | 米国拠点ブランドとの比較 |

**注**: 明確な消費者調査データは見つからなかったが、以下の価格比較から推定:
- 米国拠点ブランド（Jade Leaf Ceremonial 30g = $22）
- 日本直送ブランド（Ippodo Kan 20g = $22 → 30g換算$33、50%プレミアム）
- フランス直送モデル（Matcha Botanicals 40g = €40-60 → 極めて高いプレミアム）

## 5. 海外配送コスト

### 重要: 日本→米国の配送状況（2025年8月〜）

**Japan Post（EMS/SAL/小型包装物）→ 米国は事実上停止中**

- 2025年8月27日より、米国向けの小型包装物・小包・EMS（商品）の引受を一時停止
- 原因: 米国政府が de minimis 免税（$800以下免税）を撤廃する大統領令を発令（2025年7月30日）
- $100超の商品が関税・通関手続きの対象に
- **SAL（エコノミー航空便）**: 全面停止中（全国向け）
- **復旧時期**: 未定

### 代替配送手段

| 配送手段 | 日本→米国（500g） | 日本→欧州（500g） | 所要日数 | 備考 |
|---------|------------------|------------------|---------|------|
| Japan Post EMS | **停止中** | 推定 3,000-4,000円 | 3-5日 | 欧州向けは利用可（第3地帯） |
| Japan Post UGX | 推定 4,000-6,000円 | - | 3-5日 | 米国向け代替。通関対応可 |
| DHL Express | 推定 4,000-8,000円 | 推定 4,000-7,000円 | 1-3日 | 500gの場合。容積重量に注意 |
| FedEx | 推定 4,000-8,000円 | 推定 4,000-7,000円 | 1-3日 | |
| ePacket Light | 推定 1,500-2,500円 | 推定 1,500-2,500円 | 7-14日 | 小型・低額商品向け |

**注**: 上記はいずれも推定値。実際の料金は重量・サイズ・契約条件により異なる。

### 米国向けD2Cの実務的選択肢

1. **日本から直送**: DHL/FedEx/UGXを利用。送料$25-50/件（推定）→ 高額のため「一定額以上送料無料」で吸収するか、送料を商品価格に転嫁
2. **米国倉庫を利用**: Ippodoモデル。まとめて米国に輸出し、国内配送。スケーラブルだが在庫リスク
3. **3PLサービス利用**: ShipBob等のフルフィルメントサービス活用

## 7. 総合分析: プライシング戦略への示唆

### 価格ポジショニングマップ（30gセレモニアルグレード基準）

```
超プレミアム  $40+     │ Matcha Botanicals, Ippodo Ummon級
              │
プレミアム    $28-40   │ Golde ($39), Anatae (€36.90), Kettl
              │
ミドル        $18-28   │ Jade Leaf Ceremonial ($22), Chamberlain ($22), MatchaBar ($20)
              │
エントリー    $10-18   │ PerfectTed (£10.50), Jade Leaf Culinary ($10)
```

### 推奨プライシングレンジ（日本直送D2C）

| 戦略 | 30g価格 | 1gあたり | ターゲット |
|------|--------|---------|-----------|
| プレミアム路線 | $32-38 / €30-36 | $1.07-1.27 | 品質重視・体験重視層 |
| ミドルハイ路線 | $25-32 / €23-30 | $0.83-1.07 | Jade Leaf Ceremonialからのアップグレード層 |

### 根拠
- 「日本直送」プレミアム（+20-50%）を活かしつつ、Ippodo/Kettlほどの超プレミアム帯には入らない
- 1杯あたり$2-3で「カフェの半額以下」を訴求可能
- サブスクで10-15%OFF → 月額$25-35がスイートスポット（推定）

### リスク要因
1. **米国向け配送の不確実性**: Japan Post停止中、DHL/FedEx高コスト
2. **抹茶原料価格の高騰**: 2025年に50-170%上昇、今後も不透明
3. **関税・通関**: 米国 de minimis 撤廃により追加コスト発生
4. **為替リスク**: USD/JPY 159円台は円安水準だが変動リスクあり
