### 1-4. 産地への直接訪問

最も確実な方法。以下の手順を推奨：

1. **産地の選定**: 自社ブランドのポジショニングに合った産地を選ぶ
   - プレミアム路線 → 京都宇治、福岡八女
   - コスパ重視 → 鹿児島、静岡
   - ストーリー性 → 特定の品種（さみどり、おくみどり等）を栽培する農家
2. **茶業会議所に事前連絡**: 訪問希望を伝え、見学可能な農家・工場を紹介してもらう
3. **新茶シーズン（4-5月）に訪問**: 碾茶の製造工程を実際に見られる最良の時期
4. **通訳の手配**: 日本語でのコミュニケーションが必須。英語対応可能な農家は極めて限定的

## 2. 最初のコンタクトの取り方

### 2-1. アプローチの基本原則

日本の茶農家・製茶工場との取引開始で最も重要なのは「信頼関係の構築」。いきなり大量発注の話をするのではなく、段階的にアプローチする。

### 2-2. 推奨コンタクト手順

**Step 1: 自己紹介メール/電話**
- 自社の紹介（会社概要、ブランドコンセプト、ターゲット市場）
- なぜその農家/工場に関心を持ったか（具体的な理由）
- 取引の規模感（最初は小ロットから始めたい旨を正直に伝える）
- 工場見学・サンプル提供の依頼

**Step 2: サンプル依頼**
- 有料サンプルが一般的（1,000〜5,000円程度）
- 複数グレードのサンプルを依頼（セレモニアル、プレミアム、カリナリー等）
- サンプルの評価基準を事前に明確にしておく

**Step 3: 産地訪問**
- 対面での挨拶は日本のビジネス文化で極めて重要
- 製造工程を見学し、品質管理体制を確認
- 可能であれば茶畑も訪問

**Step 4: 小ロット試験注文**
- 初回は5〜30kgの小ロットから開始
- 品質・納期・コミュニケーションの確認

### 2-3. コンタクト時の注意点

- **日本語で連絡する**: 英語メールは無視されやすい。最低限、日本語の丁寧なメールを送る
- **名刺を用意する**: 日本語の名刺があると印象が良い
- **紹介経由が最強**: JETROや茶業会議所、既存取引先からの紹介があると話が早い
- **長期的関係をアピール**: 「1回きりの取引ではなく、長期的なパートナーシップを希望」と伝える
- **抹茶への理解を示す**: 碾茶（てんちゃ）の製造工程、品種の違い、産地の特性について基礎知識があることを示す

## 3. 商談で確認すべきポイント

### 3-1. 品質関連

| 確認項目 | 具体的な質問 |
|----------|-------------|
| **品種** | 使用している茶品種は何か（さみどり、おくみどり、やぶきた、あさひ等） |
| **被覆期間** | 碾茶用の被覆（覆い下栽培）は何日間か（20日以上が理想） |
| **石臼挽き vs ボールミル** | 抹茶の挽き方。石臼挽きはプレミアム訴求に重要 |
| **色・味・香りのスペック** | L値（明度）、a値（色相）の数値。テイスティングノート |
| **残留農薬検査** | 検査の頻度と基準（日本基準 vs EU基準 vs 米国EPA基準） |
| **重金属検査** | 鉛・カドミウムの検査結果。米国向けはCalifornia Prop 65に注意 |
| **放射能検査** | 産地によっては海外バイヤーから求められる |
| **有機認証** | JAS有機、USDA Organic、EU Organic等の認証有無 |
| **賞味期限** | 通常は製造から1年。密封アルミパック保存が前提 |

### 3-2. 取引条件関連

| 確認項目 | ポイント |
|----------|---------|
| **最小ロット（MOQ）** | 製茶工場のMOQは通常10〜100kg。農家直の場合はさらに少量可能な場合も |
| **価格体系** | kg単価。グレード別・注文量別の価格テーブル。為替リスクの負担（円建て vs ドル建て） |
| **支払条件** | 前払い/後払い/L/C（信用状）。初回取引は前払いが一般的 |
| **リードタイム** | 注文から出荷までの日数。新茶シーズン（5-6月）は混み合う |
| **パッケージング** | バルク出荷（業務用袋）vs リテールパッケージング。窒素充填の対応可否 |
| **OEM/ODMの可否** | 自社ブランドでのパッケージング対応。最小ロットと追加費用 |
| **年間供給能力** | スケールアップ時に安定供給が可能か。複数農家からの調達体制 |
| **サンプル対応** | サンプルの費用と納期 |

### 3-3. 輸出関連

| 確認項目 | ポイント |
|----------|---------|
| **輸出実績** | 海外への出荷経験があるか。どの国向けの実績があるか |
| **輸出書類** | 原産地証明書、植物検疫証明書、成分分析表の発行可否 |
| **通関対応** | FOB/CIF/EXWなどの貿易条件。輸出通関の代行可否 |
| **食品安全認証** | FSSC22000、ISO22000、HACCPなどの認証取得状況 |
| **米国FDA対応** | 米国向けの場合、FDA施設登録、FSVP（Foreign Supplier Verification Program）への対応 |

## 4. 小ロットから始める際の注意点

### 4-1. 現実的な最小ロットの目安

| 取引先タイプ | 最小ロット目安 | 備考 |
|-------------|---------------|------|
| 茶農家（直接） | 1〜5kg | 最も柔軟。ただし品質のばらつきリスクあり |
| 小規模製茶工場 | 5〜20kg | 比較的柔軟に対応してくれる場合が多い |
| 中規模製茶工場 | 20〜50kg | 標準的なMOQ |
| 大規模製茶工場/問屋 | 50〜100kg+ | MOQが高い代わりに品質安定・書類対応が充実 |
| OEMメーカー | 100〜500kg+ | リテールパッケージ込みの場合はさらに高くなる |

### 4-2. 小ロット取引のリスクと対策

| リスク | 対策 |
|--------|------|
| **kg単価が高くなる** | 初期は利益率を犠牲にしてでも品質確認を優先。売上拡大に伴い段階的に値下げ交渉 |
| **安定供給の保証がない** | 複数の農家/工場と関係を構築。メイン+バックアップ体制 |
| **品質のばらつき** | ロットごとにサンプルチェック。品質基準書を共有し、基準外はリジェクトする取り決めを事前に行う |
| **輸出対応のノウハウ不足** | 輸出実績のある工場を優先。またはJETROの輸出支援サービスを活用 |
| **最低出荷量に達しない** | 他の小規模バイヤーと合同出荷（consolidated shipping）を検討。またはYunomi等のプラットフォーム経由 |

### 4-3. 段階的スケーリングのロードマップ

```
Phase 1（0-6ヶ月）: テスト期
  - 3〜5社からサンプル取り寄せ
  - 5〜10kgの試験注文
  - 品質評価、パッケージングテスト
  - 顧客フィードバック収集

Phase 2（6-12ヶ月）: 初期取引期
  - 1〜2社に絞り込み
  - 月間20〜50kgの定期注文
  - OEM対応の検討開始

Phase 3（1-2年目）: スケール期
  - 月間100kg+の安定注文
  - 年間契約の締結
  - 品種指定・シングルオリジン展開
  - パッケージOEMの本格化
```

## 5. OEM/ODM対応の製茶工場の探し方

### 5-1. OEMとODMの違い（茶業界の場合）

- **OEM**: バイヤーの仕様（品種、グレード、ブレンド比率等）に基づき、工場が製造・パッケージング
- **ODM**: 工場が製品設計（ブレンド開発、パッケージデザイン含む）まで行い、バイヤーのブランドで販売

### 5-2. OEM対応が確認されている主要企業

既存リサーチ（matcha_supply_map.md）で確認済みのOEM/ホワイトラベル対応企業：

| 企業名 | 本拠地 | 特徴 |
|--------|--------|------|
| **あいや（AIYA）** | 愛知・西尾 | 世界最大級の抹茶メーカー。B2B/OEM実績豊富。FDA対応済。MOQは比較的高め |
| **共栄製茶（森半）** | 京都・宇治 | 1836年創業。OEM/PB対応。国内外の大手食品メーカーへの供給実績 |
| **Yunomi** | 東京 | 日本茶のマーケットプレイス。OEM/ホワイトラベルプログラムあり。小ロット対応可能 |
| **KEIKO** | ドイツ（日本から直接調達） | EU市場向けOEM対応 |
| **Matchaeologist** | ロンドン（日本から直接調達） | ホワイトラベルプログラムあり |
| **Matcha Maiden** | メルボルン（日本から調達） | OEM/ホワイトラベル対応 |

### 5-3. OEM工場を探す具体的な方法

1. **JETROの食品OEMマッチング**: JETROの各国事務所に「日本産抹茶のOEM製造先を探している」と相談。無料でマッチング支援を受けられる

2. **FOODEX JAPANでのOEM工場探し**: FOODEX会場には「OEM/PB対応可」を掲げる出展社が多数。ブース訪問時に直接確認

3. **茶業会議所経由の紹介**: 特に京都府茶業会議所（宇治）、西尾茶協同組合（愛知）はOEM対応工場の紹介実績あり

4. **シェアシマ（ShareShima）等のB2Bプラットフォーム**: 食品原料のマッチングプラットフォームでOEM対応工場を検索

5. **産地の製茶組合への直接問い合わせ**: 各産地の茶業組合には、OEM対応可能な加工場のリストがある場合が多い

### 5-4. OEM契約時の確認事項

- **最小ロット**: OEMの場合、通常100kg〜。パッケージまで込みの場合は1,000個〜が目安
- **パッケージ仕様**: 袋の素材（アルミ三方シール/スタンドパウチ等）、印刷色数、窒素充填の有無
- **品質保証**: ロットごとの検査項目と合格基準
- **独占契約**: 同じ原料を他社にも供給するかの確認
- **知的財産**: ブレンドレシピやパッケージデザインの権利帰属
- **リードタイム**: 初回は2〜3ヶ月。リピートは1〜2ヶ月が目安

## 7. 補足：抹茶原料の基礎知識（商談前に押さえておくべき用語）

| 用語 | 意味 |
|------|------|
| **碾茶（てんちゃ）** | 抹茶の原料となる茶葉。覆い下栽培で育てた茶葉を蒸して乾燥させたもの。これを石臼で挽くと抹茶になる |
| **覆い下栽培** | 茶畑に覆いをかけて日光を遮る栽培方法。20日以上の被覆でうまみ成分（テアニン）が増加し、苦渋味が減少 |
| **石臼挽き** | 花崗岩の石臼で碾茶を挽く伝統的な製法。1時間あたり約40gと少量だが、熱が発生しにくく風味が保たれる |
| **ボールミル** | 大量生産向けの粉砕方法。石臼挽きに比べ粒度が粗くなりやすいが、コスト効率が良い |
| **品種** | さみどり（甘み、鮮やかな緑）、おくみどり（バランス型）、やぶきた（汎用性）、あさひ（高級）等 |
| **L値/a値** | 抹茶の色を数値で表す指標。L値が高いほど明るい緑、a値がマイナスに大きいほど鮮やかな緑 |
| **メッシュ** | 抹茶の粒度。セレモニアルグレードは通常5〜10ミクロン |
