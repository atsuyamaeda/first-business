## 1. 飛ばしてはいけないステップ（商習慣）

### 1-1. 茶業界のサプライチェーン構造

日本茶の流通は、「生葉流通」「荒茶流通」「仕上げ茶流通」の3段階に大別される。各段階で異なるプレイヤーが介在し、分業制が基本となっている。

```
【一般的な煎茶の流通】
  茶農家（栽培・摘採）
    ↓ 生葉
  荒茶工場（蒸し・揉み・乾燥 → 荒茶を製造）
    ↓ 荒茶（一次加工品）
  茶市場（入札会）or 直接取引
    ↓
  製茶問屋（仕上げ加工：選別・火入れ・合組＝ブレンド）
    ↓ 仕上げ茶
  小売店・消費者

【碾茶・抹茶の流通（特殊）】
  茶農家（覆下栽培 → 摘採）
    ↓ 生葉
  碾茶工場（蒸し・乾燥 ※揉まない → 碾茶を製造）
    ↓ 荒碾茶
  茶市場 or 茶商への直接販売
    ↓
  茶商・製茶問屋（仕上げ：精選・火入れ・合組・熟成保管）
    ↓ 仕上げ碾茶
  石臼挽き（製茶問屋 or 専門工場）
    ↓ 抹茶
  小売・飲料メーカー・食品メーカー・消費者
```

**重要ポイント**:
- 多くの茶農家は栽培と一次加工（荒茶製造）を兼業している。荒茶工場の設備（蒸機・揉機・乾燥機）は数千万円規模の投資が必要で、小規模農家は共同利用施設や委託加工を利用することもある
- 従来の流通では「農家ではなく市場や卸に価格決定権が集中」しており、農家の収益性が低く抑えられる構造的課題がある
- 近年は農家直販（自園自製）、OEM供給、越境ECなど新しい商流も出現しているが、依然として問屋経由が主流

### 1-2. 問屋（製茶問屋）の役割と機能

茶問屋は単なる「仲介業者」ではなく、以下の重要な機能を担う「製造問屋」としての性格が強い。

| 機能 | 内容 |
|------|------|
| **仕入れ・目利き** | 官能検査（嗅覚・味覚・視覚・触覚）による荒茶の品質見極め。茶審査技術は長年の経験が必要 |
| **仕上げ加工** | 荒茶から茎・粉などの不純物を除去（精選）。碾茶の場合は葉脈・茎の除去 |
| **火入れ** | 2段階の乾燥・香味調整。保存性の向上と風味の引き出し |
| **合組（ブレンド）** | 複数の茶葉を組み合わせて品質を安定化・最適化。**これが茶問屋の最も重要な技術**。代々受け継がれる「茶師」の技 |
| **保管管理** | 大型冷蔵庫での適切な温度・湿度管理。茶は直射日光・湿度・高温に弱く、専門的な管理なしでは品質が急速に劣化する |
| **石臼挽き（碾茶の場合）** | 仕上げ碾茶を石臼で挽いて抹茶に。1時間に約40gしか挽けず、熱管理（約50度で平衡）が重要 |
| **品質の安定供給** | 年間を通じて一定品質の茶を安定的に供給。天候不順や不作時のリスクバッファー機能 |
| **小分け対応** | 農家は大量生産が基本のため、小ロット対応は問屋が担う |

### 1-3. 問屋を飛ばして農家と直接取引することの是非

#### 農家直接取引のメリット

- **ストーリー性**: 「この農家のこの畑の茶」というシングルオリジン訴求が可能。D2Cブランドの差別化に直結
- **作り手の思い**: 産地の匂い、農家の哲学を一緒に仕入れられる
- **中間マージン削減**: 理論上は原価を下げられる可能性がある
- **トレーサビリティ**: 畑から消費者まで完全に追跡可能

#### 農家直接取引のデメリット・リスク

| リスク | 詳細 |
|--------|------|
| **品質の不安定さ** | 年によって天候に左右され品質が変動する。問屋の合組技術がないと、ロット間のばらつきを吸収できない |
| **仕上げ加工の問題** | 荒茶（荒碾茶）のままでは商品にならない。焙煎機・精選機・石臼がない農家がほとんど。仕上げ加工を別途委託する先が必要 |
| **保管設備の不在** | 大型冷蔵庫を持たない農家が多い。品質劣化リスク |
| **小ロット対応の困難** | 農家は大量生産が基本。「10kgだけください」が難しい場合がある |
| **品質の見極め力が必要** | 直接交渉では自分で荒茶の品質を見抜いて買取価格を提示する必要がある。これは素人には極めて難しい |
| **業界の秩序を乱すリスク** | 問屋を飛ばすことは、既存の取引関係を無視する行為と見なされる可能性がある。特に農家が既存の問屋と長年の取引関係がある場合、問屋の面子を潰す結果になりかねない |

#### 推奨アプローチ

**結論: 新規参入のスタートアップは、まず問屋経由で取引を始めることを強く推奨する。**

- 問屋は「コンビニ」のように、生産→仕入れ→仕上げ→火入れ→挽き→保管→出荷まですべてを管理してくれる
- 品質の見極め、合組、仕上げ加工のノウハウがないうちは、問屋のプロの技術に頼るのが合理的
- 問屋との関係構築を通じて業界の知識・人脈を蓄積した上で、徐々に農家との直接取引に移行するのが現実的なステップ
- ただし、問屋経由でも「どの農家の碾茶を使っているか」を開示してくれる問屋を選べば、シングルオリジン訴求も可能

### 1-4. 茶市場（入札会）の仕組み

#### 概要

日本には現在5つの主要な茶市場がある: **静岡、鹿児島、京都、福岡、三重**。茶市場は、生産者（売手）と製茶問屋・メーカー（買手）の橋渡しをする専門市場。

#### 取引の3者構成

取引は帽子の色で識別される3者で行われる:

| 役割 | 帽子の色 | 主体 |
|------|----------|------|
| **売手** | 緑色 | 生産農家、農協 |
| **買手** | 青色 | 製茶問屋、飲料メーカー（承認制） |
| **仲立人** | 黄色 | 茶市場の社員 |

- 買手の帽子には「50万円の担保」機能がある（支払い不履行時の保証）
- 買手になるには市場の承認が必要

#### 「相対取引」という独特の仕組み

一般的な競り（オークション）ではなく、**売手と買手が直接交渉する「相対取引」**が特徴。

1. **拝見（はいけん）**: 買手が茶の外観、香り、手触りで品質を判定。希望により熱湯で抽出して味を確認
2. **取引交渉**: 「手合票（てあいひょう）」に記載された売手の希望価格をベースに、**五玉そろばん**を使って価格交渉
3. **手打ち**: 合意成立時に**「シャン・シャン・シャン」と3回手を打ち**、取引確定

この方式は入札競争による極端な価格変動を避け、「確実に適正な価格で取引ができる」利点がある。

#### 開催時期

- 年間を通じて開催されるが、**最盛期は4月中旬～6月末**（新茶シーズン）
- 静岡茶市場は1956年創立で日本初の茶市場

#### 新規参入者にとっての現実

- **茶市場で直接買い付けるのは新規参入者にはハードルが高い**。買手の承認制であり、品質を見極める目利き力が求められる
- まずは問屋経由で仕入れ、業界知識を蓄積してから市場参入を検討するのが現実的

### 1-5. 碾茶の場合の流通経路の特殊性

碾茶（抹茶の原料）は、煎茶とは異なる流通経路と加工工程を持つ。

| 特殊性 | 内容 |
|--------|------|
| **覆下栽培が必須** | 2～3週間以上の被覆栽培。覆いの設営には棚式（伝統的・高コスト）と直かけ式がある。西尾の「棚式覆下栽培」は特に高品質 |
| **揉まない** | 煎茶と異なり、碾茶は蒸した後に揉まず乾燥させる。そのため平たいフレーク状の茶葉になる |
| **合組（ブレンド）の重要性が極めて高い** | 宇治では「各生産農家から納められた多品種の碾茶を各茶業者がそれぞれの銘柄に応じてブレンド」する。この合組技術が抹茶の味を決める |
| **熟成保管** | 碾茶は低温・除湿環境で熟成させることで風味が深まる。挽く直前まで碾茶の状態で保管するのが一般的 |
| **石臼挽きは最終工程** | 需要に応じて碾茶を石臼で挽く。挽きたてが最も風味が良いため、大量にまとめて挽くのではなく、注文に応じて挽くのが高品質抹茶の基本 |
| **宇治茶の定義** | 京都・奈良・滋賀・三重の4府県産茶葉を、京都府内で宇治地域の伝統的製法で仕上げ加工したもの。つまり**原料は京都産でなくても「宇治茶」を名乗れる**が、仕上げは京都でなければならない |
| **シングルオリジンの新潮流** | 従来のブレンド主体から、「単一農園・単一品種」の碾茶をそのまま石臼で挽く動きも出始めている。古川製茶（宇治）などが実践 |

### 1-6. 新規取引を始める際の一般的なプロセス

```
Step 1: 情報収集（1～3ヶ月）
  ├── 産地選定（宇治/西尾/八女/鹿児島/静岡）
  ├── 茶業会議所・組合への事前相談
  ├── 展示会への参加（FOODEX JAPAN等）
  └── JETRO等の公的支援機関の活用

Step 2: コンタクト・紹介（1～2ヶ月）
  ├── 紹介者の確保（茶業会議所、JETRO、既存取引先等）
  ├── 丁寧な自己紹介メール/電話
  └── 名刺交換・会社案内の送付

Step 3: サンプル取り寄せ・品質評価（1～2ヶ月）
  ├── 有料サンプルの取り寄せ（3～5社）
  ├── 複数グレードの比較テスト
  └── 自社ブランドとの適合性判断

Step 4: 産地訪問（新茶シーズンが理想：4～5月）
  ├── 茶畑・製造工場の見学
  ├── 対面での挨拶・信頼関係構築
  └── 品質管理体制の確認

Step 5: 小ロット試験注文（1～2ヶ月）
  ├── 初回5～30kgの小ロット発注
  ├── 品質・納期・コミュニケーションの確認
  └── 顧客フィードバックの収集

Step 6: 取引条件の交渉・契約（1～2ヶ月）
  ├── 価格・数量・納期・品質基準の合意
  ├── 支払条件の取り決め
  └── 定期注文の開始
```

## 2. 農家と取引する上で特に大事なこと

### 2-1. 信頼関係の構築方法

茶業界は**「義理と人情」が色濃く残る世界**。数十年にわたる取引関係が基本であり、新規参入者が信頼を得るには時間と誠意が必要。

| ポイント | 詳細 |
|----------|------|
| **対面を重視** | メールや電話だけでなく、必ず産地を訪問して顔を合わせる。日本の農業ビジネスでは「顔の見える関係」が信頼の土台 |
| **抹茶への深い理解を示す** | 碾茶の製造工程、品種の違い、覆下栽培の意味、合組の概念など、基礎知識があることを示す。「にわか」と思われたら門前払い |
| **長期的関係をアピール** | 「1回限りの取引ではなく、長期的なパートナーシップを希望」と明確に伝える。スポット買いの姿勢は嫌われる |
| **農家の仕事への敬意** | 栽培の苦労、品質へのこだわり、受賞歴などに具体的に言及し、敬意を示す |
| **小さな約束を守る** | メールの返信、訪問日時の厳守、サンプル評価の報告など、小さな約束を確実に守ることで信頼を積み上げる |
| **焦らない** | 関係構築には少なくとも1～2年の時間軸で考える。最初から大量発注の話をするのは逆効果 |

### 2-2. 価格交渉の慣行

| 項目 | 慣行 |
|------|------|
| **値切りは基本的にNG** | 特に初回取引で値切るのは非常に悪い印象を与える。農家の品質と労力に対する正当な対価を支払う姿勢が大前提 |
| **適正価格の理解** | 茶市場での相場観を持っておくことが重要。相場より著しく安い価格を提示すると、農家の品質を軽視していると受け取られる |
| **量が増えれば自然と下がる** | 取引量の拡大に伴い、ボリュームディスカウントが適用されるのが自然な流れ。最初から安くしろとは言わない |
| **品質に対して正当に支払う** | Blue Farmのように「市場価格の倍近い価格で直接買い取る」姿勢が、農家にとって最も歓迎される |
| **五玉そろばんの文化** | 茶市場では五玉そろばんを使った価格交渉が伝統。互いの希望価格を擦り合わせ、「手打ち（シャン・シャン・シャン）」で決着する文化が根付いている |
| **値上げ要請への対応** | 原料高騰や不作時に農家から値上げ要請があった場合、受け入れる姿勢が長期関係の鍵。不作の年に買い叩く行為は業界で最も嫌われる |

### 2-3. 支払条件の一般的な慣行

| 条件 | 内容 |
|------|------|
| **初回取引** | **前払い（前金）が一般的**。信用がない段階では現金払いを求められることが多い |
| **継続取引（信用構築後）** | 月末締め翌月払い、あるいは翌々月払い（30日～60日サイト）が一般的 |
| **手形取引** | 大手問屋との取引では約束手形が使われることもある。手形サイトは30日～120日が一般的だが、2021年の公正取引委員会ガイドラインで60日以内が推奨されている |
| **新茶シーズンの特殊性** | 新茶シーズン（4～6月）は農家にとって年間収入の大部分が集中する時期。この時期の支払いの遅延は致命的な信用棄損になる |
| **為替リスク** | 海外向け取引の場合、円建てが基本。ドル建て等の場合は為替リスクの負担方法を事前に取り決める |

### 2-4. 契約の形態

| 形態 | 実態 |
|------|------|
| **口約束が多い** | 茶業界では長年の信頼関係に基づく口約束での取引が依然として多い。「信頼関係があるから書面は不要」という考え方が根強い |
| **書面化の推奨** | ただし、新規取引やスタートアップとの取引では、互いの保護のために書面化することが望ましい。特に品質基準、価格、数量、納期、支払条件は書面に残すべき |
| **年間契約 vs スポット** | プレミアム碾茶は年間契約（一番茶の買取を事前に約束）が一般的。スポット買いは相場変動リスクが高く、安定調達が難しい |
| **契約栽培** | 伊藤園のような大手は契約栽培（品種・栽培方法を指定して年間買取を保証）を行っているが、スタートアップが最初からこれを求めるのは現実的でない |

### 2-5. 収穫時期と発注タイミングの関係

| 時期 | イベント | 対応すべきこと |
|------|----------|--------------|
| **1～2月** | 茶園の準備期間 | 年間の発注計画を問屋・農家と共有。一番茶の予約を入れる |
| **3月** | FOODEX JAPAN等の展示会 | 新規サプライヤーとの出会い、サンプル依頼 |
| **4月中旬～5月** | **一番茶の摘採（新茶シーズン）** | 碾茶の場合、覆下栽培の茶が最も品質が高い時期。**この時期に産地訪問・買い付けが集中** |
| **5月下旬～6月** | 茶市場の最盛期 | 荒茶の取引が最も活発。価格が決まる重要な時期 |
| **6月下旬～7月** | 二番茶の摘採 | 一番茶より品質は落ちるが価格は安い。カリナリーグレード向け |
| **8月～9月** | 秋冬番茶・閑散期 | 来期の計画策定、関係構築のための訪問に好適 |
| **10月** | 宇治茶まつり等のイベント | 産地のイベントへの参加で関係強化 |
| **11月～12月** | 年末需要期、来期の準備 | 次年度の発注量の仮コミット。年末の挨拶 |

**重要**: 一番茶は「走り」（出始め）ほど品質が高く高価。鹿児島は4月上旬から、京都宇治は5月上旬からと、産地により時期が異なる。

### 2-6. 農家が嫌がること・歓迎すること

#### 嫌がること（絶対にやってはいけない）

| 行為 | 理由 |
|------|------|
| **不作の年に買い叩く** | 天候不順で品質・量が落ちた年に価格を下げろと要求するのは最大のタブー。農家の生活を脅かす行為 |
| **突然の大量キャンセル** | 農家は注文を見越して栽培計画を立てている。直前のキャンセルは収入ゼロを意味する |
| **品質を理解しない値引き交渉** | 品質の違いを理解せず「もっと安くしろ」と言うのは、農家の仕事への冒涜 |
| **既存の取引関係を無視した引き抜き** | 他の問屋と長年取引している農家に対し、問屋を飛ばして直接取引を持ちかけるのは業界のタブー |
| **支払い遅延** | 特に新茶シーズンの支払い遅延は致命的。農家にとって年間収入の大部分が集中する時期 |
| **約束を守らない** | 「来年は倍買います」と言って買わない、「長期で付き合う」と言って1年で離れるなど |
| **農家の茶を他社のものと混ぜて販売** | シングルオリジンとして買ったのに他の茶と混ぜるのは裏切り行為 |
| **忙しい時期のアポなし訪問** | 特に新茶シーズン（4～5月）は24時間体制で作業している。事前連絡なしの訪問は非常に迷惑 |

#### 歓迎すること

| 行為 | 理由 |
|------|------|
| **適正価格での安定的な買取** | 毎年一定量を適正価格で買い取ってくれる取引先は農家にとって最も価値がある |
| **茶の品質・農家の哲学への深い理解** | 自分の茶のことを本当に理解してくれるバイヤーは尊重される |
| **消費者の声のフィードバック** | 「あなたの茶を飲んだお客様がこう言っていた」という情報は農家の大きな励みになる |
| **ブランドストーリーでの農家紹介** | D2Cブランドのウェブサイトやパッケージで農家を紹介し、認知度向上に貢献すること |
| **不作の年も買い続ける** | 品質が落ちた年でも離れずに取引を続ける姿勢は、最も深い信頼を生む |
| **新茶の出来を一緒に喜ぶ** | 新茶シーズンに訪問し、出来を一緒に確認・喜び合う姿勢 |
| **業界イベントへの参加** | 品評会、茶まつりなどに顔を出すことで「業界の一員」として認めてもらえる |

## 3. スムーズに開拓を進めるためのお作法

### 3-1. 紹介経由の重要性

**茶業界では「紹介」が取引開始の最も重要な要素。飛び込みでの新規開拓は成功率が極めて低い。**

| 紹介元 | 活用方法 |
|--------|----------|
| **茶業会議所・茶業組合** | 最も正式なルート。産地の茶業会議所（京都府茶業会議所、西尾茶協同組合等）に「D2Cブランド立ち上げのため、碾茶の調達先を探している」と相談。要件に合う農家・製茶工場を紹介してもらえる |
| **JETRO** | 海外展開を視野に入れている場合は最適。無料でサプライヤーマッチング支援を受けられる |
| **展示会での出会い** | FOODEX JAPANなどで直接出会い、名刺交換した相手には「展示会でお会いした○○です」と切り出せる |
| **既存の業界関係者** | 茶道の師範、茶器メーカー、飲食店経営者など、業界周辺の人脈からの紹介も有効 |
| **同業他社（非競合）** | 煎茶ブランドや茶器ブランドなど、直接競合しない同業者からの紹介 |

### 3-2. 最初のアプローチ方法

| 手段 | 推奨度 | 備考 |
|------|--------|------|
| **紹介者経由の電話/メール** | ★★★★★ | 最も有効。「○○さんからご紹介いただきました」の一言で対応が格段に良くなる |
| **丁寧な日本語メール** | ★★★★ | 紹介がない場合の次善策。会社概要、ブランドコンセプト、なぜその農家/工場に関心を持ったかを具体的に記載 |
| **電話** | ★★★ | メール送付後のフォローとして有効。いきなりの電話は警戒される場合がある |
| **訪問** | ★★ | 事前アポなしの訪問はNG。ただし、展示会やイベントでの対面は最も強い印象を残せる |
| **英語のメール** | ★ | **無視される可能性が高い**。日本語でのコミュニケーションが必須 |

### 3-3. 産地訪問時のマナー

| マナー | 詳細 |
|--------|------|
| **必ず事前にアポイントを取る** | 最低1～2週間前に連絡。特に新茶シーズン（4～5月）は農家が超多忙なので要注意 |
| **時間厳守** | 5分前到着が基本。遅刻は論外 |
| **服装** | ビジネスカジュアル。茶畑を歩く可能性があるため、きれいめだが動きやすい格好。女性はヒールは避ける |
| **名刺** | **日本語の名刺は必須**。両面印刷（片面日本語/片面英語）が理想的 |
| **挨拶** | 丁寧な日本語での挨拶。「本日はお忙しい中お時間をいただきありがとうございます」 |
| **茶畑・工場での注意** | 指示に従う。香水・強い柔軟剤の匂いは茶葉に移るためNG。カメラは許可を得てから |
| **お茶を出されたら** | 必ず飲む。味の感想を具体的に伝える（「旨味が深い」「覆い香が素晴らしい」等）。これが品質への理解を示す最良の方法 |
| **質問を用意** | 品種、被覆期間、栽培のこだわり等について事前に調べた上で質問する。「何も知りません、教えてください」より「○○について伺いたい」の方が格段に好印象 |
| **写真撮影** | 特に工場内部・製造ラインの撮影は必ず許可を得る。企業秘密に関わる部分もある |

### 3-4. 手土産の慣行

| ポイント | 詳細 |
|----------|------|
| **手土産は持参すべき** | 初回訪問では特に重要。日本のビジネス訪問では手土産は基本マナー |
| **おすすめ** | 自分の地元の銘菓（地方出身者の場合）、有名百貨店の洋菓子・和菓子。3,000～5,000円程度が無難 |
| **避けるべきもの** | 他社の茶は持って行かない（当然だが重要）。相手の地元の名産品は避ける（わざわざ持っていく必要がない） |
| **のし紙** | 「ご挨拶」の表書きで。正式な訪問であることを示す |
| **2回目以降** | 毎回豪華な手土産は不要だが、季節の果物や自社商品（ブランド立ち上げ後）などの気遣いは喜ばれる |

### 3-5. 長期的な関係構築のポイント

| ポイント | 詳細 |
|----------|------|
| **年に最低2回は訪問** | 新茶シーズン（4～5月）と秋（9～10月）の訪問が理想。顔を見せ続けることが信頼の基盤 |
| **年末年始・お盆の挨拶** | 年賀状、お中元・お歳暮は日本のビジネス関係維持の基本。特にお歳暮は重要 |
| **品評会の応援** | 農家が品評会に出品している場合、結果を気にかけ、受賞したらすぐに祝いの連絡をする |
| **情報共有** | 消費者の反応、市場のトレンド、海外でのMatcha需要の動向など、農家にとって有益な情報を定期的に共有 |
| **困った時に支える** | 不作の年、設備投資が必要な時、後継者問題など、農家が困難に直面した時に寄り添う姿勢が最も深い信頼を生む |
| **SNSやウェブでの発信** | 農家やその製品をSNSやブランドサイトで発信し、認知度向上に貢献。ただし**必ず事前に内容の確認と掲載許可を得る** |
| **共に成長する姿勢** | 自社の売上拡大に伴い、農家からの購入量も増やしていく。「Win-Winの関係」を実現する |

### 3-6. やってはいけないタブー

| タブー | 詳細 |
|--------|------|
| **問屋を飛ばしての裏取引** | 問屋経由で知った農家に対し、問屋に無断で直接コンタクトを取るのは最大のタブー。業界から閉め出される可能性 |
| **価格情報の漏洩** | 取引価格は機密情報。他のバイヤーや競合に漏らすのは信義違反 |
| **品質クレームの公開** | 品質に問題があった場合は、直接相手に連絡する。SNSや公開の場で批判するのは業界のルール違反 |
| **一方的な値下げ要求** | 特に原材料高騰時の一方的な値下げ要求は関係破壊の原因。相互の事情を理解した上で協議する |
| **農家の技術・品種の無断転用** | 訪問時に得た栽培ノウハウや品種情報を、他の農家に伝えたり、自社で転用したりするのは厳禁 |
| **産地偽装・品質表示の不正** | 「宇治産」でないものを「宇治抹茶」と表示するなど、産地表示の不正は法律違反でもあり、業界全体の信用を毀損 |
| **短期的な利益追求の態度** | 「安く仕入れて高く売る」だけの姿勢は即座に見抜かれる。農家とバイヤーは対等なパートナーであるという認識が不可欠 |
| **お茶を粗末にする** | サンプルを乱雑に扱う、試飲時に流し捨てるなどの行為は、茶を生業とする人々への冒涜 |
