# 配送プラン v1 — 小ロット・自社オペレーション

> **方針**: 初期は自分たちで梱包・発送。仕組みをシンプルに保ち、固定費を最小化する。
> スケールが必要になったタイミングで3PLへ移行する。

---

## 全体像

```
┌─────────────────────────────────────────────────┐
│                    販売チャネル                    │
│                                                   │
│   [サブスク（自社EC）]          [Amazon US]        │
│         │                          │              │
│     自分で梱包・発送            FBA に任せる       │
│         │                          │              │
│   ネコポス / ゆうパック        Amazon が配送       │
│         │                          │              │
│       顧客                       顧客             │
└─────────────────────────────────────────────────┘
```

- **Amazon** → FBA（既定路線。梱包・配送・返品すべてAmazonが処理）
- **サブスク** → 自分で梱包して発送（以下はこちらの設計）

---

## 1. 配送パターンは2つだけ

### パターンA: 初回セット（茶碗・茶筅・茶杓 + 抹茶）

| 項目 | 内容 |
|------|------|
| 配送方法 | **ゆうパック 60サイズ** or **ヤマト宅急便 60サイズ** |
| 想定サイズ | 30cm × 20cm × 10cm 程度 |
| 想定重量 | 800g〜1.2kg |
| 送料目安 | ¥800〜1,000（関東発・全国平均） |
| 梱包 | 茶碗はプチプチ＋仕切りで固定。茶筅は筒状ケース。抹茶は隙間に配置 |
| 受取方法 | 対面 or 置き配（破損リスクを考えると置き配で十分。壊れたら再送対応） |

### パターンB: リフィル（2回目以降、抹茶のみ）

| 項目 | 内容 |
|------|------|
| 配送方法 | **ネコポス**（¥385）or **ゆうパケット**（¥360） |
| サイズ制約 | 厚さ3cm以内、A4サイズ以内 |
| 想定重量 | 個包装 × 30日分 = 60〜90g（余裕で収まる） |
| 送料目安 | **¥360〜385**（全国一律） |
| 梱包 | 専用デザイン封筒 or 薄型ボックスにそのまま入れるだけ |
| 受取方法 | **ポスト投函**（不在でも届く = 最大のメリット） |

> **PostCoffeeも同じ構造**: 初回キットは宅配便、2回目以降はポスト投函。実証済みのモデル。

---

## 2. 梱包オペレーション

### 用意するもの（初期投資）

| アイテム | 用途 | 目安コスト |
|---------|------|----------|
| ネコポス対応ボックス or 封筒 | リフィル用 | ¥30〜50/個（100枚〜で発注） |
| 60サイズダンボール | 初回セット用 | ¥50〜80/個 |
| プチプチ（緩衝材） | 茶碗保護用 | ¥1,000/ロール |
| OPP袋 | 個包装の追加保護 | ¥500/100枚 |
| ブランドステッカー or テープ | 開封体験の演出 | ¥3,000〜（初回デザイン+印刷） |
| **初期投資合計** | | **¥5,000〜10,000 程度** |

### 梱包手順

**初回セット（5分/個）**
1. ダンボールに仕切りをセット
2. 茶碗をプチプチで包んで配置
3. 茶筅・茶杓をケースに入れて配置
4. 抹茶パウダーを隙間に配置
5. 同梱物（点て方ガイド、テイスティングノート）を上に載せる
6. 封をしてブランドステッカーを貼る

**リフィル（1分/個）**
1. 個包装の抹茶をネコポスボックスに入れる
2. 同梱物（今月のカード等）を入れる
3. 封をする

## 3. 発送フロー

```
受注（Shopify通知）
  │
  ├→ 当日 or 翌日に梱包
  │
  ├→ コンビニ or 郵便局で発送
  │    ├ ネコポス → ファミマ / セブン（ヤマト）
  │    └ ゆうパケット / ゆうパック → ローソン / 郵便局
  │
  ├→ 追跡番号をShopifyに入力 → 顧客に自動通知
  │
  └→ 到着（ネコポス: 1-2日 / ゆうパック: 1-3日）
```

### 発送頻度の目安

| 月間サブスク数 | 作業量 | 現実性 |
|-------------|--------|--------|
| 〜30件 | 週1回まとめて発送（1時間程度） | 余裕 |
| 30〜100件 | 週2回発送（各1-2時間） | 一人でギリ回せる |
| 100件〜 | ほぼ毎日発送 → **3PL移行を検討** | 限界ライン |

## 4. 送料の扱い

### シンプルな方針

| パターン | 送料 | 顧客への見せ方 |
|---------|------|--------------|
| 初回セット | ¥800〜1,000 | **送料無料**（商品価格に含める）。初回の「お得感」を最大化 |
| リフィル | ¥360〜385 | **送料無料**（月額に含める）。ネコポスなら十分吸収可能 |

> 月額¥2,980〜のサブスクで送料¥360は原価率の12%。粗利70%想定なら十分吸収できる。
> 「送料無料」の一言が購入ハードルを大きく下げる。

---

## 5. 温度管理

| 季節 | 対応 |
|------|------|
| 通常期 | 個包装の密封（アルミ蒸着パウチ + 窒素充填）で十分。常温配送 |
| 夏季（7-9月） | **当面は対応不要**。個包装＋密封なら1-2日の配送で品質劣化はほぼない |
| 和菓子セット | MVP段階では扱わない。スケール後に冷蔵便で別送を検討 |

---

## 6. トラブル対応（最小限のルール）

| トラブル | 対応 |
|---------|------|
| 茶碗破損 | 写真を送ってもらう → 無条件で再送。小ロットなら年数件で済む |
| 届かない | 追跡番号で確認 → 紛失なら再送 |
| 品質不満 | 個別に連絡をもらって対応。初期の声は全部拾ってプロダクト改善に回す |

> 初期は「困ったら全部再送」でOK。年間数件のコストより、顧客体験と口コミの方がはるかに重要。

---

## 7. 3PL移行の判断基準

以下のいずれかに該当したら、3PL（ロジモプロ、オープンロジ等）への移行を検討：

- [ ] 月間出荷数が **100件を超えた**
- [ ] 梱包・発送に週 **10時間以上** 取られている
- [ ] 発送ミス（誤配送・遅延）が **月3件以上** 発生
- [ ] 自分の時間を商品開発・マーケに使いたいのに発送に追われている
