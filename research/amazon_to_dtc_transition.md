# Amazon依存脱却計画：DTC移行のロードマップとP/L比較

**作成日**: 2026年3月16日
**為替前提**: $1 = ¥150

---

## なぜAmazonから脱却すべきか

| 項目 | Amazon | 自社DTC |
|------|--------|--------|
| 売上に対する手数料 | **33.1%** | **5-6%** |
| 顧客データ | Amazonが保有（自分には見えない） | **自分が保有**（メール、住所、購買履歴） |
| リピート施策 | 不可（Amazonが顧客との関係を握る） | **サブスク、メール、SNSで直接リーチ** |
| 価格決定権 | 競合と横並び比較される | **ブランドストーリーで価格を正当化できる** |
| アカウントリスク | 停止されたら即終了 | **自分でコントロール** |

### 比較一覧

| チャネル | 売上 | 変動利益 | 利益率 | Amazon比 |
|---------|------|---------|--------|---------|
| **Amazon** | ¥3,300 | **¥1,373** | 41.6% | — |

## DTCサブスクリプションモデル

抹茶は消耗品 = **サブスクとの相性が最高に良い。**

### 30g毎月サブスク（$22/月 = ¥3,300）

| 項目 | 金額 |
|------|------|
| 売上（月額） | ¥3,300（$22、Amazonと同額） |
| COGS | -¥675 |
| Shopify決済手数料 | -¥141 |
| ePacket発送 | -¥700 |
| 梱包材 | -¥100 |
| **月あたり変動利益** | **¥1,684（51.0%）** |

| 指標 | 数値 |
|------|------|
| サブスク1人の年間売上 | ¥39,600 |
| サブスク1人の年間利益 | **¥20,208** |
| **50人のサブスク会員** | **年間利益¥1,010,400** |
| **100人のサブスク会員** | **年間利益¥2,020,800** |

> サブスク50人で年間利益¥100万。Amazon単品販売で同じ利益を出すには年間1,400個売る必要がある。

## SNSアカウント運用計画

### なぜSNSが全ての鍵か

```
SNSフォロワー → DTC訪問 → 購入 → サブスク化 → LTV最大化
     ↓
  信頼構築（農家の顔、品質の透明性、日本文化）
     ↓
  Amazon広告費ゼロでも顧客を獲得できる
```

### プラットフォーム選定

| | Instagram | TikTok |
|---|-----------|--------|
| 役割 | ブランド構築・世界観 | リーチ・バイラル獲得 |
| ターゲット | 25-40歳、健康志向、プレミアム志向 | 18-30歳、抹茶初心者、トレンド感度高い |
| コンテンツ | 農家訪問、抹茶の美しい写真、レシピ | 比較動画、点て方チュートリアル、裏側 |
| 投稿頻度 | 週3-5投稿 + ストーリーズ毎日 | 週5-7投稿 |
| 言語 | 英語メイン（日本語サブ） | 英語メイン |

### コンテンツ戦略：「日本人がやってる」を活かす

| コンテンツの柱 | 例 |
|-------------|---|
| **農家の裏側** | 茶畑訪問、被覆の様子、摘採、碾茶工場見学 |
| **品質の透明性** | 「$6 vs $22 vs $60の抹茶を比較してみた」 |
| **抹茶の知識** | 品種の違い、水温の影響、グレードの見分け方 |
| **日本の茶文化** | 茶道体験、産地の風景、農家インタビュー |
| **日常の抹茶** | 毎朝の抹茶ルーティン、レシピ、アレンジ |

> **核心: 「日本人がやってる」は国籍ではなく「コンテンツの質と信頼性」で示す。** 実際に農家に行ける、日本語で農家と話せる、品質を目利きできる——これをSNSで見せ続けることが最大の差別化。

### フォロワー獲得のマイルストーン

| 時期 | Instagram | TikTok | アクション |
|------|-----------|--------|-----------|
| M1-3 | 1,000 | 2,000 | 毎日投稿。ハッシュタグ戦略。抹茶コミュニティと交流 |
| M4-6 | 3,000 | 5,000 | コラボ投稿。バイラル狙いの比較動画 |
| M7-9 | 5,000 | 10,000 | Shopify開設。プロフィールにDTCリンク設置 |
| M10-12 | 8,000 | 20,000 | サブスク訴求。メールリスト1,000人 |
| M13+ | 15,000+ | 50,000+ | コミュニティ化。UGC（ユーザー投稿）の促進 |

### SNS → DTC への転換率の目安

| 指標 | 業界平均 | 目標 |
|------|---------|------|
| フォロワー → サイト訪問 | 1-3% | 2% |
| サイト訪問 → 購入 | 2-4% | 3% |
| 購入 → サブスク化 | 10-20% | 15% |

> Instagram 10,000フォロワーの場合:
> 10,000 × 2%（訪問） × 3%（購入） × 15%（サブスク） = **サブスク1人/月**
> → フォロワーが増えるほど自動的にサブスク会員が増える

> TikTok 50,000フォロワーの場合:
> バイラル投稿1本で数千サイト訪問 → 数十購入 → サブスク数人/月
> → TikTokは爆発力はあるがコントロール不能

## 最終まとめ：数字で見るAmazon脱却の効果

| | Amazon 100%のまま | DTC移行後（Phase 3） |
|---|-------------------|---------------------|
| 年間販売数 | 1,000個 | 1,080個 |
| 年間売上 | ¥3,300,000 | ¥3,672,000 |
| Amazon手数料 | **-¥1,092,000** | **-¥262,080** |
| **年間利益** | **¥881,000** | **¥1,654,560** |
| **利益率** | **27%** | **45%** |

**同じ商品、ほぼ同じ販売数で、利益が約2倍になる。**
その鍵は「SNSでファンを作り、自社サイトで買ってもらう」こと。
