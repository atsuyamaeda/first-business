# 日本の茶園（抹茶/碾茶用）取得ガイド

調査日: 2026年3月

---

## 1. 農地価格の相場（茶畑/畑地）

### 全国平均（令和6年/2024年）

全国農業会議所「田畑売買価格等に関する調査結果」による:

| 区分 | 中畑価格（10aあたり） | 前年比 |
|------|----------------------|--------|
| 純農業地域 | 77.1万円 | △1.0% |
| 都市的農業地域 | 約273万円 | - |

**注**: 純農業地域の農地価格は30年連続で下落中。

### 地域別の畑価格目安（10aあたり）

| 地域 | 畑価格（10aあたり） | 備考 |
|------|---------------------|------|
| 東海（静岡・愛知含む） | 約35万円 | 純農業地域 |
| 九州（鹿児島・福岡含む） | 約98万円 | 純農業地域 |
| 近畿（京都含む） | データ未公開 | 宇治市の宅地は13.8万円/m2（農地はこれより大幅に安い） |
| 全国平均 | 77.1万円 | 純農業地域の畑 |

**1反（≒10a≒991m2）あたりの概算**:
- 安い地域（東海の純農業地域）: 30〜50万円/反
- 中程度（全国平均）: 70〜80万円/反
- 高い地域（九州、都市近郊）: 100〜280万円/反

**1ヘクタール（10反）換算**:
- 安い地域: 300〜500万円/ha
- 中程度: 700〜800万円/ha
- 高い地域: 1,000〜2,800万円/ha

### 茶園としての特殊性

茶畑は通常の畑と異なり、以下の要素が価格に影響:
- 茶樹の樹齢と状態（成木園か幼木か）
- 被覆棚（遮光設備）の有無 → 碾茶用は被覆棚付きで価値が高い
- 傾斜地か平坦地か（機械化適性）
- 品種（やぶきた、さみどり、あさひ等）
- 有機認証の有無

## 2. 小規模茶園の取得総コスト（概算）

### シナリオ: 2〜4ha の碾茶（抹茶原料）用茶園

| 項目 | 概算コスト | 備考 |
|------|-----------|------|
| **土地取得費** | 600万〜3,200万円 | 2〜4ha、地域による |
| **被覆棚（遮光設備）** | 500万〜2,000万円 | 棚がけ式は高額。直がけ式は安い |
| **乗用型茶園管理機** | 300万〜1,500万円（新品） | カワサキ機工等。中古なら半額程度 |
| **茶刈機** | 50万〜300万円 | 乗用型は高額、手持ち型は安い |
| **製茶加工設備** | 2,000万〜5,000万円+ | 蒸し機、碾茶炉、選別機など。自前でなく委託加工も可 |
| **倉庫・加工場** | 500万〜2,000万円 | 既存施設を引き継げれば大幅削減 |
| **苗木・植替（必要時）** | 100万〜500万円 | 成木園を引き継げば不要 |
| **その他（申請費・仲介等）** | 50万〜200万円 | 農地法の許可申請含む |

**合計概算**: 4,000万〜1.5億円程度

**大幅削減の方法**:
- 事業承継で既存設備込みで引き継ぐ → 設備費を大幅カット
- 加工は外部委託（JA・近隣の製茶工場） → 製茶設備不要
- 耕作放棄地を活用 → 土地取得費をほぼゼロに
- リース（借地）で始める → 初期投資を数百万円レベルに

## 3. 茶園の売買・入手チャネル

### (a) 直接売買

**農地の売買情報サイト**:
- [全国農地ナビ](https://www.farmland-navi.go.jp/) - 農業委員会系の公式サイト
- [農地相場ドットコム](https://www.tochi-d.com/nouchi/) - 340,277件の農地価格データ収録
- 一般不動産サイト（SUUMO、アットホーム等）には農地は少ない

**注意**: 茶園として明示的に売りに出ている物件は極めて少ない。地元の農業委員会やJA（農協）を通じた口コミ・紹介が主流。

### (b) 農地バンク（農地中間管理機構）

全都道府県に設置されている公的機関。農地の借り手と貸し手をマッチング。

**仕組み**:
1. 貸し手（引退農家等）が農地バンクに農地を預ける
2. 農地バンクが農地を借り受け
3. 担い手（借り手）に集積・集約した形で貸し出し
4. 賃料は農地バンクが仲介して支払い

**賃料の目安**: 3,000〜10,000円/10a/年（地域による。茶園は一般畑より高い可能性あり）

**メリット**:
- 公的機関で安心
- 交渉不要（バンクが仲介）
- 契約期間満了で確実に返還
- 分散した小規模農地をまとめて借りられる

**主要茶産地の農地中間管理機構**:
- 京都府: [京都府農林水産業振興センター](https://www.pref.kyoto.jp/)
- 静岡県: [静岡県農業振興公社](https://www.shizuoka-nk.or.jp/)
- 鹿児島県: [鹿児島県地域振興公社](https://www.kagoshima-kikou.jp/)
- 愛知県: 愛知県農業振興基金
- 福岡県: 福岡県農林水産振興センター

### (c) 事業承継（第三者継承）

後継者のいない農家から事業を譲り受ける制度。

**主要マッチングプラットフォーム**:
- [農業をはじめる.JP](https://www.be-farmer.jp/recruitment/devolution/) - 全国新規就農相談センター運営
- [relay](https://relay.town/) - 事業承継マッチングプラットフォーム（鹿児島県等）
- [ジモティー](https://jmty.jp/) - 地域掲示板で後継者募集あり
- 各都道府県の農業会議所・農業委員会

**現状**: 茶農家の約7割が後継者未確保。特に和束町（京都）では過去10年で約100世帯の茶農家が減少。

**事業承継の補助金**:
- 事業承継・引継ぎ補助金（Ⅱ型: 売り手支援型）: 補助対象経費の2/3以内、上限400万円（廃業費含む場合600万円）

### (d) 耕作放棄地の活用

**現状**: 耕作放棄地は農地面積全体の約1割（富山県一つ分に相当）

**茶園特有の事情**: 和束町では耕作放棄茶園が10年間で9.4ha→38.5haに拡大（2018年時点）

**メリット**:
- 土地取得コストが極めて低い（無償〜格安で借地可能）
- 農薬不使用期間が長く、有機栽培に転換しやすい
- 自治体から補助金が出る場合がある

**デメリット**:
- 茶樹の剪定・再生に2〜3年かかる
- 雑草・竹の侵入で開墾作業が必要
- 傾斜地が多く機械化が難しい場合がある

### (e) 茶畑オーナー制度

数千円から申し込める「茶畑オーナー」制度も各地に存在。ビジネス規模ではないが、産地との関係構築に有用。

## 4. 農地法の制限（特に外国人・非農家）

### 農地法第3条の許可要件

農地の売買・貸借には農業委員会の許可が必要:

| 要件 | 内容 |
|------|------|
| 全部効率利用要件 | 取得するすべての農地で効率的に耕作すること |
| 農作業常時従事要件 | 取得者が原則年間150日以上農作業に従事すること |
| 地域との調和要件 | 周辺の農地利用に支障がないこと |
| 下限面積要件 | 取得後の経営面積が地域の基準以上であること（50a等） |

### 法人の場合

**農地を「所有」するには → 農地所有適格法人の要件が必要**:
- 法人形態: 農事組合法人、非公開株式会社、持分会社のいずれか
- 主たる事業が農業（関連事業含む）
- 役員の過半が農業に常時従事する構成員（年間150日以上）
- 農業関係者の議決権が過半数

**農地を「借りる」だけなら → 一般法人でも可能**（解除条件付き貸借）:
- 役員または権限ある使用人の1人以上が常時従事
- 農地を適正に利用しない場合は契約解除

### 2025年4月施行: 外国人の農地取得規制強化

| 改正内容 | 詳細 |
|----------|------|
| 短期在留者の制限 | 在留期間1年未満の外国人は原則として農地取得不可 |
| 報告義務 | 在留資格番号・在留期限を農業委員会に報告義務 |
| 農地台帳への記載 | 所有者の国籍、在留資格番号、在留期限を記載 |
| 外国資本企業の規制 | 外国籍株主の議決権10%超 → 特別な説明義務 |
| 外国金融機関借入 | 自己資本の50%超 → 許可審査厳格化 |
| 政府系ファンド出資 | 農林水産大臣の事前承認が必要 |

**実務的なインプリケーション**:
- 日本に永住権または長期在留資格があれば取得は可能
- 日本法人を設立し農地所有適格法人の要件を満たすのが現実的
- リース（借地）の方がハードルは低い

## 5. 代替モデル

### (a) リース（借地）モデル

**メリット**: 初期投資を大幅に抑えられる。農地所有適格法人でなくても一般法人で可能。
**賃料目安**: 3,000〜10,000円/10a/年 → 1haで3万〜10万円/年
**注意**: 碾茶用の被覆棚等の設備投資は別途必要

### (b) 耕作放棄地の再生

**実例: d:matcha（京都・和束町）**
- 2016年創業、農学系メンバーが和束町に移住
- 2017年から自社茶園を耕作開始
- 現在約4haの茶園を管理
- 耕作放棄地を再生し、有機栽培に転換
- 栽培から販売まで垂直統合型のD2Cモデル

**実例: 健一自然農園（奈良県大和高原）**
- 30か所、12haの茶園を管理
- 5年間放棄された茶園を再生
- 自然農法による三年番茶を生産
- 耕作放棄地を地域資源として活用

### (c) 伊藤園の新産地事業モデル

- 1976年開始の契約栽培事業
- 2001年から「新産地事業」で耕作放棄地を大規模茶園に造成
- 2021年時点で全国2,207haをカバー（7県9地域）
- 伊藤園が技術・ノウハウを提供し、全量買取
- 年間300haの茶園が減少する中での産地維持策

### (d) 契約栽培（自社農園を持たないモデル）

多くの抹茶D2Cブランドが採用:
- Kettl: 自社農園なし、産地の農家から直接仕入れ（年に複数回訪問）
- 一保堂: 自社農園なし、茶問屋として仕入れ・ブレンド
- 多くのD2Cブランドが仕入れモデルで事業を運営

| 企業 | 所在地 | 規模 | 取得方法 | 備考 |
|------|--------|------|----------|------|
| d:matcha | 京都・和束 | 約4ha | 耕作放棄地の再生 | 2016年創業。栽培〜D2C販売の垂直統合 |

## 8. 2025-2026年の市場環境（茶園取得に影響する要因）

### 碾茶（抹茶原料）価格の高騰

- 2025年一番茶の碾茶オークション平均価格: **8,235円/kg**（前年の約1.7倍、過去最高の4,862円/2016年を大幅更新）
- 一部ロットは平均の5.5倍で取引
- 二番茶（京都）は前年比170%の値上がり
- 京都の碾茶農家は収量25%減（猛暑の影響）

### 供給逼迫の背景

- 京都は日本の碾茶の約1/4を生産
- 2008〜2023年で日本の緑茶輸出は9倍に増加
- 世界の抹茶市場は40億ドル超に成長
- 気候変動（猛暑・干ばつ）による収量減少
- 年間300haの茶園が減少中

### 茶園取得への示唆

- 碾茶価格の高騰により、茶園の価値は上昇傾向
- 一方で農家の高齢化・後継者不足は加速しており、事業承継の機会は増加
- 耕作放棄地も増加しており、低コストでの参入機会は拡大
- **今が参入の好機**: 供給不足 × 放棄地増加のギャップがチャンス

## 9. 推奨アクションプラン

### Step 1: 産地視察と関係構築
- 和束町（京都）、西尾（愛知）、八女（福岡）の農業委員会・JA・農地バンクに問い合わせ
- d:matchaやObubu Tea Farmsのモデルを直接視察

### Step 2: 参入形態の決定
- **最もリスクが低い**: リース（借地）+ 外部委託加工 → 初期投資500万円以下で可能
- **中程度のリスク**: 耕作放棄地の再生 + 自社加工 → 初期投資2,000万〜5,000万円
- **本格投資**: 事業承継で設備込みの茶園を引き継ぐ → 初期投資5,000万〜1億円

### Step 3: 法人設立と許可取得
- 農地所有適格法人の設立（所有する場合）
- 農地法第3条許可の申請
- 外国人要件の確認（在留資格・在留期間）

### Step 4: 段階的なスケールアップ
- 初年度: 0.5〜1haのリースでテスト栽培
- 2〜3年目: 品質確認後に2〜4haに拡大
- 5年目以降: 自社加工設備の導入を検討
