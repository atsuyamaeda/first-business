# 日本の茶園（碾茶/抹茶）後継者不在・事業承継リサーチ

調査日: 2026年3月26日

---

## 1. 後継者不在の茶農家 -- 統計データ

### 茶農家の激減

| 指標 | 2005年 | 2020年 | 減少率 |
|------|--------|--------|--------|
| 茶農家戸数 | 37,617戸 | 12,325戸 | **-67%** |
| 茶園面積 | 49,000 ha | 38,000 ha | -22% |

**出典**: 農業利益創造研究所（農林業センサスベース）

### 地域別の平均年齢

| 県 | 茶農家平均年齢 |
|----|---------------|
| 鹿児島 | 56歳 |
| 静岡 | 58歳 |
| 京都 | **60歳** |

京都（宇治・和束エリア）が最も高齢化が深刻。

### 後継者不在率の推定

- 茶農家の約**7割**が後継者未確保（農林水産省・各県調査の推定値）
- 全農業における後継者不在率は約65%（2020年農林業センサス）だが、茶は労働集約性が高く、特に碾茶（抹茶原料）は被覆・手摘みの技術が必要なため、後継者確保がさらに困難
- 和束町（京都）では過去10年で**約100世帯**の茶農家が減少
- 和束町の耕作放棄茶園: 9.4ha（2008年）→ **38.5ha**（2018年） ※10年で4倍
- 全国で年間**約300ha**の茶園が消失中

### 最も危機的な地域（後継者問題が深刻な順）

1. **京都府（宇治・和束・相楽）** -- 平均年齢60歳、小規模・急傾斜の茶園が多く機械化困難、耕作放棄地が急拡大
2. **静岡県（中山間部）** -- 牧之原台地等の大規模産地は比較的安定だが、山間部の小規模園は廃業加速
3. **福岡県（八女）** -- 高品質玉露・碾茶の産地だが高齢化進行
4. **愛知県（西尾）** -- 碾茶特化産地だが後継者不足
5. **鹿児島県（南九州市）** -- 大規模機械化農業のため比較的マシだが、製茶工場の後継者問題あり

## 3. 茶園事業承継のコスト内訳

### シナリオA: 事業承継（第三者継承）で引き継ぐ場合

事業承継が最もコスト効率が良い。既存設備・顧客・ノウハウを一括承継。

| 項目 | 概算コスト | 備考 |
|------|-----------|------|
| 譲渡対価（のれん代） | 0〜2,000万円 | 多くの場合、後継者がいないため格安〜無償 |
| 農地の賃料 | 3,000〜10,000円/10a/年 | 所有でなく賃借が主流 |
| 設備引継ぎ費 | 含まれる場合が多い | 被覆棚、茶刈機、製茶設備等 |
| 改修・修繕費 | 200万〜500万円 | 老朽化した設備の修繕 |
| 運転資金（半年分） | 300万〜600万円 | 肥料、農薬、人件費、燃料 |
| 法人設立・許可費用 | 50万〜100万円 | 農地法3条許可含む |
| **合計** | **600万〜3,200万円** | ゼロから始めるより大幅に安い |

### シナリオB: 耕作放棄茶園を再生する場合

| 項目 | 概算コスト | 備考 |
|------|-----------|------|
| 土地取得/賃借 | ほぼ無償〜格安 | 放棄地は持ち主が手放したい場合が多い |
| 開墾・整備作業 | 200万〜500万円 | 雑草・竹の除去、茶樹の剪定 |
| 被覆棚の設置（碾茶用） | 500万〜2,000万円 | 碾茶生産に必須 |
| 茶刈機 | 50万〜300万円 | 手持ち型〜乗用型 |
| 製茶は外部委託 | 加工賃で対応 | 自社設備不要 |
| 運転資金 | 300万〜600万円 | |
| **合計** | **1,000万〜3,400万円** | 収穫まで2〜3年のタイムラグあり |

### シナリオC: 土地購入 + 設備新規導入（フルスクラッチ）

| 項目 | 概算コスト | 備考 |
|------|-----------|------|
| 土地取得（2〜4ha） | 600万〜3,200万円 | 地域による |
| 被覆棚 | 500万〜2,000万円 | |
| 茶園管理機（乗用型） | 300万〜1,500万円 | |
| 製茶加工設備 | 2,000万〜5,000万円 | 蒸し機、碾茶炉、選別機等 |
| 倉庫・加工場 | 500万〜2,000万円 | |
| **合計** | **4,000万〜1.5億円** | 最もコスト高 |

### 事業承継に含まれるもの（典型的なケース）

- **有形資産**: 農地（借地権）、被覆棚、茶刈機、乗用管理機、製茶設備、倉庫、住居（付帯する場合）
- **無形資産**: 栽培ノウハウ（品種特性、施肥、被覆タイミング等）、販売先（JA、問屋、直販先）、ブランド・屋号、地域コミュニティとの関係、有機認証（取得済みの場合）
- **負の資産に注意**: 老朽設備の修繕義務、農地の固定資産税、既存借入金の引継ぎ有無

## 4. 補助金・支援制度

### (a) 新規就農者向け支援（農林水産省）

#### 就農準備資金（旧：農業次世代人材投資資金 準備型）
- **金額**: 月額12.5万円（年間150万円）
- **期間**: 最長2年間
- **対象**: 都道府県が認める研修機関で研修を受ける49歳以下の者
- **条件**: 研修終了後1年以内に就農すること

#### 経営開始資金（旧：農業次世代人材投資資金 経営開始型）
- **金額**: 月額12.5万円（年間150万円）
- **期間**: 最長3年間
- **対象**: 認定新規就農者で49歳以下
- **条件**: 前年の所得が600万円未満

#### 経営発展支援事業
- **金額**: 機械・施設等の導入費用の**3/4以内**（上限1,000万円、経営開始資金との併用可）
- **対象**: 認定新規就農者
- **用途**: 農業用機械、施設、家畜等の導入

### (b) 認定新規就農者のメリット

認定を受けると以下の優遇措置:
- 上記の経営開始資金・経営発展支援事業の対象
- 青年等就農資金（無利子融資）: **上限3,700万円**、償還期間17年以内
- 農業経営基盤強化準備金（税制優遇）
- 農地の優先斡旋

### (c) 事業承継・引継ぎ補助金（経済産業省/中小企業庁）

- **対象**: 事業承継を行う中小企業・小規模事業者
- **補助額**: 補助対象経費の**2/3以内**、上限**400万円**（廃業費を含む場合600万円）
- **類型**:
  - Ⅰ型（経営革新型）: 承継後の新たな取り組みを支援
  - Ⅱ型（専門家活用型）: M&A時の仲介手数料・デューデリジェンス費用
  - Ⅲ型（廃業・再チャレンジ型）
- **URL**: https://www.chusho.meti.go.jp/zaimu/shoukei/

### (d) 各自治体独自の支援

#### 和束町（京都府）
- 移住支援: 空き家バンク、移住体験住宅
- 茶農家研修制度あり（詳細は和束町役場に要確認）
- 京都府の「新規就農者総合対策事業」の適用あり
- **問い合わせ先**: 和束町役場 産業振興課 TEL: 0774-78-3001

#### 宇治市・宇治田原町（京都府）
- 京都府茶業会議所を通じた後継者マッチング
- 宇治茶の伝統製法の研修プログラム
- **問い合わせ先**: 京都府茶業会議所 TEL: 0774-23-7713

#### 西尾市（愛知県）
- 西尾の抹茶ブランド推進事業あり
- 碾茶生産に特化した技術研修（JAあいち中央が窓口）
- **問い合わせ先**: 西尾市役所 農林水産課、JAあいち中央

#### 八女市（福岡県）
- 八女茶の後継者育成プログラム
- 福岡県農林水産振興センターを通じた農地バンク
- **問い合わせ先**: 八女市役所 農業振興課

#### 南九州市（鹿児島県）

- 鹿児島県の「かごしま茶振興計画」に基づく支援
- **問い合わせ先**: 南九州市役所 茶業課

## 5. 地域別の茶園承継プログラム

### 和束町（京都府）-- 最重要ターゲット

**現状**:
- 宇治茶の約40%を生産する主要産地
- 800年の茶栽培の歴史
- 過去10年で約100世帯の茶農家が減少
- 耕作放棄茶園が急拡大（2008年9.4ha → 2018年38.5ha）
- 急傾斜地が多く機械化が困難 → 人手不足が深刻

**新規就農の入口**:
1. 和束町の「地域おこし協力隊」として茶農家研修（任期1〜3年、月給16〜20万円程度）
2. 研修終了後、独立就農または農家の事業承継
3. 京都府の農業大学校での短期研修（1年間）も選択肢

**実績のある受入先**:
- d:matcha Kyoto（インターン・研修受入あり）
- Obubu Tea Farms（国際的なインターン制度あり）
- 和束町茶業振興会（地元農家ネットワーク）

### 宇治エリア（京都府宇治市・宇治田原町・南山城村）

**特徴**:
- 「宇治茶」ブランドの本拠地
- 碾茶・玉露の最高級産地
- 伝統的な被覆栽培（本簀被覆）の技術が集積
- 土地価格は和束より高い

**承継ルート**:
- 京都府茶業会議所を通じたマッチング
- 宇治茶の郷づくり協議会の活動に参加して人脈構築
- 南山城村は比較的土地が安く、新規参入のハードルが低い

### 西尾（愛知県）

**特徴**:
- 日本一の碾茶（抹茶原料）産地（生産量シェア約30%）
- 平坦地が多く機械化しやすい
- あいや、丸石製茶など大手の碾茶メーカーが集積
- 契約栽培が主流（独立より既存メーカーとの連携が現実的）

**承継ルート**:
- JAあいち中央の茶業部会を通じた紹介
- 西尾市農業委員会への相談
- 碾茶メーカーとの契約栽培から始めるのが現実的

### 八女（福岡県）

**特徴**:
- 玉露日本一の産地
- 碾茶生産も増加傾向
- 山間部の伝統的な茶園が多い

**承継ルート**:
- 福岡県農林水産振興センター
- JA筑後の茶業部会
- 八女市の新規就農相談窓口

### 南九州市（鹿児島県）

**特徴**:
- 茶の生産量全国1位の自治体
- 大規模・機械化農業が主流
- 碾茶への転換が進行中（煎茶価格低迷のため）
- relay.townで実際に後継者募集案件あり

**承継ルート**:
- relay.town経由の案件に応募
- 鹿児島県地域振興公社（農地バンク）
- 南九州市茶業課に直接相談

## 6. 事業承継の成功事例（ケーススタディ）

### Case 1: d:matcha Kyoto（京都府和束町）

| 項目 | 詳細 |
|------|------|
| 創業 | 2016年 |
| 創業者 | 農学系大学卒のメンバー（田中大貴氏ら） |
| 参入方法 | 和束町に移住し、耕作放棄茶園を再生 |
| 現在の規模 | 約4haの茶園を管理 |
| ビジネスモデル | 栽培→加工→D2C販売→カフェ運営の垂直統合 |
| 取得コスト | 非公開（耕作放棄地のため土地コストは極めて低いと推測） |
| 期間 | 2016年移住→2017年茶園開始→現在までに約4haに拡大 |
| 課題 | 茶栽培技術をゼロから習得、放棄地の整備、地域コミュニティへの溶け込み |
| 差別化 | 有機栽培、品質重視（若芽のみ収穫）、D2C・輸出 |
| 教訓 | 「グローバルに戦える農業」を志向。仲介業者を排除し直販で利益率を確保 |

### Case 2: 健一自然農園（奈良県大和高原）

| 項目 | 詳細 |
|------|------|
| 代表 | 伊川健一氏（調査時35歳） |
| 参入方法 | 耕作放棄茶園を積極的に引き受け |
| 現在の規模 | 約30か所、12haの茶園 |
| 取得方法 | 5年以上放棄された茶園を「荷物だと思っているところに宝がある」として引受 |
| 栽培方法 | 自然農法・三年番茶に特化。3年間剪定なし→冬に根元から刈取→薪火焙煎 |
| 特徴 | 放棄地が3mの藪状態から再生、地域の高齢者雇用、間伐材を燃料利用 |
| 教訓 | 放棄地＝コスト削減の宝庫。ニッチ商品（三年番茶）で差別化 |

### Case 3: Obubu Tea Farms（京都府和束町）

| 項目 | 詳細 |
|------|------|
| 創業 | 2004年 |
| 代表 | 松本靖治氏 |
| 参入方法 | 和束町で茶園運営を開始 |
| 特徴 | 国際的なインターン制度（年間数十名を受入） |
| ビジネスモデル | 自社栽培 + 国際向けD2C販売 + 茶教育プログラム |
| 教訓 | 「体験」と「教育」を収益の柱にすることで小規模でも成立 |

### Case 4: 酪農の第三者継承事例（参考: 鳥取県・栃木県）

農業全般の第三者継承で参考になる事例:

- **栃木県の果樹農家**: 農業開発事務所に連絡→2週間で農園のオファー→初年度売上750万円→現在は年商1,400万円を目標
- **栃木県の酪農家**: 空き牛舎を紹介され参入。青年等就農資金3,700万円を借入→3年目で黒字化→現在年商7,000万円
- **共通点**: 行政の窓口に相談→マッチング→技術研修→独立のパターン。案件紹介まで最短2週間。

## 7. 実務的なタイムライン

### パターンA: 事業承継（最短ルート）

| 期間 | ステップ | 詳細 |
|------|---------|------|
| 月1-2 | 情報収集・プラットフォーム登録 | relay.town、バトンズ、トランビに登録。農業委員会・JAに相談 |

## 8. 農地バンクの実態と課題

### 制度概要
- 2014年に全都道府県に設置された農地中間管理機構
- 貸し手（引退農家）→ 農地バンク → 担い手（借り手）のマッチング

### 実績と課題
- 2017年度の中間目標達成: **ゼロ都道府県**
- 進捗率は当初予想の3割未満
- 農地集積率は48.7% → 58%に上昇したが、半数近くは未利用
- **見ず知らずの人への貸し出しに抵抗感**が最大の障壁
- 認知度不足（YouTubeの説明動画の再生数が8ヶ月で2,700回程度）

### 茶園への示唆
- 農地バンクで「茶園」として明示的に出ている物件は極めて少ない
- 実態は**地元の農業委員会・JA・口コミ**が主要チャネル
- 現地に足を運び、地域の人間関係に入ることが最も重要

## 10. キーポイント（まとめ）

1. **茶農家の67%が15年間で廃業**。後継者不在率は約7割。特に京都（和束・宇治）が最も深刻で、チャンス大。

2. **事業承継が最もコスト効率が良い**。600万〜3,200万円で設備・顧客・ノウハウ込みで引き継げる可能性。ゼロから始める場合の4,000万〜1.5億円と比較して大幅に安い。

3. **実際の案件は少なくオンラインに出てこない**。relay.townで茶関連は3〜4件、be-farmer.jpはゼロ、バトンズ・トランビはUI制約で確認困難。**現地の口コミ・行政窓口が最も重要なチャネル**。

4. **補助金を最大限活用すれば実質負担はさらに減る**。就農準備資金（150万円/年×2年）+ 経営開始資金（150万円/年×3年）+ 経営発展支援事業（上限1,000万円）+ 無利子融資（上限3,700万円）で合計5,000万円超の支援が可能。

5. **碾茶（抹茶原料）価格の高騰が追い風**。2025年の碾茶オークション平均8,235円/kg（前年比1.7倍）。供給不足 × 放棄地増加のギャップが参入チャンス。

6. **タイムラインは最短1年（事業承継）〜3-4年（放棄地再生）**。事業承継が時間的にも最も効率的。
