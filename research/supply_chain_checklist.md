# 産地・サプライチェーン設計：検討項目チェックリスト

---

## 1. 茶葉の産地選定

- [ ] 産地の候補（宇治 / 西尾 / 八女 / 鹿児島 / 静岡 等）
- [ ] 産地ごとの味の特徴・品質の違い
- [ ] 「日本発 = 本物の抹茶」のブランドストーリーに合う産地はどこか
- [ ] 単一産地（シングルオリジン）か、ブレンドか
- [ ] 有機栽培（オーガニック認証）の要否
- [ ] 産地との独占契約・優先契約の可能性

---

## 2. 茶農家・茶園との関係構築

- [ ] 直接取引（農家直） vs 問屋経由
- [ ] 契約形態（スポット買い / 年間契約 / 複数年契約）
- [ ] 最低ロットと初期段階での調達可能量
- [ ] 収穫時期（一番茶 / 二番茶）と品質・価格のトレードオフ
- [ ] 農家の栽培方法・覆い下栽培の期間の確認
- [ ] 将来のスケール時に安定供給できるか

---

## 3. 製茶・加工

- [ ] 石臼挽きの委託先（製茶工場の選定）
- [ ] 石臼挽きの粒度管理（「ダマにならない最高品質」の基準設定）
- [ ] 挽きたて鮮度の管理（挽いてからパッキングまでのリードタイム）
- [ ] フレーバー抹茶の配合・混合工程をどこでやるか
- [ ] 食品衛生法・HACCP対応の確認
- [ ] US展開時のFDA対応・輸出要件

---

## 4. パッキング・充填

- [ ] 個包装の充填委託先（OEM / 自社）
- [ ] 窒素充填など鮮度保持技術の採用可否
- [ ] 個包装の素材選定（遮光・防湿・環境配慮）
- [ ] 充填精度（1杯分の重量ばらつき許容範囲）
- [ ] 最低ロットと段階的な発注量の設計

---

## 5. 器・茶道具の調達

- [ ] 茶碗の製造先（窯元直 / OEM陶器メーカー）
- [ ] 茶筅の調達先（奈良・高山茶筅 / 海外製）
- [ ] 茶杓の素材・製造先
- [ ] コラボ・季節限定デザインの生産体制
- [ ] 品質検品の基準と工程

---

## 6. 品質管理

- [ ] 抹茶のグレード基準の策定（色・香り・味・粒度）
- [ ] ロットごとの品質検査フロー
- [ ] 残留農薬検査・重金属検査の頻度
- [ ] 賞味期限の設定根拠（開封前 / 開封後）
- [ ] トレーサビリティの確保（畑→消費者まで追跡可能か）

---

## 7. 倉庫・保管

- [ ] 保管倉庫の選定（温度・湿度・遮光管理）
- [ ] 抹茶の劣化リスク管理（光・熱・酸素・湿気）
- [ ] 器・茶道具と抹茶の保管場所を分けるか統合するか
- [ ] 在庫回転率の目標設定

---

## 8. 物流・配送

- [ ] 国内配送業者の選定
- [ ] 梱包設計（器の破損防止 + ブランド体験の両立）
- [ ] 配送リードタイム目標
- [ ] 送料構造（自社負担 / 顧客負担 / 一定額以上無料）
- [ ] US展開時の国際物流・通関・関税

---

## 9. コスト構造の把握

- [ ] 茶葉原価（産地・グレード別）
- [ ] 加工費（石臼挽き + 充填 + 個包装）
- [ ] 器・茶道具の1セットあたり原価
- [ ] 物流費（梱包 + 配送）
- [ ] 1杯50〜100円の目標単価に対する原価率の試算

---

## 10. リスク管理

- [ ] 天候不順・不作時の代替調達先の確保
- [ ] 単一サプライヤー依存リスクの回避策
- [ ] 為替リスク（US展開時）
- [ ] 食品リコール時の対応フロー
- [ ] サプライヤーの廃業・事業縮小リスク
