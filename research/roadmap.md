# 抹茶ビジネス ロードマップ

---

## Phase 1: マーケットリサーチ

### (a) 競合他社情報の収集・整理
- [ ] 抹茶・飲料系スタートアップの一覧化
- [ ] 各社のプロダクトライン、ターゲット顧客、販売チャネルの整理
- [ ] 競合の強み・弱みの分析

> 📎 参考資料: [beverage_startup_funding_ranking.md](./beverage_startup_funding_ranking.md)

### (b) 資金調達情報の確認
- [ ] 主要プレイヤーの資金調達額・ラウンドの把握
- [ ] 投資家の傾向分析（どの領域に資金が集まっているか）
- [ ] バリュエーション水準の把握

---

## Phase 2: 市場分析とニーズの明確化

### (a) 市場を支配するニーズの特定
- [ ] 消費者が抹茶に求める価値の調査（健康、嗜好、文化体験など）
- [ ] トレンド分析（SNS、検索ボリューム、メディア露出）
- [ ] 地域別のニーズ差異の把握

### (b) 供給者（プレイヤー）の把握
- [ ] 抹茶市場における主要プレイヤーの分類（D2C、卸、カフェチェーンなど）
- [ ] 各プレイヤーのポジショニングマップ作成
- [ ] サプライチェーン上の主要アクター（産地、加工、流通）の整理

### (c) プレイヤーごとの売上高に基づくニーズの規模測定
- [ ] 公開情報・推計に基づく各社の売上規模の整理
- [ ] ニーズカテゴリ別の市場規模推計
- [ ] 成長率の高いセグメントの特定

---

## Phase 3: 戦略策定

### (a) 市場の空白ポジションの特定
- [ ] 競合マッピングから未充足のニーズ・セグメントを抽出
- [ ] 参入障壁の評価（規制、資本、ブランド認知）
- [ ] 空白ポジションの優先順位付け

### (b) 自社プロダクトの打ち出し角度（ポジショニング）の決定
- [ ] ターゲット顧客の明確化
- [ ] 提供価値（バリュープロポジション）の策定
- [ ] 差別化要素の定義（品質、価格、体験、ブランドストーリー）
