### 抹茶の産地・グレード
- **産地**: 鹿児島県・京都府宇治地域（一番茶のみ使用）
- **栽培**: 100%オーガニック（無農薬・無化学肥料）
- **認証**: 国内外のオーガニック認証取得、レインフォレスト・アライアンス認証
- **製法**: 石臼挽き（旨味・栄養素を保持）
- **土づくり**: 微生物・バクテリアを活用した有機土壌
- **水源**: 天然湧水を使用
- **農園立地**: 外部からの化学物質汚染を排除するため、周囲から隔離された環境
- **サプライチェーン**: 土づくりから流通まで自社一貫管理

## 4. ブランド戦略

### ブランドコンセプト
> **"Matcha for a healthy and beautiful life"**
> （健康で美しいライフスタイルのための抹茶）

### ミッション
> 「最高品質のオーガニック抹茶を世界に届ける」
> オーガニック抹茶は健康とサステナビリティのバランスを実現し、より健康的なライフスタイルと地球環境に貢献する

### ターゲット層
- **若い世代（20〜30代）**: ファッション・ライフスタイル志向
- **健康志向層**: オーガニック・スーパーフードに関心がある層
- **「ソバーキュリアス」層**: アルコール代替としてのカフェイン飲料を求める層
- **訪日外国人観光客**: 表参道・原宿・渋谷エリアの観光客
- **B2B**: 卸売パートナー（カフェ、レストラン、ホテル等）

### ブランドの3つの柱
1. **オーガニック認証**: 100%オーガニック・無農薬を徹底
2. **品質と味**: 風味・色・香りの高い基準を満たすブレンド
3. **文化体験**: 茶道の作法に基づく一杯ずつの丁寧な提供

### デザイン戦略
- **ミニマリスト・モダン**: 伝統的な和風デザインを意図的に避け、現代的・都会的なデザインを採用
- **白を基調とした清潔感のある店内**: 都市環境との対比で「穏やかなオアシス」を演出
- **パッケージ**: シンプルでクリーンなデザイン、グリーンと白のブランドカラー

## 7. 競合との差別化ポイント

### THE MATCHA TOKYOのユニークな点

| 差別化要素 | 詳細 |
|-----------|------|
| **100%オーガニック特化** | 日本国内でも珍しい100%オーガニック抹茶のみを扱うブランド。国内外のオーガニック認証 + レインフォレスト・アライアンス認証の二重認証 |
| **自社一貫管理** | 土づくり → 栽培 → 収穫 → ブレンド → 石臼挽き → 流通まで一貫管理 |
| **モダンライフスタイル提案** | 伝統的な和風を避け、ファッション・ウェルネス・サステナビリティの文脈で抹茶を再定義 |
| **体験型旗艦店** | カフェ + 茶室 + ヨガスペースの複合型店舗（原宿旗艦店） |
| **グローバル展開の速度** | 2018年創業から約7年で7カ国50店舗。アジア・中東・北米へ急速展開 |
| **偽抹茶への対抗姿勢** | 海外で横行する抹茶模倣品に対し、「本物の日本産オーガニック抹茶」としてのポジショニングを明確化 |
| **一杯ずつの茶道式提供** | 注文ごとに茶道の作法に基づき、竹の茶筅で一杯ずつ点てるスタイル |

### 売上高の推定ロジック

#### 推定方法1: 店舗数ベース

```
前提条件:
- 日本国内店舗: 6店舗
- 海外店舗: 約44店舗（FC/ライセンスの可能性あり）
- 国内直営店の月商: 約200〜500万円/店（表参道・原宿など好立地、客単価800〜1,000円、1日100〜200杯）
- 海外店舗からのロイヤリティ/卸収入: 不明

推定:
- 国内カフェ売上: 6店 × 300万円/月 × 12ヶ月 = 約2.2億円
- EC/Amazon売上: 年間0.5〜1億円（推定）
- 卸売（B2B）: 年間0.5〜1億円（推定）
- 海外からの収入（ロイヤリティ/原材料卸）: 年間1〜3億円（推定）
  ※海外44店舗が全て直営でない場合、抹茶パウダーの卸売 + ロイヤリティ収入

合計推定年商: 4〜7億円程度（2025年度）
```

#### 推定方法2: 従業員数ベース

```
前提条件:
- 従業員数: 50〜100名（ソースにより異なる）

推定:
- 50名 × 600万円 = 3億円
- 100名 × 600万円 = 6億円

合計推定年商: 3〜6億円程度
```

#### 推定方法3: 伊藤園の投資規模から逆算

```
前提:
- 伊藤園は5年間で300億円のM&A・資本提携計画
- THE MATCHA TOKYOは「資本業務提携」先の1社
- マイノリティ出資と推定（経営権を取得するM&Aではない）
- 出資額は数億円〜10億円程度と推定
- スタートアップの出資比率: 通常10〜30%
- 仮に5億円の出資で20%取得 → ポストバリュエーション25億円
- 仮にPSR（売上高倍率）3〜5倍で逆算 → 売上高5〜8億円

推定年商: 5〜8億円程度
```

### 「自社グループ」の意味を解読

複数の公式・メディア情報を統合すると:

1. **公式サイト（英語）**: "THE MATCHA TOKYO **group** tea farm is certified by Rainforest Alliance"
2. **madame FIGARO記事**: 「有機栽培にこだわり、土壌づくりから加工まで一貫した茶づくりを**自社グループ内**で行っており」
3. **食品新聞**: 「原料は現在、**パートナーの農家**から調達している」
4. **さんたつ記事**: 「交渉の結果、**京都と鹿児島の農園が賛同**してくれた」
5. **JETRO**: "works closely with their **tea farmers**"
6. **MATCHA JP記事**: 「鹿児島の**自社茶園**」と記載

**結論**: THE MATCHA TOKYOは自社で農地を所有しているのではなく、京都宇治と鹿児島の**契約パートナー農家**と非常に密接な関係を構築し、土づくりの段階からTHE MATCHA TOKYOの基準に基づいた栽培管理を行っている。これを「自社グループ」と表現している。いわば**専属契約農園**のモデル。
