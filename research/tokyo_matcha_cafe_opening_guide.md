## 1. 必要な資格・免許

### (a) 食品衛生責任者【必須】

- **概要**: 飲食店を営業するすべての施設に1名以上の配置が義務
- **取得方法**: 一般社団法人 東京都食品衛生協会が実施する「食品衛生責任者養成講習会」を受講（1日・計6時間）
  - 衛生法規（2時間）
  - 公衆衛生学（1時間）
  - 食品衛生学（3時間）
  - 確認テストあり（ほぼ全員合格）
- **費用**: 約12,000円
- **期間**: 1日で取得可能（ただし予約が埋まりやすいので早めに申込）
- **免除**: 調理師・栄養士・製菓衛生師等の有資格者は講習不要
- **参考**: [東京都食品衛生協会](https://www.toshoku.or.jp/)

### (b) 防火管理者【収容人数30名以上の場合必須】

- **甲種**: 延床面積300m2以上 → 約2日間（10時間）の講習
- **乙種**: 延床面積300m2未満 → 約1日（5時間）の講習
- **費用**: 7,000〜8,000円程度
- **申込先**: 一般財団法人 日本防火・防災協会、または所轄消防署

### (c) 飲食店営業許可【必須】

- **管轄**: 店舗所在地の保健所
- **申請費用**: 東京都で約16,000〜18,000円（2026年時点）
- **有効期間**: 許可の種類と施設基準により異なる（通常5〜8年、施設基準を満たせば最長8年）
- **参考**: [東京都保健医療局 営業許可・届出](https://www.hokeniryo1.metro.tokyo.lg.jp/shokuhin/kyokatodokede/youshiki.html)

### (d) その他（該当する場合）

| 許可・届出 | 条件 |
|-----------|------|
| 菓子製造業許可 | 自家製スイーツを製造・販売する場合 |
| アイスクリーム類製造業 | 自家製ジェラート・ソフトクリームを製造する場合 |
| 深夜酒類提供飲食店届出 | 深夜0時以降にアルコールを提供する場合（所轄警察署へ届出） |
| 食品衛生法に基づく届出 | 2021年改正により、営業許可対象外の業種でも届出が必要なケースあり |

## 2. 物件探しのポイント

### (a) 立地選定の考え方

抹茶カフェ（おしゃれ系・インスタ映え）の場合、以下のエリアが有力候補:

| エリア | 特徴 | 坪単価目安（賃料） |
|--------|------|-------------------|
| 表参道・原宿 | ファッション・トレンド発信地、海外観光客多い | 3〜6万円/坪 |
| 渋谷 | 若年層集客力、再開発で新施設多い | 2.5〜5万円/坪 |
| 中目黒 | おしゃれカフェ激戦区、目黒川沿い集客 | 2〜4万円/坪 |
| 自由が丘 | カフェ文化の街、女性客多い | 2〜3.5万円/坪 |
| 代官山・恵比寿 | 高感度層、落ち着いた雰囲気 | 2.5〜4.5万円/坪 |
| 浅草・蔵前 | インバウンド需要、和の世界観と相性良い | 1.5〜3万円/坪 |

### (b) 物件のスペック目安

- **坪数**: 10〜20坪（カウンター+テイクアウト中心なら10坪前後、イートイン重視なら15〜20坪）
- **席数**: 10〜25席が管理しやすい（Cha Cha Matchaは20席前後）
- **路面店 vs 2階以上**: 路面店が圧倒的に有利（視認性・テイクアウト需要）。2階以上は賃料安いが集客に工夫必要
- **1階路面+テラス席**: 抹茶ラテのテイクアウトを考えると理想的

### (c) 居抜き vs スケルトン

| 項目 | 居抜き物件 | スケルトン物件 |
|------|-----------|---------------|
| **初期費用** | 低い（200〜500万円の造作譲渡料+改装費） | 高い（内装工事に坪30〜60万円） |
| **工期** | 短い（2〜4週間） | 長い（1〜2ヶ月） |
| **自由度** | 低い（前テナントのレイアウトに制約） | 高い（完全オリジナル設計可能） |
| **おすすめ** | 資金・時間を節約したい場合 | ブランド世界観を徹底したい場合 |

**抹茶カフェとしての推奨**: Cha Cha Matchaのようなブランド体験重視のカフェの場合、**スケルトンまたは大幅改装の居抜き**がベスト。ただし初期投資が膨らむため資金計画を慎重に。

### (d) 物件探しの主要サイト

- [飲食店ドットコム](https://www.inshokuten.com/) - 飲食店専門
- [居抜き店舗.com](https://www.i-tenpo.com/) - 居抜き専門
- [テンポスマート](https://www.temposmart.jp/) - 居抜き物件
- [居抜き市場](https://inuki-ichiba.jp/) - 居抜き・貸店舗

### (e) 契約時の注意点

- **保証金（敷金）**: 賃料の6〜12ヶ月分が一般的（東京都心）
- **礼金**: 1〜2ヶ月分
- **仲介手数料**: 賃料1ヶ月分
- **フリーレント交渉**: 内装工事期間中の賃料免除を交渉（1〜3ヶ月が目安）
- **定期借家 vs 普通借家**: 定期借家は更新がないため注意
- **用途制限**: 飲食店営業可能か必ず確認
- **排気・給排水**: 厨房の設置が可能かどうか（特に上階の場合）

## 3. 事業計画書の作成

### (a) なぜ必要か

- 融資審査の最重要書類（日本政策金融公庫等）
- 自分の事業の実現可能性を客観的に検証するツール
- 補助金申請にも必須

### (b) 記載すべき項目

1. **創業の動機・事業の目的**: なぜ抹茶カフェか、どんな問題を解決するか
2. **経営者の経歴・スキル**: 飲食業経験、関連資格、強み
3. **事業コンセプト**: ターゲット顧客、提供価値、差別化要素
4. **市場分析**: 抹茶市場の成長性、競合分析
5. **メニュー・サービス内容**: 主力メニュー、価格帯、提供スタイル
6. **販売戦略**: 立地戦略、SNSマーケティング、テイクアウト/デリバリー
7. **収支計画**:
   - 初期投資額（物件取得費、内装工事費、設備費、運転資金）
   - 月次売上見込み = 席数 x 回転数 x 客単価 x 営業日数
   - 月次経費（原材料費、人件費、家賃、光熱費、雑費）
   - 損益分岐点の算出
8. **資金計画**: 自己資金、融資額、返済計画

### (c) テンプレート入手先

- [日本政策金融公庫 創業計画書テンプレート](https://www.jfc.go.jp/n/service/dl_kokumin.html) - 融資申請に使う公式フォーマット

### (a) 日本政策金融公庫【最有力】

- **新規開業・スタートアップ支援資金**
  - 融資限度額: 最大7,200万円（うち運転資金4,800万円）
  - 原則無担保・無保証人
  - 据置期間: 最大5年（元本返済不要、利息のみ）
  - 金利: 年2〜3%台（2026年時点の目安）
  - **自己資金要件**: 創業資金総額の1/10以上が目安（実態としては1/3以上あると通りやすい）

- **参考**: [日本政策金融公庫](https://www.jfc.go.jp/)

- 東京都や各区が実施する制度融資

- **東京都中小企業制度融資**: [東京都産業労働局](https://www.sangyo-rodo.metro.tokyo.lg.jp/)

## 5. 内装設計・施工の流れ

### (a) 全体フロー（約3〜4ヶ月）

```
コンセプト確定 → 設計会社選定（2〜3社相見積もり）→ 設計・図面作成（3〜4週間）
→ 保健所事前相談 → 施工業者選定・見積もり → 着工（4〜8週間）
→ 保健所検査 → 引渡し → オープン準備
```

### (b) 設計のポイント（おしゃれ抹茶カフェの場合）

- **世界観の統一**: 和モダン × ミニマル × インスタ映え
- **カラースキーム**: 抹茶グリーン、白、ウッド、アースカラー
- **フォトスポット設計**: SNS映えする壁・カウンター・ディスプレイ
- **テイクアウトカウンター**: 入口付近に独立設置（回転率UP）
- **照明**: 自然光を活かしつつ、商品が映える照明計画
- **サイン・ロゴ**: 外観の視認性。ネオンサインやロゴウォールも有効

### (c) 保健所の施設基準（主なもの）

- 調理場と客席の区画分け
- 手洗い設備（手洗い専用の流し）の設置
- 食器洗浄設備（2槽以上のシンク）
- 冷蔵・冷凍設備（温度計付き）
- 防虫・防鼠設備
- 換気設備
- 床・壁・天井: 清掃しやすい材質（タイル、ステンレス等）
- 従業員用更衣室

> **重要**: 設計段階で必ず管轄保健所に事前相談し、図面を確認してもらうこと。工事後に基準不適合が発覚すると手戻りが発生する。

### (d) 費用目安（15坪の場合）

| 項目 | 居抜き改装 | スケルトン新装 |
|------|-----------|---------------|
| 設計費 | 30〜60万円 | 50〜100万円 |
| 内装工事（床・壁・天井・塗装） | 100〜200万円 | 200〜450万円 |
| 電気・空調・給排水工事 | 50〜100万円 | 150〜300万円 |
| 厨房設備 | 100〜200万円 | 200〜400万円 |
| 家具・什器 | 50〜100万円 | 80〜150万円 |
| サイン・外装 | 20〜50万円 | 30〜80万円 |
| **合計** | **350〜710万円** | **710〜1,480万円** |

## 6. 保健所への届出・検査

### (a) 手続きフロー

```
1. 事前相談（設計段階で図面持参）
   ↓
2. 営業許可申請書の提出（工事完了の約10日前まで）
   - 必要書類:
     - 営業許可申請書
     - 施設の構造・設備を示す図面（2通）
     - 食品衛生責任者の資格証明書
     - 水質検査成績書（貯水槽・井戸水使用の場合）
     - 登記事項証明書（法人の場合）
   ↓
3. 施設の確認検査（保健所職員が現地確認）
   - 営業者の立ち会いが必要
   - 施設基準を満たしているか確認
   ↓
4. 営業許可証の交付
   - 検査合格から数日〜1週間程度で交付
   ↓
5. 営業開始
```

### (b) オンライン申請

- 厚生労働省の「食品衛生申請等システム」によりオンライン申請も可能
- [食品衛生申請等システム](https://ifas.mhlw.go.jp/)

### (c) 消防署への届出

- **防火対象物使用開始届出書**: 使用開始の7日前までに所轄消防署へ届出
- **防火管理者選任届出**: 収容人数30名以上の場合

## 7. メニュー開発・仕入れ先確保

### (a) メニュー構成の考え方

| カテゴリ | メニュー例 | 客単価目安 |
|---------|-----------|-----------|
| シグネチャードリンク | 濃茶ラテ、抹茶エスプレッソ、季節限定ラテ | 700〜1,000円 |
| 定番ドリンク | 抹茶ラテ（HOT/ICE）、ほうじ茶ラテ、煎茶 | 500〜800円 |
| 抹茶以外 | チャイ、ゆず茶、和紅茶、ノンカフェイン | 500〜700円 |
| フード | 抹茶ティラミス、抹茶パフェ、どら焼き | 500〜900円 |
| テイクアウト | テイクアウトラテ、ボトルドリンク | 500〜800円 |
| 物販 | 抹茶パウダー、茶器、オリジナルグッズ | 1,000〜5,000円 |

### (b) メニュー開発のステップ

1. コンセプトに基づく全体設計（ドリンク:フード = 7:3が目安）
2. 原価率の計算（目標: ドリンク25〜30%、フード30〜35%）
3. 試作・試飲の繰り返し
4. 食材ロスの最小化設計
5. オペレーション効率（1杯の提供時間、工程数）
6. 写真映えする盛り付け・カップデザイン

### (c) 抹茶の仕入れ先候補

| 仕入れ先 | 特徴 | 対応 |

| **あいや（AIYA）**（愛知西尾） | 西尾抹茶の老舗、業務用に強い | 業務用卸 |
| **丸久小山園**（京都宇治） | 高品質宇治抹茶、カフェ業務用あり | 業務用卸 |
| **築地 河内園** | 築地場外市場、業務用大ロット対応 | 業務用卸 |

## 10. 開業届・税務関連

### (a) 個人事業主の場合

| 届出 | 提出先 | 期限 | 備考 |
|------|--------|------|------|
| 個人事業の開業届出書 | 納税地の税務署 | 事業開始日から1ヶ月以内（※2026年1月〜は確定申告期限まで） | e-Taxでオンライン提出可 |
| 青色申告承認申請書 | 同上 | 事業開始日から2ヶ月以内 | 65万円の特別控除を受けるために必須 |
| 給与支払事務所等の開設届出書 | 同上 | 給与支払開始から1ヶ月以内 | スタッフ雇用時 |
| 源泉所得税の納期の特例の承認に関する申請書 | 同上 | 随時 | 従業員10人未満なら半年に1回の納付に |

### (b) 法人の場合（株式会社・合同会社）

- 法人設立届出書、定款、登記簿謄本等を税務署・都税事務所等に提出
- 法人住民税・法人事業税の届出（都税事務所）
- 社会保険の加入手続き（年金事務所）
- 雇用保険・労災保険の手続き（ハローワーク・労働基準監督署）

### (c) 個人 vs 法人の選択

| 項目 | 個人事業主 | 法人（合同会社） |
|------|-----------|----------------|
| 設立費用 | 0円 | 約10万円〜 |
| 手続きの簡易さ | 簡単 | やや複雑 |
| 税率 | 所得税（累進課税5〜45%） | 法人税（約23%） |
| 信用力 | やや低い | 高い |
| **推奨** | まず個人で開始、利益800万円超えたら法人化検討 | |

### (d) その他の届出

- **消費税**: 開業2年間は原則免税（課税売上1,000万円以下）。ただしインボイス制度への対応要検討
- **個人事業税**: 東京都主税局への届出（飲食業は個人事業税5%）
- **労働保険**: 従業員雇用時はハローワーク・労基署へ届出
