# 抹茶の官能評価・テイスティング 完全ガイド

---

## 1. 公式評価基準（日本茶専門家が使用する項目）

### 1-1. 外観／Appearance

**粉末の色（色沢）**
- **上級品**: 冴えた鮮やかな緑色（ビビッドグリーン）。光沢がある。くすみがない
- **中級品**: やや黄味がかった緑、または白っぽい緑
- **下級品**: 赤みがある、茶色がかっている、黄色っぽい、くすんでいる
- 色の鮮やかさは、被覆栽培によるクロロフィル含有量に直結する
- 粒度が細かいほど色が均一に見える

**粉末の細かさ（粒度）**
- 抹茶はおよそ10ミクロン程度の微粉末が理想
- **上質**: タルクのように超微細。指で触ると滑らかで肌に密着する感覚
- **下質**: 砂っぽい、ザラザラした感触。ダマになりやすい
- 石臼挽き（ゆっくり挽く）は熱による酸化を防ぎ、粒度が均一になる
- 工業的粉砕は粒度にバラつきが出やすい

### 1-2. 水色（すいしょく）／Liquor Color

お湯で点てた際の液体の色を評価する項目。

- **上級品**: 鮮やかな緑色で濁りが少ない。深みがある
- **中級品**: やや黄緑色
- **下級品**: 黄色っぽい、茶色がかっている、くすんでいる
- 被覆期間が長い茶葉ほど水色が鮮緑になる
- 碾茶の審査では「から色」（浸出後の茶殻の色）も別途評価項目となる

### 1-3. 香り（香気）／Aroma

**覆い香（おおいか）**
- 抹茶に最も特徴的な香り。被覆栽培（遮光）により生成される
- 成分: ジメチルスルフィド（DMS）が主成分
- 「青海苔のような」「海苔や昆布のような磯の香り」と表現される
- 上質なほど覆い香が豊かで品格がある

**その他の香りの種類**
- **花様の香り**: すみれのような重厚な花の香り（玉露・高級抹茶に見られる）
- **甘い香り**: 柏もちのような甘い穀物様の香り
- **若葉の香り**: フレッシュな若葉、きぬさや、ゆで枝豆のような香り
- **焙煎香**: 適切な火入れによる香ばしさ（碾茶の仕上げ工程由来）

**欠点臭**
- 青臭さ（生臭い・草っぽい）: 蒸し工程の不備
- 枯れ草臭・藁臭: 古くなった抹茶、劣化の兆候
- 金属臭: 酸化が進んだ状態
- 異臭: 保管時の臭い移り（抹茶は非常に臭いを吸着しやすい）

### 1-4. 味（滋味）／Taste

**旨味（うまみ）**
- アミノ酸（特にテアニン）由来
- 被覆栽培により増大する。上級品ほど旨味が濃厚
- 舌全体に広がる、深いコクのある味わい

**甘味**
- 旨味と一体で感じることが多い
- 上級品は最初に旨味・甘味が立ち、後から渋味がおだやかに来る

**苦味**
- カフェイン由来が主
- 適度な苦味は複雑さと奥行きを与える
- 過度な苦味は品質の低さや湯温の高すぎを示す

**渋味（しぶみ）**
- カテキン類（タンニン）由来
- 上級品は渋味が少ない（被覆によりカテキン生成が抑制されるため）
- 下級品は強い渋味・苦味が先に来て旨味が弱い

**コク（ボディ）**
- 味の総合的な厚み・深み
- 上級品は「まろやか」で「後味が良い」
- 口中に残る余韻の長さも評価対象

**品質と味のグラデーション**
| 品質 | 旨味 | 甘味 | 苦味 | 渋味 | コク | 後味 |
|------|------|------|------|------|------|------|
| 上級 | 濃厚 | 豊か | 少ない | 少ない | まろやか | 良い |
| 中級 | 中程度 | 中程度 | やや有 | やや有 | 普通 | 普通 |
| 下級 | 薄い | 少ない | 強い | 強い | 薄い | 雑味残る |

### 1-5. 泡立ち／Foam Quality

- **上質な泡**: きめ細かく均一な微小泡（マイクロフォーム）。クリーミーな層が2-3分持続
- **下質な泡**: 大きな泡がすぐに消える。泡がほとんど立たない
- 泡はPickering安定化（超微細粒子が気泡壁をコーティング）で持続する
- 5-10μmの超微粒子が安定した泡を形成。粗い粒子ではこの効果が得られない
- 泡の径が半分になると、泡の消滅速度は約4倍遅くなる
- 泡は苦味を和らげ、旨味・甘味を引き立てる効果がある
- 濃茶（こいちゃ）は泡立てず練る。薄茶（うすちゃ）は泡を立てる

## 2. プロフェッショナルな審査制度

### 2-1. 全国茶品評会

日本最大の茶の品評会。農林水産省が主催。

**審査対象茶種（8部門）**
1. 普通煎茶
2. 深蒸し煎茶
3. かぶせ茶
4. 玉露
5. てん茶（碾茶）← 抹茶の原料
6. 蒸し製玉緑茶
7. 釜炒り茶
8. （その他）

**審査項目と配点（合計200点満点・減点法）**

普通煎茶の場合:
| 項目 | 配点 |
|------|------|
| 外観 | 20点 |
| 香気 | 70点 |
| 水色 | 30点 |
| 滋味 | 80点 |
| **合計** | **200点** |

碾茶（てん茶）の場合:
| 項目 | 配点 |
|------|------|
| 外観 | 40点 |
| 香気 | 65点 |
| 水色 | 20点 |
| 滋味 | 65点 |
| から色 | 10点 |
| **合計** | **200点** |

**注目ポイント**:
- 碾茶は他の茶種と異なり「から色」（浸出後の茶殻の色）が独立した審査項目
- 碾茶は外観の配点が高い（40点）。抹茶にしたときの挽き色を推定するため
- 煎茶は滋味の配点が最も高い（80点）が、碾茶は香気と滋味が同配点（各65点）
- 覆い香の評価が碾茶では特に重要

**審査方法（減点法）**
- 最高品質のものを満点とし、そこから1点または2点ずつ減点
- 隣り合うサンプルを比較して明確な差がある部分で減点
- 全項目の合計点で順位を決定
- 1等～3等を入賞とする

**受賞ランク**
- 農林水産大臣賞（最高位）
- 農産局長賞
- 産地賞

**2025年（第79回）碾茶部門結果例**:
- 1位: 古川芳嗣（宇治市五ヶ庄）200点（満点）
- 2位: 古川芳嗣（宇治市五ヶ庄）199点
- 3位: 小山元也（宇治市）199点
- 上位5位までが全て宇治市からの出品

**審査員の講評**:
> 受賞茶は「色沢は鮮やかで明るい緑、碾茶特有の被覆による芳香、水色には深みと濃さがあり、旨味が豊富で内質が優れている」

### 2-2. 関西茶品評会

京都府茶業会議所が主催。宇治茶の産地として特に碾茶の審査に実績がある。

### 2-3. 審査点数の読み方

**200点満点中の目安（碾茶）**:
- 198-200点: 最高級。全ての項目でほぼ完璧。農林水産大臣賞レベル
- 195-197点: 一等相当。覆い香・旨味が突出して優れる
- 190-194点: 二等相当。全体的に高品質だが一部に改善余地
- 185-189点: 三等（入賞ライン）。品質は良いが上位とは明確な差
- 185点未満: 入賞外。何らかの項目に顕著な弱点がある

## 3. 実践的なテイスティング方法

### 3-1. テイスティングセッションの準備

**必要な道具**
- 拝見盆（黒色の漆塗りトレー）: 粉末の色を見やすくする
- 拝見茶碗（白磁の茶碗）: 水色を正確に確認するため白色
- 茶杓または計量スプーン
- 茶筅（80-120本立て）
- お湯（温度計付きポット）
- 水（軟水を推奨）
- テイスティングスプーン
- 記録用紙・ペン
- 白い紙（粉末の色比較用）

**環境条件**
- 無臭の空間（香水・食べ物の匂いを排除）
- 自然光または色温度の安定した照明
- 午前中が味覚・嗅覚のコンディションが良い
- 直前にコーヒー・香辛料など強い味のものを避ける

### 3-2. 標準的な評価手順

**Step 1: 外観評価**
1. 拝見盆に各サンプルの粉末を約2g載せる
2. 色の鮮やかさ、均一性、光沢を比較
3. 指で触り、粒度の細かさ・滑らかさを確認
4. 乾燥状態での香りを確認

**Step 2: 薄茶で点てて評価（標準条件）**
- 抹茶: 2g
- 湯温: 80°C（評価用。70-80°Cが味のバランスが最も良い）
  - 注意: 品評会では欠点を見つけやすくするために熱湯を使う場合もある
- 湯量: 60ml
- 茶筅で「W」または「M」字を描くように素早く20-30秒泡立てる

**Step 3: 水色の評価**
- 白磁の茶碗で液体の色を確認
- 鮮緑色の度合い、濁り、透明感を記録

**Step 4: 香りの評価**
- 点てた直後に素早く香りを嗅ぐ（冷めると香りが飛ぶため速度が重要）
- 覆い香の強さ、甘い香り、花様の香りなどを記録
- 欠点臭（青臭さ、枯れ草臭、金属臭）がないか確認

**Step 5: 味の評価**
- スプーンで少量を口に含む
- 舌の上で転がし、旨味・甘味・苦味・渋味のバランスを確認
- 後味（余韻）の質と長さを確認
- コク（ボディの厚み）を評価

**Step 6: 泡の評価**
- 泡のきめ細かさ（微小泡か大きな泡か）
- 泡の均一性
- 持続時間（2-3分持続すれば良質）

### 3-3. 濃茶での評価（上級者向け）

濃茶は品質の良し悪しがより顕著に出るため、高品質の見極めに最適。

- 抹茶: 4g
- 湯温: 約80°C
- 湯量: 40ml
- 泡立てず、茶筅でゆっくり練るように混ぜる
- 上級品は濃茶でもまろやかで旨味が凝縮される
- 下級品は濃茶にすると苦渋味が耐えられないほど強くなる

### 3-4. テイスティング用語・表現方法

**緑茶キャラクターホイール（三井農林 お茶科学研究所開発）**

55種の緑茶サンプルから、ティーテイスターと研究員が官能評価を実施して構築。

- **香り**: 10系統43語
  - Green系: 若葉、きぬさや、ゆでほうれん草、青草
  - Dry系: 柏もち、穀物様
  - Marine系: 海苔、昆布（覆い香系）
  - Floral系: すみれ
  - Bean系: ゆで枝豆
  - 他5系統

- **味**: 8語（旨味、甘味、苦味、渋味、コク、キレ等を反映）

- **水色**: 7語（液体の色を表す表現）

**よく使われる専門表現**
| 日本語 | 英語相当 | 説明 |
|--------|----------|------|
| 覆い香 | Covered aroma / Ooika | 遮光栽培由来の海苔様の香り |
| 香高（かだか） | Fragrant, aromatic | 香りが高く上品な状態 |
| まろやか | Mellow, smooth | 口当たりが柔らかく角がない |
| コク | Body, depth | 味の厚み・深み |
| キレ | Clean finish | 後味の切れの良さ |
| 冴え | Brightness, clarity | 色の鮮やかさ、味のクリアさ |
| 青海苔様 | Nori-like | 覆い香の一形態 |
| 火香（ひか） | Roasted aroma | 火入れ由来の香ばしさ |
| 青臭い | Grassy, raw | 蒸し不足による未熟な香り |
| むれ香 | Stale/stuffy | 保管不良による劣化臭 |

## 4. 品質欠点の識別

### 4-1. 劣化・鮮度低下

| 指標 | 新鮮 | 劣化 |
|------|------|------|
| 色 | 鮮やかな緑 | 黄色、茶色、カーキ色 |
| 香り | 覆い香が豊か | 枯れ草臭、金属臭、無臭 |
| 触感 | ふわふわ、柔らかい | 砂のようにザラザラ |
| 味 | 旨味豊か、まろやか | 苦味強い、味が薄い |
| 泡 | きめ細かく持続 | 泡が立たない、すぐ消える |

**原因**: 空気（酸素）、光、熱、湿気、臭い移り
- カテキン類が酸素と反応し、風味変化と色の褪色を引き起こす
- 粉末状のため表面積が大きく、茶葉よりも酸化が格段に速い

### 4-2. 粉砕不良

- **症状**: ザラザラした口当たり、不均一な色、泡が立ちにくい
- **原因**: 工業的高速粉砕による発熱酸化、または刃の磨耗
- **良質な石臼挽き**: 1時間で約40g（薄茶10杯分）のペース。ゆっくり挽くことで発熱を防ぐ

### 4-3. 被覆不足

- **症状**: 旨味が薄い、渋味が強い、色が黄緑～黄色
- **原因**: 遮光期間が短い、または遮光率が低い
- テアニン→カテキンへの変換が進み、渋味優位になる

### 4-4. 蒸し工程の不備

- **蒸し不足**: 青臭い、生っぽい香り
- **蒸し過ぎ**: 色がくすむ、香りが弱い

### 4-5. 保管不良

- **高温多湿**: 覆い香が「傷んだ香り」に変質し、色が赤みを帯びる
- **光**: 退色（色が抜ける）
- **臭い移り**: 他の食品の匂いを吸着（密封不十分）
- **湿気**: ダマになる、カビのリスク

## 5. トレーニング・資格

### 5-1. 日本茶インストラクター認定試験

NPO法人日本茶インストラクター協会が認定する日本茶の最高位資格。

**受験資格**: 満20歳以上（通信講座受講は推奨だが必須ではない）

**試験構成**:
- **第一次試験（筆記・11月）**: 5択のマークシート方式
  - 出題範囲: 茶の歴史、茶業概要、栽培、製造法、利用、化学、品質審査と鑑定、健康科学、淹れ方、伝え方の基本
- **第二次試験（実技・翌2月、一次合格者のみ）**:
  - 茶鑑定試験（約35分）: 荒茶の外観鑑定、仕上げ茶の品質外観鑑定、茶種判定、荒茶品質内質鑑定
  - インストラクション実技試験（約30分）: 指定された茶種の淹れ方を試験監督に実演・説明

**茶鑑定試験の内容**:
- 5種類程度の乾燥茶葉を並べ、茶種を当てる
- 品質の良い順に並べる
- 摘まれた時期が早い順に並べる
- 見た目・色・香り・感触から総合判断

**費用**:
- 受験料: 22,000円（税込）
- 通信講座: 77,000円（税込・任意）

**合格率**: 約35%

**試験会場**: 札幌、東京、静岡、名古屋、京都、福岡、鹿児島

**関連資格**:
- **日本茶アドバイザー**: インストラクターの下位資格。基礎的な日本茶知識
- **茶審査技術**: 全国茶審査技術競技大会での段位認定（茶師No.1を決める大会）

### 5-2. パレット（味覚）トレーニング法

**基本トレーニング**
1. **五味識別訓練**: 旨味・甘味・苦味・渋味・塩味の各成分の閾値を確認
   - テアニン溶液（旨味）、砂糖水（甘味）、カフェイン溶液（苦味）、タンニン溶液（渋味）で訓練
2. **ブラインドテイスティング**: 目を閉じて香りに集中。視覚情報を排除して嗅覚を鍛える
3. **同時比較**: 1セッションで5-10種を比較し、差異を言語化する訓練
4. **記録習慣**: 毎回テイスティングノートに記録。語彙を増やす

**推奨比較エクササイズ**

**(a) 同品種・異産地**
| 品種 | 産地A | 産地B | 産地C |
|------|-------|-------|-------|
| やぶきた | 宇治 | 西尾 | 鹿児島 |
| おくみどり | 京都 | 静岡 | 鹿児島 |

- テロワール（土壌・気候・標高）が味に与える影響を体感
- 宇治は香り・旨味・余韻に優れ、西尾はコク・冴えた緑・泡立ちが特徴、鹿児島は品種多様性

**(b) 同産地・異品種**
| 産地 | 品種A | 品種B | 品種C |
|------|-------|-------|-------|
| 宇治 | さみどり | やぶきた | おくみどり |
| 鹿児島 | やぶきた | おくみどり | あさつゆ |

- 品種固有の特性を理解する
- さみどり: 柑橘類・青菜類のような芳醇な香り。色鮮やかな緑。宇治の代表品種
- おくみどり: 癖のない自然な甘み、上品。色が濃く味もしっかり
- やぶきた: 日本茶の約75%を占める。バランスが良く標準的な味わい

**(c) 同品種・異等級**
- 同じ生産者・同品種で、特選・上選・標準・料理用を飲み比べ
- 価格と品質の相関を体感する最も効果的な方法

**(d) 鮮度比較**
- 開封直後 vs. 開封後1週間 vs. 開封後1ヶ月
- 劣化プロセスを実体験で理解

**(e) 湯温比較**
- 同じ抹茶を60°C、70°C、80°C、90°Cで点てて比較
- 温度が味のバランスにどう影響するかを学ぶ
- 高温ほどカテキン（苦渋味）が抽出されやすい

### 5-3. 日本の等級分類

| 等級 | 用途 | 特徴 |
|------|------|------|
| 特選（とくせん） | 濃茶 | 最高品質。旨味が極めて濃厚で苦渋味がほぼない |
| 上選（じょうせん） | 薄茶（上級） | 優れた香味バランス |
| 標準（ひょうじゅん） | 薄茶（普通） | 日常使いに適した品質 |
| 料理用（りょうりよう） | 製菓・料理 | 苦渋味が強く、加工用途向け |
