## 2. 主要産地 詳細プロファイル

### 2-1. 宇治（京都府）

**基本情報**
- 地域: 京都府南部（宇治市、和束町、宇治田原町、城陽市、八幡市 等）
- ※「宇治茶」の定義上、京都・奈良・滋賀・三重の4府県産茶葉を京都府内で京都府内業者が宇治地域の製法で加工したものも含む

**気候特性**
- 温暖湿潤で年間降水量が多い
- 宇治川から立ち上る朝霧が天然の遮光効果を持ち、新芽を霜害から保護
- 昼夜の寒暖差がアミノ酸の蓄積を促進

**土壌**
- 宇治川の沖積扇状地: 排水性が良く、ミネラルを豊富に含む
- 有機物の堆積が豊かな表土

**標高**
- 平地〜丘陵地（和束町などの山間部は100〜300m程度）

**主要品種**
- **さみどり**: 宇治在来種。鮮やかな緑色と繊細な旨味。柑橘系・青野菜を思わせる香気。手摘みに適した直立型の樹形
- **あさひ**: 極早生。さみどりより7日早く摘採可能。強い覆い香（海苔様の香り）。極めて優れた香味
- **うじひかり**: 宇治生まれ・宇治育ち。独特の揮発性アロマが被覆栽培で素晴らしい覆い香に変化
- **ごこう**: 旨味が強く、濃厚な味わい

**風味プロファイル**
- 深い旨味（umami）、クリーミーな舌触り、繊細な甘味
- 苦味は最小限
- 鮮やかな翡翠色（jade-green）
- 余韻が長い

**歴史的意義**
- **日本抹茶の発祥地**: 鎌倉時代に栄西が中国から茶の種を持ち帰り、明恵上人が栂尾に植えたのが起源。その後、宇治が最適産地として発展
- **800年以上の歴史**: 室町時代には「宇治七名園」が確立
- **本簀（ほんず）栽培**: 伝統的な葦簀と藁を使った被覆栽培法が現存
- **品評会の圧倒的強さ**: 全国茶品評会てん茶部門で上位10席を京都が独占。30席中18席を占める

**市場ポジション**
- 全国生産量の約3%（量）だが最高のブランドプレミアム
- 茶道用・最高級儀式用として唯一無二の地位
- 2025年4月の霜害で手摘み碾茶生産が前年比40%減（10,216kg→6,140kg）
- オークション価格は116%上昇し¥43,330/kgに

### 2-2. 西尾（愛知県）

**基本情報**
- 地域: 愛知県西尾市、安城市、旧幡豆郡吉良町

**気候特性**
- 温暖で湿度が適度
- 三河湾からの海風による安定した気候
- 茶栽培に理想的な穏やかな環境

**土壌**
- 砂壌土（sandy loam）: 窒素含有量が高く、旨味を強化
- 有機肥料による計画的な施肥管理
- ハウス型の被覆栽培施設による年間通じた品質安定

**標高**
- 約600m（一部）、平地〜丘陵地

**主要品種**
- **やぶきた**: 約50%
- **さみどり**: 約30%
- **おくみどり**: 約20%
- 品種の組み合わせにより収穫期間を分散し安定供給を実現

**風味プロファイル**
- 鮮やかな緑色が最大の特徴
- 苦味がマイルドで、植物的（vegetal）な味わい
- 甘味と旨味のバランスが良い
- ナッティーで旨味のある風味

**歴史的意義**
- 約140年の碾茶生産の歴史
- 碾茶の生産に95%を集中する日本唯一の「抹茶特化」産地
- 西尾茶協同組合による組織的な品質管理

**市場ポジション**
- 自治体単位では日本最大の抹茶生産地（年間約290トン）
- 全国シェア約20%
- 高級スイーツ店、パティスリーからの指名買いが多い
- ロット間の品質の均一性が商業利用で高く評価される
- ラテ・スイーツ用途で最もバランスの取れた選択肢

### 2-3. 八女（福岡県）

**基本情報**
- 地域: 福岡県八女市、八女郡星野村、黒木町 等

**気候特性**
- **昼夜の大きな寒暖差**: 山間部特有。アミノ酸（テアニン）の蓄積を促進
- **矢部川からの朝霧**: 河川流域に頻繁に発生し、天然の遮光効果
- 内陸性気候で冬は冷え込む

**土壌**
- 筑後川・矢部川の沖積平野: 落葉などの堆積物が豊富な肥沃な土壌
- カリウム・リンが豊富 → フローラルなアロマとマイルドな甘味に寄与

**標高**
- 200〜300m（山間部。霧が頻繁に発生する高度帯）

**主要品種**
- やぶきた、おくみどり、さえみどり、さみどり、かなやみどり
- やまかい、おくゆたか、ごこう、あさつゆ

**風味プロファイル**
- **濃厚な甘味と深い旨味**: 他産地を凌ぐアミノ酸含有量
- 苦味・渋味が極めて少ない
- 「液体の旨味」と表現されるスープ的な濃厚さ
- フルボディでクリーミーなテクスチャ、余韻の長い甘味
- 深みのある、ややトースティーな風味ノート

**歴史的意義**
- **日本茶栽培の発祥地**: 1191年、栄西禅師が中国から持ち帰った茶の種を背振山に植えたのが日本で最初の茶栽培とされる
- **玉露の最高峰**: 「八女伝統本玉露」はGI（地理的表示）登録済み。全国の玉露生産の約45%を占める
- **本簀（ほんず）被覆**: 自然素材（藁・葦簀）による伝統的被覆栽培法を維持。合成資材ではなく天然素材にこだわる
- 20〜30日間の長期被覆により L-テアニン含有量が突出

**市場ポジション**
- 全国シェアは3-4%と小さいが、超高品質帯
- 茶道向け・高級パティスリー向けで台頭中
- 品評会での評価も高い
- プレステージの上昇傾向

### 2-4. 鹿児島県

**基本情報**
- 地域: 知覧（南九州市）、霧島市、枕崎市、志布志市 等
- 広大な平坦地に大規模茶園が広がる

**気候特性**
- **亜熱帯気候**: 年間を通じて温暖。日照時間が長い
- 暖かい気候により全国で最も早い新茶の収穫（3月下旬〜4月）
- 年間複数回の収穫が可能
- 温暖なため有機栽培の病害虫管理が比較的容易

**土壌**
- **火山灰土壌（シラス台地）**: 桜島の火山灰に由来
- 排水性が極めて良好
- ミネラル（特に鉄分）が豊富
- 独特の風味特性を生み出す

**標高**
- 平地〜丘陵地（大規模平坦地が中心）
- 霧島高原地域は標高が高め

**主要品種**
- **ゆたかみどり**: 全国シェア2位の品種。温暖な気候に適する
- **さえみどり**: やぶきたと静岡在来16号の交配品種。美しい緑色、クリアな甘味
- **あさつゆ**: 早生品種
- **やぶきた**: 全国標準品種
- **おくみどり**: 遅摘み品種。2025年霜害を回避して注目度上昇

**風味プロファイル**
- 力強く、フルボディ
- 自然な甘味と豊かな旨味
- 苦味・渋味は控えめですっきりした後味
- 植物的（vegetal）なニュアンス

**歴史的意義**
- 1990年代から有機農法を積極導入 → 日本最大の有機JAS認証茶園の集積地
- **輸出用有機抹茶の80%**が鹿児島産と推定
- 2023年に碾茶生産量で京都を初めて抜いて全国1位に
- 2024年にはお茶全体の生産量でも静岡を初めて上回る（一番茶8,440トン vs 静岡8,120トン）
- 機械化された大規模生産が強み

**市場ポジション**
- **最もコストパフォーマンスが高い産地**: 品質に対して手頃
- 小売価格: 30gあたり$20-40（セレモニアルグレード相当）で宇治の半額以下
- 有機・オーガニックを求める海外市場での圧倒的強さ
- ラテ・カフェ用途、大量調達向け
- 若手農家がシングルカルティバー（単一品種）抹茶に挑戦する動きも

### 2-5. 狭山（埼玉県）

**基本情報**
- 地域: 埼玉県入間市、狭山市、所沢市 等（狭山丘陵周辺）
- 「色は静岡、香りは宇治、味は狭山でとどめさす」の名句で知られる三大銘茶の一つ

**気候特性**
- **寒冷な気候**: 茶の経済栽培の北限に近い
- 冬の寒さにより茶葉が厚くなる（芽重型）
- 年間を通じて2〜3回の収穫（温暖地の4〜5回に対して少ない）

**土壌**
- 関東ローム層（火山灰由来の赤土）
- 武蔵野台地の水はけの良い土壌

**標高**
- 丘陵地帯（50〜200m程度）

**主要品種**
- **やぶきた**: 主力品種
- **狭山かおり**: 地域独自品種。強い香気
- **ふくみどり**: 地域で栽培される品種

**風味プロファイル**
- **厚い茶葉に由来する濃厚な旨味**: 寒冷地ならではの深い味わい
- 「狭山火入れ」による独特のロースト香と甘味
- 力強いコクとボディ

**歴史的意義**
- 江戸時代から関東の銘茶として発展
- 「狭山火入れ（さやまびいれ）」: 独自の強火仕上げ技術が最大の特徴。保存性向上と焙煎香による風味増強
- 碾茶・抹茶の生産は歴史的には限定的だったが、近年急速に拡大中

**市場ポジション**
- 栽培面積783ha（全国8位）、荒茶生産量728トン（全国10位）※2021年
- 抹茶としてはニッチだが「狭山火入れ」による独特のロースト感は差別化要因
- 入間市が生産量の約60%を占める
- 東京に近い産地としての希少性

### 2-6. その他の注目産地

#### 静岡県
- **全国最大の茶生産県**（全体の30-35%）だが、碾茶シェアは12-13%
- 煎茶中心から碾茶への転換が急速に進行中
- 富士山の火山性土壌と多様な標高が独特のテロワールを形成
- **風味**: 明るく爽やかで、柑橘系のノートが特徴。西洋人好みのプロファイル
- **課題**: 山間部の急斜面は機械化困難。農家の平均年齢65歳以上

#### 三重県（伊勢茶）
- 栽培面積・生産量で全国3位の茶所
- **かぶせ茶**の生産量は全国1位
- 鈴鹿山脈の麓: 豊富な伏流水、扇状地の水はけの良い土壌 → 碾茶生産が拡大中
- やぶきたが主力品種
- 「やまた抹茶（Yamata Matcha）」として知られ、甘味と軽い渋味のバランスの取れた上品な味わい

#### 和束町（京都府）
- 宇治茶の中でも特に有名な産地。宇治茶の約40%を生産
- 急斜面の山間部: 手摘み・少量生産のアルチザナル抹茶
- テロワールの個性が極めて強い限定品

#### 熊本県
- 火山性土壌と革新的な有機栽培
- 確立された産地と比較して良好なコストパフォーマンス
- 注目度が上昇中

## 3. テロワールが抹茶の品質に与える影響

### 3-1. テロワールの構成要素

テロワール（terroir）とは、ワインと同様に、気候・土壌・地形・水・日照・人的技術の総合環境を指す。抹茶においては以下の5要素が品質を決定する。

| 要素 | 影響メカニズム | 品質への効果 |
|------|-------------|------------|
| **気候（気温）** | 寒冷地=成長が遅く茶葉が厚くなる / 温暖地=成長が早く年複数回収穫可能 | 寒冷地: 濃厚な旨味 / 温暖地: すっきりした味わい |
| **霧・湿度** | 天然の遮光効果 → カテキン（苦味）の生成を抑制、テアニン（旨味）を保持 | 霧が多い産地ほど自然に苦味が少なく旨味が強い |
| **土壌** | 窒素含有量 → 旨味、カリウム・リン → フローラルなアロマ、火山灰 → ミネラル感 | 土壌のミネラル組成が風味の「骨格」を決める |
| **標高** | 高標高 = 昼夜の寒暖差大 → アミノ酸蓄積 / 低標高 = 安定した生産量 | 高標高: 複雑で奥行きのある風味 / 低標高: 均質な品質 |
| **緯度** | 北方 = 日照時間短く成長遅い / 南方 = 長い日照、早い収穫 | 北方: 繊細で複雑 / 南方: 力強くボリューミー |

### 3-2. テロワールが影響する品質3要素

**色（Color）**
- 被覆栽培の期間・方法 + 土壌の窒素含有量が葉緑素量を決定
- 鮮やかな翡翠色ほど高品質とされる
- 西尾の砂壌土・高窒素環境 → 特に発色が良い
- さえみどり品種 × 鹿児島の火山灰土壌 → ネオングリーンの鮮烈な色

**味（Flavor）**
- L-テアニン（旨味・甘味）: 被覆期間、土壌窒素、品種で決定。八女・宇治が突出
- カテキン（苦味・渋味）: 日照量に比例。被覆栽培で抑制
- 同じ品種でも産地が異なれば味が異なる（マイクロクライメイトの影響）

**香り（Aroma）**
- **覆い香（おおいか）**: 被覆栽培により生じる海苔のような独特の香り。伝統的な本簀栽培で最も強く発現
- 品種固有の揮発性成分: さみどり=柑橘系、あさひ=強い覆い香、うじひかり=独特の揮発性アロマ
- 土壌のカリウム・リン含有量 → フローラルなアロマ（八女に顕著）

### 3-3. 一番茶 vs 二番茶以降

- **一番茶（4月下旬〜5月）**: 栄養素が最も濃縮され、甘味が最強。全産地共通で最高品質
- 鹿児島は温暖なため一番茶の収穫が最も早い（3月下旬〜）
- 宇治・八女は鹿児島より遅い
- **二番茶以降**: より力強く、洗練度は下がる。料理用グレードに適する

## 4. 主要品種と産地の関係

### 4-1. 碾茶（抹茶）用の主要品種

| 品種名 | 特徴 | 主な栽培産地 | 風味プロファイル |
|--------|------|------------|----------------|
| **やぶきた** | 全国茶畑の75%。中生。万能型 | 全産地 | バランスが良いが、抹茶用としては色がやや黄味。苦味が出やすい |
| **さみどり** | 宇治在来種。手摘み向き | 宇治、西尾(30%) | 鮮やかな緑色、繊細な旨味、柑橘系〜青野菜の香気 |
| **おくみどり** | 遅摘み。やぶきたの7-8日後 | 西尾(20%)、八女、鹿児島 | 深い緑色、ベルベットのような舌触り、苦味ゼロ。ラテに最適 |
| **あさひ** | 極早生。覆い香が卓越 | 宇治 | 強い香気、新鮮で爽やかな香味 |
| **うじひかり** | 宇治固有品種 | 宇治 | 独特の揮発性アロマ → 被覆で海苔様の覆い香に |
| **ごこう** | 旨味が強い | 宇治、八女 | 濃厚な旨味、深い味わい |

### 4-2. 品種選択の戦略（2026年の視点）

- **おくみどり**: 2025年4月霜害でやぶきたが壊滅的被害を受けた中、7-8日遅い萌芽時期のおくみどりは被害を回避。供給安定性の点で2026年の注目品種
- **さえみどり**: 視覚的インパクトは最大だが、早生ゆえの霜害リスク。鹿児島産に限定して調達推奨
- **やぶきた**: シェア70-75%と支配的だが、気候変動リスクへの脆弱性が顕在化

## 5. シングルオリジンの概念とマーケティング

### 5-1. 定義

| 用語 | 定義 |
|------|------|
| **シングルオリジン（Single Origin）** | 単一の産地・農園から調達された茶葉のみで作られた抹茶 |
| **シングルカルティバー（Single Cultivar）** | 単一品種の茶葉のみで作られた抹茶 |
| **ブレンド（Blended）** | 複数の産地・品種をブレンドして一定の風味プロファイルを実現した抹茶 |

### 5-2. シングルオリジン vs ブレンド

| 観点 | シングルオリジン | ブレンド |
|------|----------------|---------|
| **風味** | テロワールの個性が鮮明。ヴィンテージ感 | 安定した風味プロファイル |
| **ストーリー** | 農園・産地・品種の物語が語れる | 「技術・ブレンダーの腕」が物語 |
| **一貫性** | ロット毎にばらつきあり | 年間通じて均一 |
| **価格** | プレミアム（2-3倍のマージン可能） | スケーラブル |
| **ターゲット** | 愛好家・体験重視の消費者 | 日常使い・カフェ・菓子 |

### 5-3. マーケティングにおけるシングルオリジンの活用

**ワインのアナロジー**
- スペシャルティコーヒーの成功モデルを抹茶に適用
- 「テロワール × 品種 × 作り手」の三位一体でストーリーを構築
- 消費者に「違いを味わう」体験を提供

**D2Cブランドでの活用法**
- **季節限定のシングルオリジン**: 一番茶の時期に産地別の抹茶をリリース → 「体験 + コレクタビリティ」
- **サブスクリプション**: 毎月異なる産地の抹茶を届ける「産地めぐり」モデル
- **ライブ配信石臼挽き**: 産地の物語を語りながら挽きたてを月次配送

**価格帯の三層構造**（2025-2026年の市場）
- **ボリューム帯**: ¥3,000-5,000/kg（鹿児島・静岡の料理用グレード）
- **ミドルプレミアム**: ¥6,000-9,000/kg（西尾・鹿児島のラテグレード）
- **セレモニアル/シングルオリジン**: ¥10,000+/kg（宇治・八女の高級品）→ 利益率2-3倍

### 6-3. 認証制度の意義

- **宇治抹茶の定義の広さ**: 4府県の茶葉を京都で加工すれば「宇治抹茶」を名乗れるため、実際には奈良産の原料が京都で加工されるケースも多い。これは品質保証の観点で議論がある
- **西尾の厳格さ**: 栽培から石臼挽きまで全工程が西尾市周辺で完結する必要があり、トレーサビリティが明確
- **高額取引における認証の重要性**: 同位体分析（isotope analysis）による産地偽装の検証も行われる
