## 1. 栽培 (Cultivation)

### 1-1. 被覆・遮光 (Shading)

抹茶の原料である碾茶は、収穫前に茶園を遮光して栽培する「被覆栽培」が必須。遮光により光合成が抑制され、アミノ酸（特にL-テアニン）がカテキン（苦渋味成分）に変換されるのを防ぐ。

#### 遮光方法の比較

| 項目 | 棚掛け・本簀 (Tana-gake / Honzu) | 棚掛け・寒冷紗 (Tana-gake / Kanreisha) | 直掛け (Jika-gake) |
|------|------|------|------|
| **構造** | 茶園上部に木製の棚（高さ約180-200cm）を設置し、その上に葦簀（よしず）と藁を敷く | 茶園上部に木製or金属製の支柱で棚を設置し、黒い化繊の寒冷紗を掛ける | 茶樹に直接、黒い化繊シートを被せる |
| **素材** | 天然素材（葦簀＝葦・蒲を麻紐で編んだもの、藁） | 化学繊維（黒い寒冷紗） | 化学繊維（黒い穴あきプラスチックシート） |
| **遮光の特徴** | 自然光のように不規則に光を透過させる（木漏れ日に類似）。最高品質 | 均一に遮光。品質は本簀に次ぐ | 均一に遮光。コスト最安だが品質は劣る |
| **通気性** | 優れる | 良好（棚構造のため茶樹と遮光材の間に空間あり） | 劣る（直接被せるため熱がこもりやすく、風で葉が擦れて傷つくリスク） |
| **コスト** | 極めて高い（天然素材は再利用不可、毎年調達が必要） | 中程度（化繊は複数シーズン再利用可能） | 最も安い |
| **希少性** | 非常に稀。宇治の最高級品に限定 | 現在の主流方式 | 大量生産向け |

#### 遮光の段階的プロセス（本簀の場合）

1. **第1段階（約7-10日間）**: 葦簀のみを棚上に敷く → 日光の55-60%を遮断
2. **第2段階（約10日間）**: 葦簀の上に藁（600-700kg）を追加 → 日光の95-98%を遮断
3. **合計被覆期間**: 約20日間（高級品）

寒冷紗・直掛けの場合も同様に段階的に遮光率を上げる:
- 最初の1週間: 70-80%遮光（1枚目のシート）
- その後: 98%遮光（2枚目のシートを重ねる）

#### 被覆期間とグレードの関係

| グレード | 被覆期間 | 遮光率 |
|---------|---------|--------|
| 濃茶用（最高級） | 30-40日以上 | 95-98% |
| 薄茶用（高級） | 20-30日 | 95-98% |
| プレミアム | 14-20日 | 70-95% |
| 料理用（culinary） | 7-14日 | 70-80% |
| かぶせ茶（参考） | 7-10日 | 軽度の遮光 |

#### 遮光による生化学的変化

| 成分 | 遮光の効果 | メカニズム |
|-----|-----------|-----------|
| **L-テアニン（アミノ酸）** | 大幅に増加（甘味・旨味UP） | 光合成が抑制されることで、テアニン→カテキンへの変換が抑えられる。冬の休眠期間中に根に蓄積されたテアニンが春の新芽に集中 |
| **クロロフィル（葉緑素）** | 増加（鮮やかな緑色に） | 少ない光を最大限に活用するため、光合成色素の生産を増加。遮光85%が最も含有量が高いという研究あり |
| **カテキン（ポリフェノール）** | 減少（苦渋味DOWN） | カテキンは紫外線防御物質。遮光により生合成が抑制される |
| **カフェイン** | 増加 | 遮光により含有量が上昇 |
| **カロテノイド** | 増加 | クロロフィルと同様に光捕集分子として増産 |
| **覆い香（おおいか）** | 発生 | 遮光栽培特有の甘い香り（ジメチルスルフィド等）。海苔のような独特の芳香 |

### 1-2. 収穫 (Harvesting)

#### 一番茶 vs 二番茶

| 項目 | 一番茶 (Ichibancha / First Flush) | 二番茶 (Nibancha / Second Flush) |
|------|------|------|
| **収穫時期** | 4月下旬〜5月（八十八夜前後） | 6月〜7月（一番茶の45-50日後） |
| **品質** | 最高品質。冬の休眠期に蓄えた栄養が凝縮 | 一番茶に比べ品質は劣る |
| **L-テアニン含有量** | 最大（二番茶の約3倍） | 一番茶より30-40%低下 |
| **カテキン含有量** | 低い（苦味が少ない） | 高い（夏の日照でカテキン生合成が活発化。苦渋味が強い） |
| **クロロフィル** | 高い（鮮やかな緑色） | 低い（くすんだ緑〜黄緑色） |
| **葉の状態** | 柔らかく薄い | 細胞壁が厚く硬い |
| **用途** | 濃茶・薄茶（ceremonial / premium） | 料理用（culinary）、フードサービス |
| **価格** | 高い | 安い |

> 品質の最大の予測因子は収穫時期（flush）である。一番茶が圧倒的に高品質。

#### 手摘み vs 機械摘み

| 項目 | 手摘み (Tezumi) | 機械摘み (Kikai-zumi) |
|------|------|------|
| **品質** | 最高品質。柔らかい新芽のみを選別して摘採可能 | やや劣る。古い葉や傷んだ葉が混入するリスク |
| **効率** | 低い（機械摘みの約1/100の速度） | 高い |
| **味の特徴** | 滑らかで甘く、深い旨味。苦味が極めて少ない | より鋭く、円やかさに欠け、苦味が出やすい |
| **被覆期間** | 一般に長い被覆期間（ポテンシャル最大化） | 一般に短い被覆期間（2番茶・3番茶も収穫するため樹に体力を残す必要がある） |
| **現状** | 宇治の最高級品に限定。極めて稀 | 日本全体の収穫の大部分を占める。はさみ刈り（バリカン型）が主流 |
| **価格** | プレミアム価格 | 標準的 |

## 2. 碾茶の製造工程 (Tencha Processing)

収穫から碾茶完成までの工程:

```
摘採 → 蒸し → 冷却・散茶 → 乾燥（碾茶炉） → 荒碾茶
→ 熟成（低温保存） → 仕上げ（選別・切断） → 仕上げ碾茶
```

### 2-1. 蒸し (Steaming)

摘採した生葉は、酸化酵素を失活させるために**収穫当日中に**蒸される。これにより酸化発酵が止まり、緑色・風味・栄養素が保持される。

#### 蒸し条件
- **温度**: 約100C（蒸気）
- **時間**: 10-20秒（碾茶は煎茶より短く蒸す傾向）
- **方式**: 蒸気による加熱

#### 浅蒸し vs 深蒸し

| 項目 | 浅蒸し (Asamushi / Shallow Steaming) | 深蒸し (Fukamushi / Deep Steaming) |
|------|------|------|
| **蒸し時間** | 約10-20秒（碾茶の標準） | 約30秒〜1分以上 |
| **色** | 明るい黄緑色 | 深い濃緑色 |
| **味** | カテキンが多く残り、やや渋味あり。伝統的な風味 | テアニンが増加し、より柔らかく甘い味わい |
| **香り** | 覆い香（おおいか）が際立つ | 覆い香はやや弱まる |
| **碾茶での使用** | 伝統的な碾茶製造の主流 | 一部の近代的な製法で採用。より深い色の抹茶を求める場合 |

> 碾茶は煎茶と異なり、蒸した後に**揉まない**（不揉）。これが碾茶と煎茶の最大の製法上の違い。

### 2-2. 冷却・散茶

蒸し直後に、高さ5-6mまで風で吹き上げる「散茶機」を使用して急速冷却。この工程で:
- 蒸しで付着した水分を除去
- 蒸しでくっついた葉を分離
- 葉温を速やかに下げて品質劣化を防止

### 2-3. 乾燥 (Drying) — 碾茶炉

碾茶の乾燥は、煎茶のような揉捻工程を経ずに行われる（**不揉乾燥**）。

#### 碾茶炉の構造
- **全長**: 10メートル以上
- **構造**: 3段式コンベヤーベルト（レンガ造りの碾茶炉）
- **工程**:
  1. 下段: 高温で粗乾燥
  2. 風で上段へ送られる
  3. 上段→中段: 仕上げ乾燥（水分をゆっくり蒸発させ、風味を凝縮）

この工程で得られるのが**荒碾茶（あらてんちゃ）**。葉は平たい形状のまま乾燥される（煎茶のような針状ではない）。

### 2-4. 熟成 (Aging / Maturation)

碾茶製造における重要だが見過ごされがちな工程。

#### 熟成条件
- **温度**: 5C以下（冷蔵庫・低温倉庫）
- **環境**: 暗所、密封容器、湿度管理
- **期間**: 通常3〜6ヶ月（高級品は6ヶ月以上、超プレミアムでは最大36ヶ月の事例も）

#### 熟成による変化

| 熟成前 | 熟成後 |
|--------|--------|
| 新鮮だが青臭く鋭い風味 | 角が取れ、まろやかに |
| 渋味・苦味が目立つ | 渋味が軟化 |
| 香りが単調 | 香りが複雑で丸みを帯びる |
| 旨味が不十分 | 旨味が深まり、甘みが前面に |
| — | クリーミーで新鮮なミルクのような口当たり |

> 海外では「新鮮な抹茶＝高品質」という誤解があるが、実際には適切な熟成を経ていない抹茶は「平坦で苦い」味になる。日本の伝統的な抹茶製造では熟成は不可欠な工程。

#### 歴史的背景
- 伝統的に碾茶は茶壺に詰めて保管された（口切りの茶事：11月に茶壺を開封する儀式がある）
- 5月収穫 → 11月口切り = 約6ヶ月の熟成期間

### 2-5. 仕上げ (Refining / Finishing)

荒碾茶から製品としての仕上げ碾茶を作る工程:

1. **選別**: 葉の大きさでふるい分け
2. **茎・葉脈の除去（脱梗・脱脈）**: 茎（くき）と葉脈（すじ）を取り除く。これが不十分だと:
   - 最終粉末に白い繊維質が混入
   - 石臼が詰まる原因になる
   - 苦味が増加
3. **切断**: 石臼で挽きやすいサイズ（約5mm）にカット
4. **二次乾燥**: 残存水分を最終調整
5. **審査・合組（ブレンド）**: 専門の審査員が碾茶を評価し、グレーディング。複数のロットをブレンドして目標の味・色・香りを実現

## 3. 粉砕・製粉 (Grinding / Milling)

仕上げ碾茶を微粉末にする工程。製粉方法は抹茶の品質を大きく左右する。

### 3-1. 石臼挽き (Ishi-usu / Stone Mill)

伝統的な製法。12世紀から使用されている。

#### 仕様・特性

| 項目 | 詳細 |
|------|------|
| **構造** | 上臼と下臼の2枚の御影石（花崗岩）が合わさる。上臼が回転し、碾茶が溝を通って粉砕される |
| **回転速度** | 約30 RPM（1回転あたり約3秒） |
| **生産能力** | 手挽き: 約30g/時間、機械化された石臼: 約40-50g/時間 |
| **粒径** | 5-20μm（最高級品は1-10μm）、中央値は約15-16μm |
| **粒子形状** | 不規則な形状（角張っている） |
| **温度** | 低速回転により摩擦熱がほとんど発生しない → 香り・色・栄養素の劣化を防止 |
| **品質** | 最高品質。風味・香り・食感（口溶け感）・泡立ちが最も優れる |
| **コスト** | 高い（石臼自体の価格＋低い生産性） |

#### 石臼の溝（目）
- 石臼の表面には精密な溝（ふくみ）が彫られている
- 溝のパターンが碾茶を中心から外周へ送り出しながら粉砕する
- 溝は使用により摩耗するため、定期的な「目立て」（溝の彫り直し）が必要
- 専門の石臼職人が手作業で目立てを行う

#### 石臼が品質に優れる理由
- 低速・低温で挽くため、揮発性の香気成分が飛散しにくい
- ゆっくりとした圧力で粉砕するため、細胞壁が穏やかに破壊され、滑らかな口溶け感を生む
- 不規則な粒子形状が茶筅で点てた際に空気を含みやすく、きめ細かい泡が立つ

### 3-2. 工業用石臼 (Industrial Stone Mill)

基本原理は伝統的な石臼と同じだが、モーターで駆動される量産仕様。

- 回転速度・温度管理は手動石臼に準じる
- 24時間連続稼働が可能
- 高級抹茶の大量生産に使用
- 品質は手動石臼に近い（ただし職人の微調整には及ばない場合も）

### 3-3. ボールミル (Ball Mill)

#### 仕組み
回転する円筒容器の中にセラミック製のボールを充填。容器が回転することでボール同士の摩擦・衝撃で碾茶を粉砕する。

#### 仕様・特性

| 項目 | 詳細 |
|------|------|
| **粒径** | 15-30μm（中央値約20μm）。石臼よりやや粗い |
| **生産能力** | 石臼の数十〜数百倍。大量生産向き |
| **温度** | 摩擦熱が発生しやすい → 香り・色の劣化リスク |
| **品質への影響** | クロロフィルの減少（色がくすむ）、苦味の増加、ザラザラした食感 |
| **コスト** | 低い |
| **用途** | 料理用（culinary）グレード、食品原料用の抹茶 |

### 3-4. ビーズミル (Bead Mill / Stirred Media Mill)

#### 仕組み
円筒形チャンバーの中にジルコニア製またはステンレス製の小さなビーズ（0.8-2.0mm）を充填。高速アジテーター（攪拌機）がビーズを動かし、衝突・圧縮・せん断力で碾茶を粉砕する。

#### 仕様・特性

| 項目 | 詳細 |
|------|------|
| **粒径** | 5-20μm（中央値約19-20μm） |
| **生産能力** | 石臼より大幅に高い。ホールセール・グローバル需要に対応可能 |
| **温度管理** | 冷却水循環システムにより低温粉砕が可能（原料5Cに予冷 → チャンバー10C維持 → 粉砕後即座に-20C保存） |
| **品質** | 条件次第で石臼に匹敵する品質が可能。ただし官能評価では「生臭い（fishy）」ノートが出るとの研究報告あり |
| **コスト** | 中〜高（設備投資は大きいが、生産性で補う） |
| **用途** | 中〜高品質の抹茶の量産。近年最も注目されている製粉技術 |

> ビーズミルはボールミルに比べ、同じ粉砕レベルで省エネルギーであり、特に18μm以下の微粉砕において優位性がある。

### 3-5. ジェットミル (Jet Mill / Air Jet Mill)

#### 仕組み
圧縮空気または不活性ガスの高速気流（ジェット）でパーティクル同士を衝突させて粉砕する。粉砕メディア（ボールやビーズ）を使用しない。

#### 仕様・特性

| 項目 | 詳細 |
|------|------|
| **粒径** | 5μm以下が可能（D50: 5-10μm）。全方式中最も微細。粒度分布が狭い |
| **粒子形状** | 球形に近い（真円度が高い）。石臼の不規則形状とは対照的 |
| **温度** | 粉砕熱がほとんど発生しない（気流のみで粉砕するため）。熱に敏感な素材に最適 |
| **汚染リスク** | 極めて低い（接触する粉砕メディアがないため） |
| **エネルギー消費** | 高い |
| **品質への影響** | 官能評価で「葉っぽい（leafy）」ノート。超微粉末のため口溶けは良いが、球形粒子のため泡立ちは石臼に劣る可能性。粒子が細かすぎて風味・香りの複雑さが失われるとの指摘あり |
| **用途** | 食品原料用、サプリメント、一部の高品質ブレンド |

### 3-6. 粉砕方法の総合比較

| 項目 | 石臼 | ボールミル | ビーズミル | ジェットミル |
|------|------|-----------|-----------|------------|
| **粒径（中央値）** | 15-16μm | 15-20μm | 19-20μm | 5μm以下 |
| **粒子形状** | 不規則 | やや不規則 | やや不規則 | 球形に近い |
| **生産能力** | 30-50g/時間/台 | kg単位/時間 | kg単位/時間 | kg単位/時間 |
| **発熱** | 極めて少ない | 多い | 少ない（冷却可） | 極めて少ない |
| **風味** | ロースティ（roasty） | 苦味増加 | 生臭い（fishy） | 葉っぽい（leafy） |
| **色の保持** | 最良 | 劣化あり | 良好 | 良好 |
| **泡立ち** | 最良（不規則粒子が空気を含む） | 劣る | 良好 | やや劣る（球形のため） |
| **コスト** | 最高 | 最安 | 中 | 中〜高 |
| **主な用途** | ceremonial | culinary | premium〜ceremonial | 食品原料・サプリ |

## 4. 品質グレーディング (Quality Grading)

### 重要な前提: 公的な等級基準は存在しない

「ceremonial grade」「culinary grade」という分類は**業界の慣習・マーケティング用語**であり、政府機関や国際機関による公式な規格・認証は存在しない。各社が独自の基準で等級を付けているため、A社のceremonial gradeがB社のpremium gradeと同等、ということが起こり得る。

日本国内では、碾茶は専門の審査員が**外観・色・香り・味**を官能評価し、点数をつけてグレーディングする。この評価は茶業者間の取引基準として機能する。

### 4-1. グレード比較

| 項目 | Ceremonial Grade（濃茶・薄茶用） | Premium Grade | Culinary Grade（料理・ラテ用） |
|------|------|------|------|
| **収穫** | 一番茶のみ（4-5月） | 一番茶〜早い二番茶 | 二番茶〜三番茶 |
| **被覆期間** | 20-40日 | 14-20日 | 7-14日 |
| **製粉方法** | 石臼挽き（30 RPM） | 石臼 or 高品質ビーズミル | ボールミル等の工業用ミル |
| **粒径** | 5-15μm | 10-20μm | 20-30μm |
| **色** | 鮮やかなエメラルドグリーン | 明るい緑色 | オリーブ〜カーキ色 |
| **味** | 甘い旨味、滑らか、苦味ほぼなし | 軽い渋味あり | はっきりした苦味・渋味 |
| **飲み方** | ストレート（お湯のみ）で飲める | ストレート or ラテ | ラテ、菓子、料理の原材料 |

### 4-2. 成分含有量の目安（1gあたり）

| 成分 | Ceremonial | Premium | Culinary |
|------|-----------|---------|----------|
| **L-テアニン** | 20-40 mg | 14-25 mg | 6-14 mg |
| **クロロフィル** | 10-18 mg | 7-12 mg | 3-7 mg |
| **総カテキン** | 100-130 mg | 110-140 mg | 120-150 mg |

> カテキンが多い = 苦い。Culinary gradeのほうがカテキン含有量は高い（遮光が短い→カテキン生合成が活発）。

### 4-3. グレードを決定する主要因子（重要度順）

1. **収穫時期（flush）**: 最大の品質決定因子。一番茶 >> 二番茶 >> 三番茶
2. **被覆期間**: 長いほどL-テアニン↑、クロロフィル↑、カテキン↓
3. **被覆方法**: 本簀 > 寒冷紗棚掛け > 直掛け
4. **手摘み vs 機械摘み**: 手摘みが最高品質だが極めて稀
5. **碾茶の仕上げ精度**: 茎・葉脈の除去が不十分だと品質低下
6. **熟成期間**: 適切な熟成がまろやかさと旨味の深さを生む
7. **製粉方法**: 石臼挽きが最高品質。粒径・形状・温度が味・色・泡立ちに影響
8. **茶品種（cultivar）**: さみどり、あさひ、おくみどり等の品種選択も品質に影響
