# 缶＋茶杓一体型 要件定義書

---

## 1. プロダクト概要

抹茶パウダーと茶杓を1つの缶に内蔵し、「缶を開けたらすぐ飲める」体験を実現する。
ユーザーが用意するのはマグカップとお湯だけ。

---

## 2. 茶杓の要件

### 2-1. 計量

| 項目 | 仕様 |
|------|------|
| 目標計量 | 1すくい = 2g（薄茶1杯分） |
| すくい部の容量 | 約2.5〜3cc（抹茶の嵩密度 0.6〜0.8g/cc から算出） |
| 許容誤差 | ±0.3g（1.7〜2.3g） |

> **根拠**: 抹茶パウダーの嵩密度は約0.6〜0.8g/cc。2gを実現するには 2g ÷ 0.7g/cc ≒ 2.9cc の容量が必要。すりきり1杯で2gになるよう、すくい部の深さ・幅を調整する。

### 2-2. 形状・寸法

| 項目 | 仕様 |
|------|------|
| 全長 | 缶の内径以内（10〜11cm） |
| すくい部 | 幅 約2.5cm × 長さ 約3cm × 深さ 約5mm |
| 柄 | 持ちやすさを考慮した細身設計 |
| すりきり設計 | すくって軽くならすだけで2gになる形状（山盛りにしなくてよい） |

### 2-3. 素材

| 候補 | メリット | デメリット |
|------|---------|-----------|
| ステンレス（SUS304） | 錆びない、洗いやすい、食品安全、安価 | 質感がやや工業的 |
| 真鍮＋食品安全コーティング | 高級感、エイジングで愛着 | コーティング耐久性の検証要 |
| 木製（黒檀等の硬木） | 温かみ、伝統との接続 | 湿気・カビのリスク、密封缶内での耐久性要検証 |

> **第一候補: ステンレス**。コスト・衛生・耐久性のバランスが最も良い。ブランドの成熟に応じて真鍮や限定素材を展開する余地を残す。

### 2-4. 固定方式（外蓋裏マウント）

- **外蓋の裏面**にマグネットで茶杓を横向きに固定する
- 外蓋を開けると茶杓が一緒に持ち上がる。茶杓は外蓋と内蓋の間の空間に収まる
- 茶杓が粉に一切触れない衛生的な構造（内蓋が粉を密封）
- 片手で外せる程度の保持力（落ちない＋外しやすいの両立）
- 平型缶のため茶杓は横向きに寝かせた状態で固定（縦型より自然な配置）

| 固定方式 | メリット | デメリット |
|---------|---------|-----------|
| マグネット（シリコン被覆ネオジム） | 着脱が最もスムーズ。平型缶の広い蓋面積で配置自由度◎ | 食品接触安全基準への適合要確認 |
| シリコンクリップ | 安価、食品安全 | 繰り返し使用での劣化 |
| はめ込みレール | 確実な固定 | 着脱にやや手間 |

> **第一候補: マグネット方式**。「パチッと付く・サッと外れる」が日常の所作として最も気持ちいい。平型缶の広い蓋裏面はマグネット配置に余裕があり設計しやすい。

## 3. 缶本体の要件

### 3-1. サイズ

| 項目 | 仕様 |
|------|------|
| 直径 | 11〜12cm（平型。茶杓が横に収まるサイズ） |
| 高さ | 4〜5cm（浅型。すくいやすさ重視） |
| 容量 | 抹茶 30g（約15杯分）を収納 + 茶杓の空間 |

> **形状変更の理由**: 縦長→平型に変更。(1) 茶杓が蓋裏に横向きで自然に収まる (2) 浅いので粉をすくいやすい (3) 蓋面積が広くブランドロゴの表現力が上がる (4) 化粧品コンパクトのような上品さで市場に無い差別化形状

### 3-2. 構造（二重蓋）

```
[外蓋] デザイン面 — 被せ式。銘柄名・ブランドデザイン。裏面にマグネットで茶杓を横向き固定
  ↓
  （外蓋と内蓋の間の空間に茶杓が収まる）
  ↓
[内蓋] 密封蓋 — パッキン付き。気密性を担保
  ↓
[本体] 浅型。抹茶パウダー
```

- **外蓋**: ブランド体験の顔（広い面積でロゴ表現力◎）。裏面にシリコン被覆ネオジムマグネットで茶杓を横向きに固定。外蓋を開けると茶杓が一緒に持ち上がる
- **内蓋**: 気密性を担保。パッキン（シリコン）で酸化・湿気を防止。茶杓が粉に触れない構造
- **本体**: 食品グレードのブリキ or アルミ

> **茶杓の位置**: 外蓋の裏側（外蓋と内蓋の間）。茶杓が粉に一切触れず衛生的。内蓋が独立して密封するため、外蓋の開閉で気密性に影響しない

### 3-3. 気密性

| 項目 | 仕様 |
|------|------|
| パッキン | 食品グレードシリコン |
| 遮光 | 金属缶のため完全遮光 |
| 目標 | 開封後30日間、品質劣化を最小限に抑える |

### 3-4. デザイン要件（DECIDED.mdより）

- インテリアとして「ずっと置いておきたい」レベルの美しさ
- 銘柄名がデザインの一部として統合（後付けラベルではない）
- 銘柄ごとにバリエーション → コレクション性
- ギフトとしてそのまま渡せる仕上がり

### 3-5. トーン＆マナー（カラー）

**方向性**: Burberryのような「大胆だけど洗練」。深い緑を主役に、ピンクと木の色で柔らかさと温かみを加える。

| 役割 | カラー | HEX | イメージ |
|------|--------|-----|---------|
| **メイン** | 抹茶グリーン（深めの緑） | `#2D6A4F` | ブランドの顔。Burberry店舗のような存在感のある緑。缶本体・外蓋の主色 |
| **差し色** | 桜ピンク（薄いピンク） | `#F2C4C4` | 桜・桜の精を連想させる淡いピンク。ロゴ、銘柄名、ライン等のアクセントに使用 |
| **サブ** | バーチウッド（薄い木の色） | `#D4B896` | 温かみ・ナチュラルさ。茶杓の素材感や、パッケージのテクスチャ要素として |

**配色バランス**: 緑 70% / ピンク 15% / ウッド 15% 程度

**トーンの指針**:
- 緑は「和」に寄りすぎない。モダンで都会的な深緑（Burberry的なリッチグリーン）
- ピンクは甘すぎない。桜の花びらのような儚さ・上品さ
- ウッドは主張しすぎない。緑とピンクの間を繋ぐ「温度」の役割
- 3色の組み合わせで「日本の自然（抹茶・桜・木）」を抽象的に表現する

### 3-6. 3Dビューワー

缶の3Dプロトタイプを確認できるビューワーを用意している。

- ファイル: [can-viewer/index.html](can-viewer/index.html)

## 4. ユーザー体験（UXフロー）

### 毎日の使用フロー（目標: 15秒）

```
1. 外蓋を外す（茶杓が外蓋にくっついて一緒に上がる）
2. 茶杓を外蓋から外す
3. 内蓋を開ける
4. 抹茶をすくう（1すくい = 2g = 1杯分）← 平型なのですくいやすい
5. マグカップに入れる
6. 内蓋を閉める（粉を密封）
7. 茶杓を外蓋の裏に戻す（マグネットでパチッ）
8. 外蓋を被せる
9. お湯を注いでくるくる回す（or 茶筅で点てる）
```

> **平型缶のメリット**: 浅い缶なので粉にアクセスしやすく、すくう動作が楽。縦型では奥の粉に手が届きにくかった問題を解消。

### 重要なUX原則

- **「1すくい = 1杯」を迷わず実現できること** — 計量スプーン不要
- 茶杓が粉に触れないこと（外蓋裏マウント＋内蓋で物理的に隔離）
- 内蓋を閉めれば気密性が保たれる（外蓋の開閉頻度に影響されない）
- リフィル時に茶杓が邪魔にならないこと（外蓋側に付いているので問題なし）
- 洗浄は基本不要（乾燥した粉をすくうだけ）。気になる場合は水洗い

## 6. 検証が必要な項目

| 項目 | 内容 | 優先度 |
|------|------|--------|
| 2g計量の精度 | 試作品で実測。すりきり時のバラつきを確認 | 高 |
| マグネットの食品安全性 | シリコン被覆で食品接触基準（食品衛生法）をクリアできるか | 高 |
| 気密性テスト | 二重蓋＋パッキンで30日間の品質維持を検証 | 高 |
| 茶杓固定の耐久性 | 毎日の着脱（年365回）に耐えるか | 中 |
| 缶内での粉の振る舞い | 平型缶で振動・傾き時の粉の偏りを確認 | 中 |
| 外蓋と内蓋の間の高さ | 茶杓の厚み（すくい部5mm＋マグネット）が収まるか確認 | 高 |
| 平型缶の安定性 | 低重心で転倒しにくいはず。棚やカウンターでの安定性検証 | 低 |
| リフィルの注ぎやすさ | パウチから缶への移し替えがスムーズか | 中 |
