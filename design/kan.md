# 缶＋茶杓一体型 要件定義書

---

## 1. プロダクト概要

抹茶パウダーと茶杓を1つの缶に内蔵し、「缶を開けたらすぐ飲める」体験を実現する。
ユーザーが用意するのはマグカップとお湯だけ。

---

## 2. 茶杓の要件

### 2-1. 計量

| 項目 | 仕様 |
|------|------|
| 目標計量 | 1すくい = 2g（薄茶1杯分） |
| すくい部の容量 | 約2.5〜3cc（抹茶の嵩密度 0.6〜0.8g/cc から算出） |
| 許容誤差 | ±0.3g（1.7〜2.3g） |

> **根拠**: 抹茶パウダーの嵩密度は約0.6〜0.8g/cc。2gを実現するには 2g ÷ 0.7g/cc ≒ 2.9cc の容量が必要。すりきり1杯で2gになるよう、すくい部の深さ・幅を調整する。

### 2-2. 形状・寸法

| 項目 | 仕様 |
|------|------|
| 全長 | 缶の内径以内（8〜9cm） |
| すくい部 | 幅 約2.5cm × 長さ 約3cm × 深さ 約5mm |
| 柄 | 持ちやすさを考慮した細身設計 |
| すりきり設計 | すくって軽くならすだけで2gになる形状（山盛りにしなくてよい） |

### 2-3. 素材

| 候補 | メリット | デメリット |
|------|---------|-----------|
| ステンレス（SUS304） | 錆びない、洗いやすい、食品安全、安価 | 質感がやや工業的 |
| 真鍮＋食品安全コーティング | 高級感、エイジングで愛着 | コーティング耐久性の検証要 |
| 木製（黒檀等の硬木） | 温かみ、伝統との接続 | 湿気・カビのリスク、密封缶内での耐久性要検証 |

> **第一候補: ステンレス**。コスト・衛生・耐久性のバランスが最も良い。ブランドの成熟に応じて真鍮や限定素材を展開する余地を残す。

### 2-4. 固定方式（蓋裏マウント）

- 蓋（内蓋）の裏面にホルダーを設け、茶杓を固定する
- 蓋を開けると茶杓が一緒に持ち上がり、粉に埋まらない
- 片手で外せる程度の保持力（落ちない＋外しやすいの両立）

| 固定方式 | メリット | デメリット |
|---------|---------|-----------|
| マグネット（シリコン被覆ネオジム） | 着脱が最もスムーズ | 食品接触安全基準への適合要確認 |
| シリコンクリップ | 安価、食品安全 | 繰り返し使用での劣化 |
| はめ込みレール | 確実な固定 | 着脱にやや手間 |

> **第一候補: マグネット方式**。「パチッと付く・サッと外れる」が日常の所作として最も気持ちいい。

## 3. 缶本体の要件

### 3-1. サイズ

| 項目 | 仕様 |
|------|------|
| 直径 | 8〜9cm（片手で持てるサイズ） |
| 高さ | 8〜10cm |
| 容量 | 抹茶 30g（約15杯分）を収納 + 茶杓の空間 |

### 3-2. 構造（二重蓋）

```
[外蓋] デザイン面 — 被せ式。銘柄名・ブランドデザイン
  ↓
[内蓋] 密封蓋 — パッキン付き。裏面に茶杓ホルダー
  ↓
[本体] 抹茶パウダー
```

- **外蓋**: ブランド体験の顔。インテリアに置いたとき見える面
- **内蓋**: 気密性を担保。パッキン（シリコン）で酸化・湿気を防止。茶杓は内蓋裏に固定
- **本体**: 食品グレードのブリキ or アルミ

### 3-3. 気密性

| 項目 | 仕様 |
|------|------|
| パッキン | 食品グレードシリコン |
| 遮光 | 金属缶のため完全遮光 |
| 目標 | 開封後30日間、品質劣化を最小限に抑える |

### 3-4. デザイン要件（DECIDED.mdより）

- インテリアとして「ずっと置いておきたい」レベルの美しさ
- 銘柄名がデザインの一部として統合（後付けラベルではない）
- 銘柄ごとにバリエーション → コレクション性
- ギフトとしてそのまま渡せる仕上がり

### 3-5. トーン＆マナー（カラー）

**方向性**: Burberryのような「大胆だけど洗練」。深い緑を主役に、ピンクと木の色で柔らかさと温かみを加える。

| 役割 | カラー | HEX | イメージ |
|------|--------|-----|---------|
| **メイン** | 抹茶グリーン（深めの緑） | `#2D6A4F` | ブランドの顔。Burberry店舗のような存在感のある緑。缶本体・外蓋の主色 |
| **差し色** | 桜ピンク（薄いピンク） | `#F2C4C4` | 桜・桜の精を連想させる淡いピンク。ロゴ、銘柄名、ライン等のアクセントに使用 |
| **サブ** | バーチウッド（薄い木の色） | `#D4B896` | 温かみ・ナチュラルさ。茶杓の素材感や、パッケージのテクスチャ要素として |

**配色バランス**: 緑 70% / ピンク 15% / ウッド 15% 程度

**トーンの指針**:
- 緑は「和」に寄りすぎない。モダンで都会的な深緑（Burberry的なリッチグリーン）
- ピンクは甘すぎない。桜の花びらのような儚さ・上品さ
- ウッドは主張しすぎない。緑とピンクの間を繋ぐ「温度」の役割
- 3色の組み合わせで「日本の自然（抹茶・桜・木）」を抽象的に表現する

### 3-6. 3Dビューワー

缶の3Dプロトタイプを確認できるビューワーを用意している。

- ファイル: [can-viewer/index.html](can-viewer/index.html)

## 4. ユーザー体験（UXフロー）

### 毎日の使用フロー（目標: 10秒）

```
1. 外蓋を外す
2. 内蓋を開ける（茶杓が蓋と一緒に上がる）
3. 茶杓を外す
4. 抹茶をすくう（1すくい = 2g = 1杯分）
5. マグカップに入れる
6. 茶杓を内蓋に戻す
7. 内蓋を閉める → 外蓋を被せる
8. お湯を注いでくるくる回す（or 茶筅で点てる）
```

### 重要なUX原則

- **「1すくい = 1杯」を迷わず実現できること** — 計量スプーン不要
- 茶杓が粉に埋まらないこと（蓋裏マウントで解決）
- リフィル時に茶杓が邪魔にならないこと（蓋側に付いているので問題なし）
- 洗浄は基本不要（乾燥した粉をすくうだけ）。気になる場合は水洗い

## 6. 検証が必要な項目

| 項目 | 内容 | 優先度 |
|------|------|--------|
| 2g計量の精度 | 試作品で実測。すりきり時のバラつきを確認 | 高 |
| マグネットの食品安全性 | シリコン被覆で食品接触基準（食品衛生法）をクリアできるか | 高 |
| 気密性テスト | 二重蓋＋パッキンで30日間の品質維持を検証 | 高 |
| 茶杓固定の耐久性 | 毎日の着脱（年365回）に耐えるか | 中 |
| 缶内での粉の振る舞い | 振動・傾きで粉がホルダー部に詰まらないか | 中 |
| リフィルの注ぎやすさ | パウチから缶への移し替えがスムーズか | 中 |
