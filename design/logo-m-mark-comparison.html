Changes Applied

  • Letter spacing: 0.35em → 0.20em (tighter, more cohesive)

  • Matcha green: #3A6B4E → #5B9A6A (brighter, truer to real matcha)
  • Eye position: h*0.05 → h*0.13 (lower, reads as a face more naturally)
  • Eye/smile size: arc w*0.07 → w*0.09, stroke w*0.025 → w*0.03
  • "&" size: 100% → 75% scaled down as connector
  • Black version pool: now uses green pool (was invisible same-color black)
  • 2nd M (me): pool + eyes both present (unified concept)
  • Small marks: stroke width w*0.07 → w*0.10 (better at small sizes)
