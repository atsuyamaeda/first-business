## 0. 前提

### 目的
- プロダクト完成前に、ターゲット層の反応を定量的に測る
- ランディングページで「メール登録（ウェイトリスト）」のCVRを計測する
- チラシからランディングページへの導線を作り、オフラインでもリーチする

### 成功指標
| 指標 | 目標 | 計測方法 |
|------|------|---------|
| ウェイトリスト登録数 | 100人（最初の1ヶ月） | フォーム送信数 |
| LP訪問→登録のCVR | 10%以上 | アクセス解析 |
| チラシ→LP流入数 | QRコードのスキャン数で計測 | UTMパラメータ付きQR |

### ブランド名
- **matcha & me**（DECIDED.mdにて決定済み）

### カラーパレット（kan.mdより）

| 役割 | カラー | HEX | 配分 |

| サブ | バーチウッド（薄い木の色） | `#D4B896` | 15% |

**トーン**: Burberry的な「大胆だけど洗練」。和に寄りすぎない、モダンで都会的

### 追加カラー（背景・テキスト用）

| 役割 | カラー | HEX |
|------|--------|-----|
| 背景（明） | クリームホワイト | `#FFFAF5` |
| 背景（暗） | ディープグリーン | `#1B3D2F` |
| テキスト（主） | ほぼ黒 | `#1A1A1A` |

| テキスト（暗背景上） | クリームホワイト | `#FFFAF5` |

### デザインリファレンス

**参考ブランド**: Cha Cha Matcha × Burberry

| 要素 | Cha Cha Matchaから取る | Burberryから取る |
|------|----------------------|-----------------|
| 色味 | 深緑 × ピンクの組み合わせ、クリーム背景 | — |
| タイポグラフィ | — | セリフ体の見出し、タイトなレタースペーシング、大文字の使い方 |
| レイアウト | — | 圧倒的な余白、要素を限界まで減らす |
| 写真の空気感 | ライフスタイル寄り、明るいトーン | 落ち着いた照明、陰影のある上質感 |
| 全体のトーン | ポップさ、親しみやすさ（20%） | 静かな自信、大人の洗練（80%） |
| ボタン・UI | — | 控えめなホバー、細い罫線、ミニマルなインタラクション |
| 動き | — | 抑制されたアニメーション。派手なトランジションは使わない |

**Cha Cha Matchaとの差分（意図的に変えるポイント）**:
- Cha Cha Matchaの「可愛い・ポップ・ストリート」は抑える
- **フォントをセリフ体に変更**（ゴシック体はカジュアルすぎる。セリフ体がバーバリー的な格を出す）
- **ピンクの使い方をより控えめに**（差し色程度。ピンクが主張すると可愛くなりすぎる）
- **写真のトーンを少し暗く・深く**（白飛びしたカフェ写真ではなく、陰影のある上質な空気感）
- **テキストは詰め気味のレタースペーシング**（広いと軽くなる。詰めると高級感が出る）

## 1. ランディングページ

### 1-1. ページ構成（1ページ完結・スクロール型）

```
[セクション1] ヒーロー — ファーストビュー
[セクション2] 課題提起 — 「抹茶、飲みたいけど面倒じゃない？」
[セクション3] 解決策 — 缶ひとつで完結する体験
[セクション4] 4つの理由 — 美味しい・体にいい・手軽・おしゃれ
[セクション5] 使い方 — 3ステップ（開ける→すくう→点てる）
[セクション6] CTA — ウェイトリスト登録
[フッター] SNSリンク・コピーライト
```

### 1-2. 各セクション詳細

#### セクション1: ヒーロー（ファーストビュー）

- **背景**: クリームホワイト（`#FFFAF5`）。Burberry的なクリーンな余白
- **レイアウト**: 左にテキスト、右に缶のビジュアル（デスクトップ）。モバイルは縦積み
- **キャッチコピー**: セリフ体、大きく、1行（例: 「朝の30秒を、変えよう。」）
  - ※コピーは別途検討。ここでは方向性のみ定義
- **サブコピー**: サンセリフ体、小さめ、ダークグレー。1〜2行（例: 「缶を開けて、すくって、点てる。それだけで本格抹茶。」）
- **CTA**: 控えめだが確実に目に入る。深緑（`#2D6A4F`）背景 × クリーム文字。角丸なし or 極小
- **ナビゲーション**: 最上部に「matcha & me」のロゴタイプのみ。他のリンクは不要（1ページLP）
- **要件**: スマホファーストでデザイン。ファーストビューにCTAが収まること

#### セクション2: 課題提起

- **見出し**: 「抹茶、好きだけど自分では作れない？」
- **内容**: 従来の抹茶体験の面倒さを簡潔に伝える（道具が多い・作法がわからない・敷居が高い）
- **ビジュアル**: 茶碗・茶こし・茶筅・茶杓・茶缶…が並んだごちゃごちゃしたイメージ vs 缶1つのシンプルさ
- **トーン**: 共感。責めない。「わかる、そうだよね」

#### セクション3: 解決策

- **見出し**: 「缶ひとつで、本格抹茶。」
- **内容**: 缶（茶杓内蔵）＋ 茶筅のセットで完結。マグカップがあればすぐ飲める
- **ビジュアル**: 缶の美しいビジュアル（3Dレンダリング or 写真。MVP段階では3Dモックアップで可）
- **ポイント**: 「缶＋茶筅の2点だけ」を視覚的に強調

#### セクション4: 4つの理由

decided.mdの4つの柱をカード形式で:

| カード | 見出し | 要点 |
|--------|--------|------|
| 1 | 美味しい | 覆い香・うまみ。冷やしても美味しい |
| 2 | 体にいい | L-テアニン、EGCG。800年前に禅僧が見つけた |
| 3 | 手軽 | 缶を開けて30秒。コーヒーより手間がかからない |
| 4 | おしゃれ | インテリアに溶け込む缶。SNS映えする鮮やかな緑 |

各カードにアイコン or 小さなイラスト

#### セクション5: 使い方

3ステップを横並び（モバイルは縦積み）:

```
① 開ける        ② すくう        ③ 点てる
缶の蓋を開けて    茶杓で1すくい    お湯を注いで
茶杓を取り出す    マグに入れる     茶筅で点てる
```

各ステップにシンプルなイラスト or アイコン

#### セクション6: CTA（ウェイトリスト登録）

- **見出し**: 「一番乗りで体験する」
- **フォーム**: メールアドレスのみ（入力項目は最小限）
- **ボタン**: 「登録する」（桜ピンク背景）
- **補足テキスト**: 「発売時に優先的にご案内します。スパムは送りません。」
- **登録後**: サンクスメッセージ表示（「登録ありがとうございます！発売が近づいたらお知らせします。」）

#### フッター

- SNSリンク（Instagram / X）— アカウントは先に作っておく
- コピーライト: `© 2026 matcha & me`
- 特定商取引法に基づく表記へのリンク（必要に応じて）

### 1-3. 技術要件

| 項目 | 仕様 |
|------|------|
| ホスティング | Vercel or Netlify（無料枠で十分） |
| フレームワーク | Next.js（静的エクスポート）or シンプルなHTML/CSS/JS |
| フォーム送信先 | Google Forms / Formspree / Mailchimp（ウェイトリスト管理） |
| アクセス解析 | Google Analytics 4 or Plausible |
| QRコード計測 | UTMパラメータ付きURL（`?utm_source=flyer&utm_medium=qr`） |
| レスポンシブ | モバイルファースト。ブレイクポイント: 768px / 1024px |
| パフォーマンス | Lighthouse 90+ を目標 |
| OGP | SNSシェア時のサムネイル・タイトル・説明文を設定 |
| ドメイン | ブランド名決定後に取得。それまでは Vercel のサブドメインで運用 |
| 言語 | 日本語。グローバル展開時に英語版を追加 |

## 2. チラシ（フィジカル）

### 2-1. 目的
- オフラインでの認知獲得 → QRコードでLPに誘導
- カフェ、コワーキング、ジム、ヨガスタジオ等に設置 or 手配り

### 2-2. サイズ・仕様

| 項目 | 仕様 |
|------|------|
| サイズ | A5（148mm × 210mm）— 手に取りやすく、設置もしやすい |
| 印刷 | 両面カラー |
| 用紙 | マットコート紙 135kg（上品な質感。光沢は避ける） |
| 部数 | 初回 200〜500枚（反応を見て増刷） |

### 2-3. 表面（キャッチ面）

```
┌─────────────────────────────┐
│                             │
│   [ブランドロゴ]              │
│                             │
│   朝の30秒を、変えよう。       │
│                             │
│   [缶のビジュアル]            │
│                             │
│   缶ひとつで、本格抹茶。       │
│                             │
│                             │
└─────────────────────────────┘
```

- 背景: 抹茶グリーン（`#2D6A4F`）
- テキスト: 白 or 桜ピンク（`#F2C4C4`）
- **情報量は最小限**。目を引くビジュアルとキャッチコピーだけ
- 缶のビジュアルが主役

### 2-4. 裏面（情報面）

```
┌─────────────────────────────┐
│                             │
│  ① 開ける ② すくう ③ 点てる   │
│  [3ステップのアイコン]         │

│                             │
│  ✓ 道具は缶＋茶筅だけ         │
│  ✓ 1すくい = 1杯、計量不要     │
│  ✓ コーヒーより手間なし        │
│                             │
│  ─────────────────────────  │
│                             │
│  [QRコード]                   │
│  ▲ 発売前にウェイトリストに登録  │
│                             │
│  @matchaandme ← Instagram     │
│                             │
└─────────────────────────────┘
```

- 背景: 白 or バーチウッド（`#D4B896`）の薄い色
- テキスト: 抹茶グリーン（`#2D6A4F`）
- QRコードはUTMパラメータ付きURLにリンク
- 情報は箇条書きで簡潔に

### 2-5. デザイン原則（LP・チラシ共通）

#### タイポグラフィ
- **見出し（英語）**: セリフ体。候補: Playfair Display / Cormorant Garamond / EB Garamond。ウェイトはMedium〜SemiBold。大文字使い（letter-spacing: -0.02em〜0em、詰め気味）
- **見出し（日本語）**: 明朝系。候補: Noto Serif JP / Shippori Mincho。ウェイトはMedium
- **本文**: サンセリフ。候補: Inter / Noto Sans JP。ウェイトはRegular〜Light。読みやすさ優先
- **ブランド名「matcha & me」の表記**: 小文字。セリフ体。レタースペーシング広め（+0.1em）でロゴ的に扱う
- **注意**: 「和」に寄りすぎる筆文字・極太明朝は使わない。あくまで洋のセリフ体が主役

#### 写真・ビジュアルのトーン
- **照明**: 自然光ベースだが、やや陰影がある。白飛びしない。落ち着いた明るさ
- **色温度**: ウォーム寄り（バーチウッドのトーンと合わせる）
- **構図**: 余白多め。被写体は中央 or 三分割法。引きの構図で空間を見せる
- **スタイリング**: ミニマル。小物は最小限。缶・茶筅・マグカップだけ
- **MVP段階**: 3Dモックアップで可。実物写真はOEM試作後に差し替え

#### レイアウト
- **余白を大きく取る**（利休の引き算の美学。decided.mdより）。Burberry的な「余白で語る」アプローチ
- **セクション間のパディング**: 最低 120px（デスクトップ）/ 80px（モバイル）
- **テキストの最大幅**: 640px（長い行は読みにくい。狭く保って上品さを維持）
- **グリッド**: 12カラム。コンテンツは中央6〜8カラムに収める

#### トーン・マナー
- **コピーのトーン**: 静かだが芯がある。「！」は使わない。句読点で間を作る
- **色使い**: 3色（緑・ピンク・ウッド）+ クリームホワイト（`#FFFAF5`）+ テキスト用ダーク（`#1A1A1A`）のみ。色数を絶対に増やさない
- **アニメーション**: フェードイン程度。派手なスライド・バウンスは禁止。Burberry的な抑制
- **ボタン**: 角丸は小さめ（4px以下）or 角丸なし。パディング広め。ホバーは色の微変化のみ
- **罫線**: 使う場合は細く（1px）、色は薄く。存在を主張しない

## 3. 実装の優先順位

| 優先度 | タスク | 備考 |
|--------|--------|------|
| ★★★ | LPの実装 | ブランド名「matcha & me」で着手 |
| ★★★ | ウェイトリスト用フォームの設定 | Mailchimp or Formspree |
| ★★☆ | 缶の3Dモックアップ画像の作成 | LP・チラシのメインビジュアル用 |
| ★★☆ | チラシのデザイン・入稿 | LPと同じビジュアルを流用 |
| ★☆☆ | SNSアカウント作成（Instagram / X） | @matchaandme 等で取得 |
| ★☆☆ | OGP画像の作成 | SNSシェア用 |
