# Web

自社Webサイトの企画・設計・開発を管理するディレクトリ。

## 構成（予定）

```
web/
├── design/         ← ワイヤーフレーム・UI/UXデザイン・ブランドガイドライン
├── copy/           ← サイトコピー・商品説明文・LP原稿
├── dev/            ← ソースコード（Shopify / Next.js等）
└── seo/            ← キーワード調査・SEO戦略
```

## 検討事項

- [ ] プラットフォーム選定（Shopify / 自社構築）
- [ ] ドメイン取得
- [ ] ブランドカラー・ロゴ策定
- [ ] 商品ページ設計
