# Social Media

SNS運用に関する戦略・コンテンツ・分析を管理するディレクトリ。

## 構成（予定）

```
social-media/
├── strategy/       ← 投稿戦略・トーン＆マナー・ペルソナ設計
├── content/        ← 投稿原稿・画像素材・キャプション
├── analytics/      ← フォロワー推移・エンゲージメント分析
└── competitors/    ← 競合SNSアカウントの定点観測
```
